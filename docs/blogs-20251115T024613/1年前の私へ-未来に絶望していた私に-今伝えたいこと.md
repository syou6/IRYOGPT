# 【１年前の私へ】未来に絶望していた私に、今伝えたいこと

ーーこのまま会社員として、社会の歯車に組み込まれて一生を終えるのかな

1年前の3月31日。深夜１時過ぎ、窓の外には眠らない東京の素敵な夜景が広がっていたけれど、私の気持ちは真っ暗だった。

テレビ画面ではスプラトゥーンの派手な色彩が躍っているのに、私の心はモノクロのまま。

部屋の中に響くゲームの効果音と、自分の焦燥感。何も変えられない自分に苛立ちながら、夜だけが過ぎていく。

１年前のあなたは、会社に入ることに酷い絶望感を感じていた。

Discordで社会人の友人に「明日から会社に毎日通勤するのなんて嫌だ」と愚痴を吐きながらも、エビスビールを片手にスプラトゥーンをして現実逃避していた姿を覚えている。

ーー朝８時に起きて満員電車に乗る。会社に行くと上司にこき使われ、帰宅できるのは夜10時。気がついたら寝ていて、次の日もまた同じことを繰り返す。

そんな灰色の人生に絶望しているあなたのために、私は今日、手紙を書くことにした。

## 会社員の現実

実際に会社員としての生活は、あなたが思っている通り、楽しいものではなかった。仲の良い同期は一人もできない。周りと話をあわせるのも疲れるので、入社から２週間もすれば単独行動が当たり前になる。

同僚と話すのは疲れて仕方がない。気を遣わされるのが嫌。そのため、在宅勤務が許可された瞬間に一度も出社しなくなるほど、職場との距離を取るようになる。

研修も仕事も嫌い。4月2日に給与テーブルを見て努力が無駄であることを悟る。毎日オフィスで「帰ってからやることリスト」を作ったり、窓に両手を当てて「ここから出してくれ！」と囚人のマネをして、現実逃避で時間を潰していた。次第に会社で読書をするようになってメンタルは安定するけどね。

## 突然訪れた転機

でもあなたはこれから打ち込めるものに出会う。

あなたは、6月から半年に渡り、本を書くことになるんだ。

信じられないかもしれないのも無理はないよね。あなたはまともに文章を書いた経験なんてなかったから。100日チャレンジを始めたのだって、ChatGPTに授業のレポートを書かせるところからスタートしたくらいだし、文章を書くのが嫌でお金を払って知り合いに書いてもらったこともある。

4月5日、日経BPを訪れて企画書を作ることになる。そして２ヶ月後、その企画が通り、6月から本を書くことになったんだ。

企画が通った瞬間、あなたはまるで新しいおもちゃを買ってもらった子どものように喜んでいた。そのくらい、既存のエンタメの消費では満たされない、感情エネルギーの放出先を探していたんだね。その日から、朝から晩まで、100日チャレンジと同じくらい夢中になって文章を書くことになる。

## 厳しい下積み

周りから企画が通ったことを祝福されたのも束の間、苦しい日々を送ることになる。

もちろん簡単には進まない。最初の２ヶ月間はまったく思ったように文章が書けず、苦しい日々を過ごすことになる。

毎週編集者に目標の半分くらいの量の原稿を提出しては、編集者から会議室では厳しい指摘をされる。

私の拙い原稿を読んだ編集者から、何度も遠回しに「ここはどのような意図で書いたんですか？」と尋ねられた。そのたびに言葉を尽くして説明したけれど、返ってくるのはいつも「それなら、はじめからそのように書いてください」という言葉だった。伝えたいことがうまく表現できない自分に、強い無力感と自責の念を覚えていた。

会議の後はいつもイタリアンに行って、ワインを飲みながら励まされた。編集者は私の酷い文章の中でも良いところを探し、私にお酒が入っているこのときだけは好意的なフィードバックをくれた。

目標だった３ヶ月での出版はできなかった。本当に出版まで辿り着けるのか、毎晩不安に襲われて眠れない日もあった。毎日朝から晩まで本の出版のことだけを考えていた。

でも、7月中旬、手元のメモや#100日チャレンジ中の日記をほとんどすべて編集者に明かしてから、少しずつ書くのが上手くなっていく。

そうして、12月4日、本は完成した。

窓の外には再び、東京の街が広がっていた。あの日と同じ眠らない景色だけれど、私の気持ちはあのときと全く違う。画面に映る最終稿を前に、心は静かな達成感で満たされていた。

## 未来へのメッセージ

この１年は確かに辛い下積みの時代だった。だけど、１年後の今、その苦労は全て報われた。本はベストセラーになり、収入源は多角化され、会社を設立することになる。

会社では友達ができなくても、あなたは次第に色々な場所で講演や国際会議に参加し、個性豊かな友人と繋がっていく。あなたを応援してくれるファンもたくさんついてくれる。

2024年の残りの９ヶ月は、まだまだ苦労が絶えないかもしれない。でもあなたの努力が私の今を作っている。この１年で、あなたは本当にできる限りのことをやり抜いた。そのおかげで、今私はとても明るい未来を見られている。ありがとう。

この２年で私は、宿題をサボるためにChatGPTを使った結果、学会に出ることになり、ノリで始めた100日チャレンジがエンジニアへの道を開き、そして本を出し、ついには会社を設立するところまで来た。

１年前、想像すらしなかった未来が今ここにある。だからきっと、１年後の今もまた、私の想像を超えた場所にいるのだろう。

あなたがくれたこの未来を、私は大切にしていこうと思う。
