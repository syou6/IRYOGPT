# 23歳、人生初の月収100万円で見えた世界

**「社会人1年目で、月収100万円を稼ぎました!」**

こう聞くと、多くの人が想像するのは、まるで映画のワンシーンみたいなきらびやかな世界かもしれない。

高級ホテルでシャンパンを開けて祝ったり、ブランドショップで衝動買いをしたり、あるいはSNSに「ついに月収100万円！」と華々しく宣言したり。

**“もうお金の心配なんてしなくていい”“人生イージーモード突入！”**

そんな、何かが劇的に変わったかのようなイメージが浮かぶだろう。

しかし、私が23歳で初めて月収100万円を手にしたとき、現実は想像していたよりずっと静かで、淡々としたものだった。

## 現実は静かに、数字だけが増えていく

2025年1月、私は初めて単月で100万円稼いだ。

起業したのは2024年12月3日。それからわずか2か月で達成したことになる。

新入社員時代の月収は28万円（残業を入れても35万円ほど）。そこから一気に3倍近くになった計算になる。

よく「その月は何かお祝いしたの？」と聞かれるけれど特にしていない。事業収入って、会社員のように月末に一斉に振り込まれるわけではないからだ。

私が「100万円達成した」と気づいたのは、実際に口座に振り込まれた瞬間ではなく、帳簿を締めた翌月末だった。

実は私が「100万円稼いだ」と気づいたのは、振り込みを確認したときではなく、帳簿を締めた翌月末のことだった。

1月は60万円と40万円の案件を同時にこなしていたけれど、報酬の入金にはタイムラグがある。

それに、当時とても忙しかった。ざっと挙げるとこんな感じになる。

note100日チャレンジ(note毎日投稿）

-

気付けば、1月1日からGWのヨーロッパ出張前日の4月25日まで、116日間連続で働いていた。

だから、“一気に100万円が手に入る”ようなドラマはまったくなく、
ただ銀行口座の数字が静かに増えていくのを時々確認し、「あ、増えてるな」と思うだけだった。

## 「毎日カプチーノ」を気にせず飲めるようになった

それでも、月収100万円になったことで、日々の暮らしに“小さな余裕”が生まれたのは確かだった。

例えば、毎日カプチーノを飲んでも、チェーン店以外のカフェやレストランで外食しても、お金のことを気にしなくてよくなった。

そんなちょっとした贅沢も、お金のことを気にせず楽しめるようになった。

正直、それまでも月に35万円くらいは稼いでいたので、「我慢ばかりの生活」ではなかったけれど、それでも“普段と同じ生活をしているのに、お金が明らかに余る”という安心感は格別だった。

実は会社設立時、手元にあったのは30万円ほど。会計ソフトの導入や諸経費がかさんだこともあった。

（とはいえ、本を書き終えた後に大阪で遊んだり、ウィーン出張ついでにスウェーデンに行ったときの30～40万円の出費のほうが大きかったかもしれない）

だからこそ、2か月くらい経ってから「気づけば妙にお金に余裕がある」という実感がじわじわと湧いてきた。

##  「自分への投資」に惜しみなく使い始めた

収入が増えて、もうひとつ大きく変わったのは、「自分への投資」に対してケチらなくなったこと。

月収100万円を経験してからは、整体やパーソナルジムに通い始め、長時間の仕事に耐えるためのフットマッサージャーを買い、Apple Watchを手に入れ、生成AI関連のサービスにもどんどん課金した。

[https://note.com/amiotsuka/n/n2103429bff6c](https://note.com/amiotsuka/n/n2103429bff6c)

これが「投資」なのか「贅沢」なのか、自分でも分からない。
でも、“自分の可処分時間を増やすため”のお金は惜しまなくなったし、
働けば働くだけお金が入る状態だったので（当時は1日12～13時間働いていた）、
疲労や体調不良でダウンするくらいなら予防のためにどんどん使おう、という気持ちに切り替わった。

その後も収入はさらに上がっていったけれど、支出は2025年2月がピークで、そこから整理して今に至る。

“お金は、不快感をほとんど取り除いてくれた”――それが一番の変化かもしれない。

## 静かに変わる世界と、未来の私へ

「月収100万円」と聞くと、多くの人がきらびやかな世界を想像する。
でも、私の現実は、口座の数字が静かに増えていき、生活にちょっとした余裕が生まれ、「自分にお金を使うこと」をためらわなくなった――そんな穏やかな変化だった。

本当にうれしかったのは、日々の小さな贅沢を気にせず楽しめるようになったこと、そして、将来の自分に投資できる“自由”を得たこと。
生活そのものが劇的に変わったわけじゃないけれど、「お金が人生の選択肢を少しだけ増やしてくれた」ことが、何よりの収穫だったと思う。

これから先、また新しいフェーズに進んでいく中で、おそらく私はもっと豊かになれる。

数字では測れない「本当の豊かさ」を、少しずつ自分の手で見つけていきたい。
