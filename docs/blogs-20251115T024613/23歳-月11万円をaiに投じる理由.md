# 23歳、月11万円をAIに投じる理由

「23歳、月11万円をAIに使っている。」

そう伝えると、多くの人が驚き、疑問の表情を浮かべる。「そこまでAIにお金をかける価値があるのか？」と。

実際その主張はよく理解できる。2年前のAIは安価で頼りなかった。出力された結果には膨大な回数の修正が必要だった。私自身も5000時間以上、ChatGPTと会話しながらプログラミングを学んできた経験がある。

しかし、時代は変わった。**これは未来を掴むための戦略的投資だ。**

AIはすでに私にとっては贅沢品ではなく、この業界で生き残るための必需品になりつつある。

## 月11万円のAI利用、その内訳

私が毎月使う11万円の内訳は以下の通りだ。

-

Devin：80,000円

-

ChatGPT：30,000円

-

Claude：3,000円

-

Notta：1,200円

これらをすべて使っているのは、**用途がまったく異なるから**だ。

ChatGPTは主に文書作成や相談相手として使っている。文章を素早くまとめたり、意見をもらう際には非常に頼りになるパートナーだ。

一方、Devinは自律型AIで、指示書を作成しておけば、あとはほぼ自動的に作業を進めてくれる。このツールのおかげで、仕事を任せている間に出かけることができ、休憩時間を確保できる。

Claudeは、コーディングをしてもらうために使用している。

ChaGPTよりも出すコードの品質が良いので、コードの実装速度が劇的に上がる。800行以内のプログラムに落とし込むことができれば、これを使うことができる。

さらにNottaは会議や打ち合わせでの議事録作成に役立つ。日本語の精度は微妙だけれど、英語の録音・文字起こし精度は非常に高く、会議するときには欠かせない存在となっている。

## AIを使わない = 開発競争に負ける

AIを使いこなす最大の理由は、開発速度を圧倒的に高める必要があるからだ。

現在、AIはまだ自動で完璧にプログラムを作成することはできない。だから、人間と協力して開発を進める必要がある。

私はこれらのAIをまるで自分の部下のように活用している。1日中AIに指示を出し、それを受領して修正する。もはや自分でコードをゼロから書くなんてことはしない。

また、何か問題があればすぐにフィードバックが返ってくるため、私自身の設計能力を鍛えるのにも役立っている。

コードを色々と作ってもらいどこか異変を感じ取れると、それはだいたい設計のミスだ。AIの精度は相当高いので、私サイドに問題があることがほとんどだからだ。しかし、毎日何度も試行錯誤を繰り返して設計をしているうちに、自然とプログラム設計の腕が上がっている。

通常、設計のフィードバックはそう簡単に得られないけれど、私は数時間で得られる。これだけの学習効率を得られている人は世の中にほとんどいないのではないか？

## AI時代は経済力の勝負に突入した

2023年当初、GPTをはじめとするAIサービスは月額3000円程度で利用可能だった。価格が安かった反面、AIの性能も低く、修正の手間を考えると自分で書いた方が早いという人も多かった。

しかし2025年現在、AIは劇的に進化を遂げた。性能が飛躍的に向上するのに伴い、コストも急激に上がってきている。

今後さらにAIの性能が向上するにつれて、その費用はますます高額化するだろう。

**最新AIを駆使して開発を行うためには、経済力が不可欠になる。**

稼げない人は高品質なAIを維持できず、結果として開発競争に敗北してしまう。

**生き残りたいならば労働生産性を上げる必要がある。**

最新AIを活用して仕事のスピードと品質を向上させ、AIを単なる話題や流行としてではなく、**本気の開発ツールとして使いこなさなくてはならない。**

これから警戒すべき相手は、AIそのものでも、SNS上で煽る人々でもない。**本当に警戒すべきは、自分の仕事の質を高めるためにAIに積極的に投資している同業者**だ。

この業界で生き残るには、AI投資の余力を持つために、継続的に高単価の案件を獲得し続けることが重要だ。AIに月11万円を投じるのは、一見すると高額な投資に見えるかもしれない。

しかし、未来を見据えればそれは不可避の選択であり、むしろ当然の判断なのだ。
