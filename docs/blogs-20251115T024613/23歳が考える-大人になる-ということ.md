# 23歳が考える、「大人になる」ということ

あなたは成功するために何を犠牲にしてきただろうか？

ーー私の場合は、**孤独**だった。

## 成功に向けて

大学4年の10月、私はすべてを捨てて「100日チャレンジ」という企画に没頭した。何もかもを投げ打って、その企画にすべての時間とエネルギーを注いだ。寝ても覚めても頭の中は企画のことばかりだった。その結果、その企画は成功し、学会で賞を獲り、招待講演を受け、国際会議でスペインにまで行った。帰国すると[メディア](https://ascii.jp/elem/000/004/185/4185964/)に取り上げて貰い、私はエンジニアとしてのキャリアをスタートさせた。

社会人1年目のとき、今度は[本の執筆に全力を注いだ。](https://note.com/amiotsuka/n/nda94c4fa32cf)毎日、原稿と向き合った。友達との飲み会や同期との集まりなど、誘われてもことごとく断った。次第に一切誘われなくなったけれど、私は朝から晩まで執筆に没頭した。

結果として、その本はベストセラーになり、他の事業の好調も重なり、私は23歳で[会社を立ち上げることになった。](https://note.com/amiotsuka/n/n777f583e302f)

今日で、大学を卒業してから、ちょうど1年経った。

これらの挑戦を後悔してはいない。むしろ私は、「上手くやった」部類に入るだろうと自分でも思っている。

## 意図的に選んだ孤独

しかし、この成功を掴む過程で、私は意図的に孤独を選んでいた。

私が打ち込む挑戦に対して否定的な意見を持つ人や、自分のエネルギーを吸い取るような人々を、積極的に遠ざけていた。挑戦を否定され、せっかく高まった情熱に水を差されるのが嫌だったからだ。

私は卒業後1回だけ、同期や大学の友人と会った。安い居酒屋の薄暗い照明の中で、彼ら・彼女らは就職活動の苦労話や、会社や講義の愚痴を笑いながら話している。

**でも私には、その時間が耐えられなかった。**

心のどこかで「つまらない」と思っている自分がいた。あまりにも退屈な会話に、時間を無駄にしているとすら感じていた。

かといって、そこで私の刺激的な体験を話すこともできなかった。話した瞬間に、周囲の空気が微妙に変わることを何度も経験していたからだ。

結果として、私の周囲には仕事の関係者や私と環境が似ている人間だけが残った。

## イエスマンの誘惑と現実

よく、経営者や何かの挑戦で失敗した人が語る理由の一つに、「周囲がイエスマンばかりだった」という話を聞く。もちろんその危険性は理解している。こういう経営に関する本やドキュメンタリーは学生時代から好きで、何度も読んだ。

**しかし、現実はそんなに簡単ではない。**

否定的な意見をすべて取り入れていたら、何一つ前に進めない。挑戦とは多かれ少なかれ、嵐のように襲いかかる周りの否定を突破することだ。私にとっては、周囲の否定的な意見を黙殺することが成功の秘訣だった。

だが、それと引き換えに私は多くのものを失ったようだ。

昔の友人との関係、学生時代の思い出話で盛り上がるような関係が、ほぼすべて切れてしまった。いま私が感じる孤独は、まさにその結果だろう。

SNSや上の世代からの意見では、「学生時代の友達を大事にしろ」「同期とのつながりを大切にしろ」とよく目にする。その言葉の意味を今さらになって噛み締めている。成功を追い求める過程で、私はこれらのゆるく繋がった、利害関係のない人間関係を置き去りにしてきた。

今、私の周囲には、たくさんのファンや支えてくれる人々がいる。多くの人が私の成功を喜び、応援の声を届けてくれる。皮肉なことに、それでもなお、私は孤独だと感じる。

## 「効率の良い」孤独な遊び

だから私は今、刹那的な遊びに手を出すようになったのだと思う。

オンラインゲームで出会う、名前も知らない人と一晩だけの協力プレイをする。アミューズメントポーカーで偶然隣に座った人と数時間の交流を楽しむ。バーに行き、見知らぬ誰かと意味のない会話を交わす。

刹那的な遊びは私の心の隙間を埋めてくれる。自販機のジュースのように、お金を入れればすぐに喉を潤してくれる。

これはあまりにも効率がいい。お金さえ払えばすべて解決する。その場のやり場のない気持ちを吐き出すには簡単で、リスクも負担も少ない。周りに迷惑もかからないし、余計な心配もかけずに済む。それに、こういう若者向けの娯楽は、そこまでお金はかからない。

## 孤独は必要だったのか？

「効率の悪い」人間関係はだいたい消え去ってしまった。でも、これは仕方がなかった。こうしないと、自らの挑戦を成功させられないから。周囲を心理カウンセラー代わりにするのも気が引けるし、情報が漏洩するのも怖い。そもそも、専門家相談以外で機密情報を開示することは、職業倫理的にできない。

**これは、「大人になる」ということの一つの形なのだろうか?**

けれど、今の私に「後悔しているか？」と問いかけられても、はっきりとは答えられない。多分、後悔はしていないのだろう。私は、孤独になる必要があった。でも、この生き方が本当に私にとって一番良い選択肢だったかは分からない。

それでも私はまた、孤独を求め、次の挑戦に取り組んでいくのだと思う。

孤独は確かに辛い。しかし、孤独を選んだからこそ、私は自由に挑戦することができたのだ。ときどき、誰とも比較されず、誰にも気を使う必要のない「孤独な自由」に、安心を覚える。

大人になるとはきっと、こういう矛盾を抱えながら生きていくことなのかもしれない。
