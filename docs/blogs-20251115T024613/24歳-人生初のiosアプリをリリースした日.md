# 24歳、人生初のiOSアプリをリリースした日

「日々の小さなひらめきや、大切な思い出をもっと気軽に残せたらいいのに」

日常の中で浮かぶアイデアや、ふとした気づき、あるいは旅行先での一瞬の感情を記録したい。でも、従来のメモアプリでは「書くぞ」と身構えないと続かない、そんな不便さを感じてきた。

そこで生まれたのが、今回リリースする新しいメモアプリ ****[「メモるん」](https://apps.apple.com/jp/app/%E3%83%A1%E3%83%A2%E3%82%8B%E3%82%93-memolune/id6752797808) だ。コンセプトはシンプル。「チャットのように気楽に書ける」「音声でも写真でも残せる」「検索しやすい」という三本柱。まるで友達と会話するように思い出やアイデアをストックできるアプリを目指した。

この記事では、このアプリがどのように生まれたのか、どんな体験を届けたいのか、そしてなぜこのアプリを世の中に出そうと思ったのかを綴っていく。

## 100日チャレンジから見えた“記録”の価値

私はこれまで、生成AIや技術を使いながら100日間で100個のアプリを作るチャレンジを行い、それを本にもまとめてきた。

その過程で痛感したのが、「アイデアや経験を記録する仕組みが、自分にとっていかに重要か」ということだ。日記やメモ帳に書き留めておかないと、思考の流れや大切な出来事はあっという間に風化してしまう。

けれど、従来のメモアプリはどこか「作業っぽさ」があった。UIが複雑だったり、音声や写真を取り込むまでのステップが多すぎたり。探す手間が多かったり、入力したメッセージがどこにあるのか分からなくなってしまったり。

もっと直感的に、もっと軽やかに残せる仕組みはないのか？ そう考えたとき、「チャット形式ならどうだろう」とひらめいた。普段からLINEやメッセージアプリを使い慣れている私たちにとって、チャットUIは最も自然で、感情をそのまま書き出せる形だからだ。それに、周りの友人はみんな、LineやX、Instagramなどで一人グループを作って、そこにメモをしていた。

## 3つの柱で支える「メモるん」の仕組み

「メモるん」には3つの大きな特徴がある。

-

**チャットUIでの記録**
まるで友人に送るメッセージのように、気楽に文章を残せる。日付ごとに自動整理されるので、「あの時の気持ち」を振り返るのも簡単!

-

**音声・OCR対応**
アイデアが浮かんだ瞬間に音声をワンタップで録音し、自動で文字起こし。さらに写真やスクリーンショットから文字を抜き出すOCR機能も搭載している。会議資料や本の一節なども即座にメモ化可能。

-

**カレンダーと検索の便利さ**
現代の私たちは、カレンダーに色々な予定を書き込むため、カレンダーの予定からなんとなくいつの出来事かを予想し、思い出そうとする習慣が身についている。そのため、日付を使えば簡単に見つかるのではないかと思い、主要な検索方法を、カレンダーにした。　もちろん、記録したメモはカレンダーから振り返れるだけでなく、キーワード検索もスムーズ。「あの時のあの言葉」を探すストレスを大幅に減らした。

これらを組み合わせることで、「記録する負担を極限まで減らす」ことを意識した。ユーザーがメモを残すこと自体を楽しめるように設計した点が、最大の工夫だ。

## 「生活の一部になるメモ」を目指して

私がこのアプリに込めた想いは、「メモが生活の一部になる」体験を届けることだ。仕事のタスク管理だけではなく、旅先での感動、友人との何気ない会話、夜中に思いついた夢の断片……それらを気軽に残せることで、人生はもっと豊かになると信じている。

さらに、ユーザー一人ひとりの「デジタルな日記帳」としての役割も果たせるよう意識した。スマホを開けば、そこに過去の自分との対話が並んでいる。読み返すたびに「こんなこと考えてたんだ」と気づける。それは単なる便利さを超え、自分自身を深く理解するためのきっかけになるはずだ。

「メモるん」はまだ始まったばかりのアプリです。これからも機能改善やアップデートを重ねながら、ユーザーと一緒に成長していきたいと思っています。

## これからも一緒に進化していくアプリへ

アプリ開発において一番大切なのは、「自分自身が本当に欲しいと思えるものを作ること」だと私は考えている。「メモるん」はまさに私自身が欲しかったアプリであり、同じように感じてくれる人がきっといるはずだ。

私がアプリ開発をしたり、本を書いたり、メディアでのネタやフレーズを考えたするときに使いたい。「100日チャレンジ」と同じくらいの人生の確変をもう一度起こしたい。そのために私は、3ヶ月で2人月以上の時間をかけて、このアプリを作った。

もしあなたが、「もっと気軽にメモしたい」「思い出をチャットみたいに残したい」と感じているなら、ぜひ一度試してみて欲しい。あなたの日常を、少しだけ便利に、そして楽しく彩る存在になることを願っている。
