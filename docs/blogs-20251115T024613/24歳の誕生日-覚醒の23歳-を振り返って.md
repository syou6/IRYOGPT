# 24歳の誕生日。“覚醒の23歳”を振り返って

「たった1年で、人生はこんなにも変わるのか。」

1年前の自分が今の私を見たら、きっと目を疑うと思う。
本を書き、会社をつくり、メディアにたくさん出て、いつの間にかベストセラー作家になっていた――。
数年前の私には想像もできなかった現実が、気がつけばすぐ隣にある。

今日は、私の24歳の誕生日。
この1年は、間違いなく私の人生で最も濃く、最も劇的だった。

それは「覚醒」という言葉がぴったりの、かつてないほど自分の可能性が開いた一年だった。

## 煌びやかな理想と、閉塞感に満ちた現実

23歳を迎えた頃、私は新卒1年目の会社員だった。

学生時代は、周りからは「絶対に大物になる」と期待されていた。
国内の学会で2回賞を取り、名古屋で招待講演をし、スペインの国際学会に行った。周りからの称賛や祝福から、私は艶やかな未来を想像していた。

[https://ascii.jp/elem/000/004/185/4185964/](https://ascii.jp/elem/000/004/185/4185964/)

しかし、実際の社会人になってからの生活は、全く違った。

朝から晩まで働かされた挙げ句、月収は26万円。どれだけ頑張っても、年度末の最高評価でやっと月給が5,000円上がるだけ。この先、何十年もこのままなのかと考えると、胸が苦しくなった。

周りは「Hello World!」から始めるほどのプログラミング初心者なのに、私はそんな周りと同列に扱われ、報酬も同じ。屈辱的だった。

**本当にこのままでいいのか？**

社会人生活は、意外なほど単調で、思った以上に“自分の人生”という感覚が持てなかった。だから、入社してしばらくは、現実が受け入れられず、バーやポーカーなど、夜の街に吸い込まれていった。

そんな中で、唯一“自分の人生を変えるチャンス”だと感じられたのが、「本の執筆」だった。

2024年5月末から始まった本の執筆は、最初は本当に手探りでした。文章をまとめる難しさ、自分の経験や考えをどう言葉にすればいいのか、何度も壁にぶつかった。そもそも、レポート課題をやりたくなくて生成AIにハマった私にとって、文章を書くなんてことは、できるわけがなかった。生成AIにやらせる前は、周りを買収して宿題をやらせていたくらいだ。

とはいえ、人生の閉塞感から脱するためにはこれしかない。私は、7か月かけて、本を書いた。

## 会社設立：今年一番の英断

11月15日、本の原稿を書き終えた私は、まさに燃え尽き症候群のような状態だった。
やっと大仕事を成し遂げた後の虚無感に包まれ、しばらくは何も手につかなかった。

そんなとき、思いがけず受注開発の相談がいくつか舞い込んできた。
**「iOSで、サーバーと連携したアプリを作ろうと思っているんです」**
クライアントのひとりがそう話してくれたのをきっかけに、私は詳細を聞くことにした。

しかし、そのクライアントはこう付け加えた。
**「インボイスの関係で、個人には発注できないんですよね」**

提示された報酬額は、当時の私の月給の2ヶ月分ほど。
私は咄嗟に**「今、会社設立の手続きを進めている最中です！」**と答え、さらに「アプリについて、もう少し詳しく教えていただけますか？」と切り返した。
実際のところ、私はiOSアプリもサーバーサイドも未経験で、もちろん会社設立の手続きもこれからだった。

それでも、迷っている余裕はなかった。
金曜日に話をもらったその日のうちに会社設立の準備を始め、週末には関連する技術書を3冊購入。
火曜日までの数日間、私はひたすら新しい技術を勉強し続けた。

詳細をもらったタイミングで「会社設立手続きは少し遅れています」と正直に伝えつつ、アプリ開発は「できます」と自信を持って答えることができたーーできる道筋が見えたからだ。

その後、私はヨーロッパの大学で講演するため渡航した。（これは、国際学会で知り合った研究者の方からのご縁だった。）
ヨーロッパ滞在中も、空いた時間を使ってアプリ開発に取り組み、会社設立の手続きを着実に進めていった。

## 本の発売と、発信活動!

年が明けて、2025年1月14日に拙書[「#100日チャレンジ 毎日連続100本アプリを作ったら人生が変わった」](https://amzn.to/40QQbd8)が発売された。

その出版に合わせて、1月1日から、「#note100日チャレンジ」と題して、毎日noteに文章を投稿し始めた。また、それまで止まっていたXでの発信も再開し、少しずつ自分の考えや日常を外に向けて発信することを意識した。

本の発売直後は正直、どれくらいの人が読んでくれるのか、どんな反応が返ってくるのか、不安でいっぱいだった。

ところが1月末を過ぎた頃から、取材依頼が毎週のように舞い込むようになり、自分の本が思いがけず多くの人に届いていることを実感した。

その一方で、裏側では受注開発の案件にも追われていた。
慣れない仕事に戸惑い、思うように進まない日も多かったけれど、「自分で選んだ道を自分で切り拓いていく」という覚悟が、私を前に進ませてくれた。

## “自分の物語”が動き出す

2025年3月末、本の売れ行きは衰えるどころか、むしろ加速していった。

出版社からは「今月が取材のピークですね」と1月末から何度も言われていたけれど、4月に入ってもその勢いは全く衰えなかった。それどころか、取材依頼だけではなく、[テレビ](https://www.youtube.com/watch?v=iD1sVwUwgrg)や[ラジオの出演依頼](https://x.com/AmiOtsuka_SE/status/1940200355750678885)や、[講演依頼](https://rd.hitachi.co.jp/_ct/17776040)も来るようになった。

気づけば、私の本は“ベストセラー”という称号を手にしていた。

[https://note.com/amiotsuka/n/nc62eee0e8191](https://note.com/amiotsuka/n/nc62eee0e8191)

ただ、華やかな表舞台の裏側では、受注開発の仕事が増えたり、講演の準備をしたり、それらの単価を上げるための交渉や調整にも追われていた。

そのため、noteの毎日更新も、4月上旬からは忙しさに追いつけなくなり、投稿が途切れがちに。「#note100日チャレンジ」を終えた後は週1回出せるかどうか、という感じになってしまっていた。

私の強みは、「エンジニアで研究者で作家」であるということ。だから、色々な所から仕事が来て、仕事が途切れることはなかった。おかげさまで毎週カレンダーは綺麗に埋まっていた。

「一人で何役もこなす」のは、とても大変だったけどね。

## 「覚醒」のその先へ

振り返れば、この1年は、ジェットコースターのようだった。

夢中で走り抜けた日々は、達成感や喜びに満ちていた反面、不安やプレッシャー、そして絶え間ない葛藤もついてまわった。「人生を変える！」と勢いで会社をつくり、本を出版し、走りながら学び、何度も壁にぶつかった。周りから見れば華やかに見えるかもしれないけれど、実際は、泥臭くて、不器用で、毎日が綱渡りの連続だった。

それでも、確信できることがある。

この1年、**私はずっと「自分の人生」を取り戻そうとし続けてきた**、ということだ。

社会人になり、どこか他人の物語をなぞるように生きていた私が、ようやく「自分で考え、自分で選び、自分で動く」感覚を思い出せた。ベストセラー作家、起業家、エンジニア、研究者――どんな肩書きも、結局は全部「私」という人間の一部にすぎない。どんなに華やかな実績も、「自分で納得できる人生を生きる」ことに勝るものはないのだ、と今なら思う。

これからのことは、正直まだ全然わからない。来年の今ごろ、また何をしているのかも想像もつかない。ただ、ひとつだけ決めていることがある。

**もう、どんな選択をしても、後悔しない。**

たとえ失敗しても、転んでも、また立ち上がればいい。自分で決めた道を、自分の足で歩いていく。その覚悟さえあれば、何度でも人生は書き換えられる。

1年前、私は「このままでいいのか」と悩み、立ち止まっていた。

だけど今の私は、どんな道も自分で切り拓けると信じている。

人生は思い描いた通りには進まない。むしろ、想像もしていなかったチャンスや出会いが、思いもよらぬタイミングで舞い込んでくる。そのたびに戸惑い、迷い、悩みながらも、「自分で選ぶ」ことをやめなければ、どこまでも進める。

24歳の今、私には“これが私の人生だ”と胸を張れる実感がある。そして、これから先もきっと、たくさん迷い、間違い、挑戦して、また新しい自分に出会い続けるのだろう。

未来のことは誰にもわからない。
けれど、「自分で考え、自分で決めて、自分で動く」――その習慣だけは、ずっと手放さないでいたい。
どんな場所に辿り着いても、それは“誰かの人生”じゃなく、確かに“私の人生”だと、誇れるように。

始めるのに、遅すぎることなんてない。
何度でも、やり直せる。

これからも私は、自分の手で可能性の扉を開き続けていきたい。

もし一年後、また今の自分を振り返る日が来たとしたら――
そのときはきっと、今よりもっと自由に、もっと軽やかに、“自分の物語”を歩いていると信じている。

どんな未来が待っていようとも、私はこの手で、何度でも未来を書き換えていく。

24歳も、また違う私と出会う一年になりますように。
