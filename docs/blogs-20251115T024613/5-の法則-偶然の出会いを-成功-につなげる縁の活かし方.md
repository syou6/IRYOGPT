# 【5%の法則】偶然の出会いを「成功」につなげる縁の活かし方

「23歳で著書がベストセラーとなり、会社設立するほど成功した。」

そんな話を聞いた人の多くは、「きっと綿密な人生設計があったんだろう」と考えるだろう。

実際にはそうではない。私は特別な才能や明確な人生計画を持っていたわけではない。ChatGPTを使って宿題をサボり、授業中に別のことをしている、怠惰でありふれた普通の大学生だった。

しかし、偶然が偶然を呼んだことで、私の人生は予想もしなかった方向へ動き始めた。

## 偶然の出会い

すべてが始まったのは、ある5月の授業中だった。退屈だった私は、授業そっちのけでパソコンでオセロゲームを作っていた。それを偶然見かけた伊藤先生から、「学会に出てみないか」と提案を受けた。

「無茶ぶりではないか」と思いながらも、一生に一度の機会だと思い受諾。全力で挑んだ結果、なんと学会で受賞することができた。その成功はさらに招待講演、さらには国際会議でスペインに行くという、次の縁を生むことになる。

伊藤先生から、書評を貰った。

## 偶然の取材

次の偶然の縁を運んできてくれたのが[遠藤さん](https://x.com/hortense667)だった。[佐々木さん](https://x.com/gclue_akira)に誘われて訪れたMFT(Maker Fair Tokyo) で偶然知り合った遠藤さんとは何度か会って話しており、私が「100日チャレンジ」をしていることを知ると、「終わったら取材してあげるよ」と言われた。

その後実際に[春日のオフィスで私を取材してくれ](https://ascii.jp/elem/000/004/185/4185964/)、その縁で編集者を紹介してもらえた。

私はこの偶然の機会を逃すまいと、「#100日チャレンジ」をテーマにした本を書くことにし、出版社で企画を通した。その後、半年以上かけて執筆に明け暮れることとなる。

遠藤さんからも、書評を寄稿してもらった。

[https://bookplus.nikkei.com/atcl/column/070600261/122300009/?cx_testId=3&cx_testVariant=cx_1&cx_artPos=4&i_cid=nbpbkp_atcl_rec#cxrecs_s](https://bookplus.nikkei.com/atcl/column/070600261/122300009/?cx_testId=3&cx_testVariant=cx_1&cx_artPos=4&i_cid=nbpbkp_atcl_rec#cxrecs_s)

## 偶然の書評

そして、もう一つの偶然が起きた。本が出版されると、今度は遠藤さんの記事をたまたま目にした漫画『ゲームセンターあらし』『こんにちはパソコン』の作者であるすがやみつる先生が、なんと「世界最速の書評」を書いてくださったのだ。

[https://bookplus.nikkei.com/atcl/column/041100040/120600045/](https://bookplus.nikkei.com/atcl/column/041100040/120600045/)

そのおかげもあり、『#100日チャレンジ』は一気に話題を呼び、ベストセラーにまで登りつめることとなった。

## 偶然を必然に変える生き方

私がこの経験から強く学んだことは、人生の偶然を成功に変える方法だ。それは偶然に出会ったときに、「これはチャンスだ」と直感し、そこにほぼ全てのリソースを投入する勇気を持つことだ。※

縁はあくまでもきっかけに過ぎないし、これ自体は偶然の代物。確実性なんてものは存在しない。20人に1人くらいの割合で訪れる偶然の縁は、何もせずにいたらただ通り過ぎていくだけ。しかし、

**そこに全力を注ぐことで、初めて意味を持つ。**

私が今こうして本を出し、自ら会社まで設立するに至ったのも、偶然をつかみ、全力で活かした結果なのだから。
