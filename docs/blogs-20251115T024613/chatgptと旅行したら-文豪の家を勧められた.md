# ChatGPTと旅行したら、文豪の家を勧められた

「旭川でおすすめの観光スポット教えて」

暑いお盆の某日、私は行き先を探していた。正直、予定は白紙だ。暑くて混んでいる渋谷から逃げるように北海道に来たはいいけれど、ホテルと飛行機を取っただけで、予定は全く決まっていない。

予定表は真っ白、頭の中も真っ白。自然や動物園はもう見飽きたし、Google mapを見ても分からないし、旅行雑誌を見るのもめんどくさい。

そんな中、私はスマホを手に取りChatGPTに頼った。

ChatGPTは自信満々に言った。「井上靖記念館がおすすめです。」

観光地らしからぬチョイスに思わず笑ってしまったけれど、作家である私は心が揺れた。知られざる日常や創作の裏側を覗けるかもしれない。気づけば、私は静かな住宅街へ向かって車を走らせていた。

## ChatGPTに聞いたら、勧められた

今回の旭川旅行は、特に目的はなかった。けれど、職業柄なのか分からないけど、どこへ行っても「これ、ブログのネタ/ 話のネタになるかな？」と考えてしまう。

特に、同業の作家が使っていた机や書斎、日々の習慣を知るのは、私の創作にも直結する。井上靖さんは戦後文学を代表する文豪でありながら、新聞記者出身という経歴の持ち主。私は文学なんて全然読まないけれど、同業者として少し親近感を持った。

実のところ、旭川での観光地候補があまり思いつかず、ChatGPTに尋ねて出てきたのがこの記念館だった。場所は旭川駅から車で15分ほどの閑静な住宅街の一角。外観は落ち着いた和風建築で、まるで古い邸宅に招かれたような感覚だった。

## 栄光までの助走は長かった

展示室を歩きながらまず驚いたのは、井上靖さんが“遅咲き”だったことだ。芥川賞を受賞したのは30代前半。それまで新聞記者として働きながら、短編や詩を雑誌に投稿していたが、作家業だけで食べていけるようになるまでには時間がかかっていた。

賞を取った後は一気に活躍の場が広がり、本が一気に売れるように。けれど、それ以前は地道に原稿を送り、時には落選もしながら、書き続ける日々だったという。

私は「成功する人は一気に頂点まで駆け上がる」というイメージを持っていたが、この展示を見て、その裏には膨大な積み重ねがあるのだと痛感した。

私は最初の1冊を書くのに7か月かかった。だから、他の作家は何か特別な力があるんじゃないかと思っていたけれど、展示品を見て、やっぱり楽して書くことは出来ないんだな、と悟った。

## 執筆と取材、家族と生活を背負って

作品リストを見てさらに衝撃を受けた。ざっと数えても、毎年3冊以上の単行本を出し、さらに新聞や雑誌に詩や随筆を寄稿している。私も寄稿したり取材を受けたりしているからわかるけれど、こういった仕事は決して高額ではない。基本的には、Webメディアに寄稿したり取材されたりするのと単価はあまり変わらない。
（注：時代の違いにより単価が異なる可能性はあるけれど、さすがに昔のことは分からないので割愛）

当時も同じで、作家として生活するには、量をこなすか、大きな印税が見込めるヒット作を出すしかなかったはずだ。

「氷壁」のような国民的ヒット作を出すまで、井上さんは相当な執筆量と取材活動をこなしながら、家族を養っていたのだろう。出版不況が叫ばれる現代とは状況は違えど、“楽ではない”という点では共通しているのかもしれない。

## 散らかった机は創作の証

応接間には、分厚い資料や書籍がぎっしりと並んでいた。研究者の書斎のようで、情報の海に身を浸す生活が想像できる。あれだけのインプットがあれば、次々に新しい作品が生まれるのも納得だ。

また、原稿用紙にモンブランの万年筆で執筆していたというエピソードも展示されていた。正直、かっこいい。

対して私はというと、先生から“永久に貸りている”Macで執筆し、外では100均のスケッチブックに会社からこっそり拝借したボールペンでメモを取っていた（最近は100均でゲルインキのボールペンを買って使っている）。道具の高級感は雲泥の差だけれど、気持ちだけは負けてない（はず)。

スタッフの方によれば、井上さんは書斎で仮眠をとり、その周りに資料や原稿が山積みになっていたらしい。私も原稿執筆期には机の上が散らかり放題になるので、「ああ、文豪も同じなんだ」と妙な安心感を覚えた。

## 文豪の記念館で見た“人間らしい”一面

井上靖記念館は、ただ“文豪の家”を見せる場所ではなかった。そこには、成功の裏にある地道な努力、生活のために書き続けた日々、そして大量のインプットを惜しまない姿勢が詰まっていた。

旭川の観光地といえば自然や動物園が有名だが、もしあなたが文章を書く人なら、この記念館は必ず心に何かを残してくれるはずだ。

私も、早く次の本を出さないとな、と思った。
