# FIREの次は“LLM無職”──生成AIブームが生み出した、新しい生き方

平日午前、渋谷のカフェでMacを開く若者。画面に映るのはSlackではなくChatGPT。失業中かと思いきや、彼の月収は会社員平均を軽く超える──そんな噂が現実味を帯びている。

“LLM無職” と呼ばれる彼らは、「AIに仕事を奪う」のではなく、「AIに仕事を任せ、自分は無職になる」。

生成AIに何万円も注ぎ込み、仕事を任せ、自分はXに日常をポストするだけ。

## LLM無職って何者！？

LLM無職とは、**定職を捨てて〝生成AIにフルコミットする〟ことを覚悟を公言した人**を指す新語だ。生成AIにより数年後に会社に解雇されると予想される現代、そんな時代の変化が訪れる前に退職し、生成AIを用いて一旗揚げようとしている人が中心となって動き出したムーブメントだ。人によって解釈は異なるものの、SNSやブログを概観したところ、以下のような定義で用いられていることが多い。

-

大規模言語モデル（LLM）を日常的に 使いこなし、学習・創作・業務改善などあらゆるアウトプットの主軸に据えている。

-

定職を持たない、もしくは、企業・大学などに籍があっても、実質的なタスクや成果義務を負わず、いわゆる “社内ニート／勉強しない学生” のように組織貢献をほぼ行っていない。

-

肩書や所属を棚上げし、自らを「無職」あるいは同義の自嘲ラベルで公言している。

## 背景：なぜ今、無職でいられるのか

無職でいられる理由は、大きく分けて３つが挙げられる。

第一に、生成AIの性能向上と低価格化が“個人レバレッジ”を桁違いに拡大したことだ。月数万円のAPI課金で、企画・文章・コード・画像編集まで一人で量産できる。これにより、開発から販促まで、すべての仕事を一人で完結することができ、スモールビジネスにおいて人を雇わずにできる仕事量が格段に増加した。

第二に、プラットフォーム経済の成熟だ。X(旧Twitter)やYouTubeでのUGC(ユーザー生成コンテンツ)収益、noteのサブスク、Udemy講座など、アウトプットを即マネタイズする導線が整っていることだ。今や、少し有名になれば、数万円の収入であれば簡単に得られる時代になっている。

第三に、終身雇用と年功序列の揺らぎだ。大企業であっても早期退職募集が常態化し、「会社員＝安全」という神話は崩壊寸前。実際に、米国の大企業は、生成AIにかかる費用を捻出するために社員を解雇している。もはや“職”にしがみつく方がリスキー。──そんな時代認識がLLM無職を後押ししている。

## AIに集う、開拓者たち

LLM無職は、以下のような方法で稼いでいる方が多い。

-

AIライティングクリエイター
書籍や記事を生成AIと共著する人。ちなみに、生成AIの書く文章は微妙なので、生産効率は手動の倍くらいにとどまる。

-

プロンプトコンサルタント
企業や行政向けに業務効率化プロンプトを設計し、導入後も伴走サポートを行う専門家。この分野は結構儲かるらしいが、コネが必要。

-

AIクリエイティブアーティスト
イラストや動画を生成し、広告やエンタメ分野で受託制作を手掛けるクリエイター。

-

SNS依存症
Xに毎日朝から晩まで張り付いて投稿し、インプレッションやサブスク収入で稼ぐ。月に100万円稼ぐ変人もいる。

いずれも「AIを使って社会に価値を届け、その対価で生活する」というスタンスは共通で、無職だからこそ時間を柔軟に使い、学びと挑戦を絶えず循環させている点が印象的だ。

## LLM無職は終身雇用の終焉の前兆!?

かつて「会社員＝安定」だった時代の裏側で、フリーランスは不安定の代名詞とされてきた。自由と称せるのはごく一部で、請負や週40時間の準委任契約などで縛られながらギリギリの生活をしている人も多かった。

しかし2025年、LLMを武器にした無職は“究極の自己投資期間”とも言える。
モデル性能と自動化環境が指数関数的に伸びる現在、「会社員」が普通であるという時代にも終わりが近づいているという指摘も多くの著名人からなされている。

指示をすることができれば、生成AIがあれば全部できる。人間にやってもらう必要がない。このような意識はまだ、生成AIを高度に使いこなす一部の人にとどまっているが、近い未来、この考え方は普及するだろう。

そんな激しい競争と１００年に一度とも言われる荒波を乗り越えるために、LLM無職は“失うものがない”軽さを武器に時代の最前線を走り続けるのだろうか？
