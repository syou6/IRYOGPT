# LLM無職は、どのように稼いでいるのか？

「働かずに生きていけたらいいのに」
――そんな夢のような暮らしを、現実にしてしまう人たちがいる。

彼らは会社勤めをやめ、生成AIを武器に、安定した収入を得ながら自由に暮らしている。

一見すると、仕事らしい仕事はしていない。平日の昼間から国内外のイベントに顔を出し、高価なガジェットを手に入れ、その様子をSNSに投稿する。

フォロワーは増え、広告収入やサブスク契約が舞い込み、「無職なのに稼いでいる」という不思議な肩書きが成立する。

だが、これは単なるラッキーや偶然ではない。

そこには“案件を探さなくても向こうからやってくる仕組み”と、“無職を名乗れるほどの熱量”が隠れている。

## X発信型

最も目立つのは、Xを拠点にした発信型だ。

X発信型のLLM無職は、ただAIの話を垂れ流すわけではない。海外の大型イベントに参加し、高価なガジェットを買い、目を引く活動を積極的に行う。

その一つひとつを「こんな場所に行きました」「こんなものを買いました」と投稿してフォロワーを惹きつけ、Xの広告収入や有料サブスク契約につなげる。

特に、国内のエンジニア向けのイベントに行くたびにプレゼンをし、そのプレゼンをきっかけにサブスクユーザーを増やしていく流れは定番だ。WEBメディアやニュースサイトでも同じように取材を受け、サブスクユーザーを増やしている。

驚くべきは、講演や取材を無償で引き受けるケースが多いこと。謝礼を捨ててでも「露出」という資産を優先するのが、このスタイルの特徴だ。結果として「発信が発信を呼び」、案件もフォロワーも加速度的に増えていく。

ただし、Xerと呼ばれているこのやり方で稼げる方は、個人ブランドが既に構築されていて、かつ、個人に特別な魅力のある方が多い。一部の人が100万円を軽々と超える収入を叩き出す一方で、一般のXerは月に10万円はなかなか超えない。

## スキル代行型

派手さでは劣るものの、数としては一番多い。私の知り合いも大抵がこのパターンであり、Xerで有名になってからこちらで仕事を貰って、生活している。

これらの方々は、伝統的なBtoBビジネスを、生成AIで効率化することで、設けている。つまり、受託開発、資料作成など、LLMで効率化できる業務を複数引き受けることで、稼いでいる。

現在においても開発業務は、単価はまだまだ下がっていない。そのため、受注開発にかかる時間を削減でき、稼げる額は従来の倍になっている。

中でも「顧問」契約は安定性と効率性の両面で優れている。週数時間のチャット対応や、必要に応じたツール・文章の納品といった柔軟なサポートで高単価を維持できる。

このモデルの利点は、SNSでの派手な発信が必須ではない点だ。実績や信頼を積み重ね、リピートや紹介で案件をつなげられるため、外向けのブランディングに時間を割けない人にも向いている。

そのため、noteやZenなどでこまめに発信できる、マメなタイプに向いているだろう。

## 美味しい案件の引き寄せ方

X発信型もスキル代行型も、共通するのは「案件が向こうから来る」仕組みを持っていることだ。

多くの人は案件獲得のために営業活動をするけれど、LLM無職は逆に、自分の存在を市場に“見せる”ことに注力する。AI関連の最新情報や、自分の知見を無料で発信し続けることで、フォロワーが数千人に満たなくても企業や個人から声がかかる。

実際、数件の高単価案件を受ければ生活費は十分にまかなえる。ポイントは「見込み客に直接価値を届ける発信」をすること。発信テーマは広すぎず、特定領域に深く踏み込むほど依頼が来やすくなる。フォロワーの数はそこまで重要ではなく、発信している「実績」や、人が多く来るイベントに出席していることが重要のようだ。

案件探しの時間が不要になるため、その分をスキルアップや新しい企画に充てられ、さらに魅力的な仕事が舞い込むという好循環が生まれる。そのため、LLM無職の大半は、朝から晩まで生成AIを触って勉強している。

## 楽そうに見えて、実は趣味で突き詰めている世界

とはいえ、この生活は外から見るほど楽ではない。

本当に価値のある情報はすぐに拡散されるため、独自の発信を続けるには常にアンテナを張り、深掘り調査をする必要がある。その作業は時間も労力もかかり、成果が出る保証もない。

生成AIの情報を追いかけ続けても、得られるものは意外と少ない。普通に会社員として働きながら副業として隠れて受託開発をしていた方が間違いなく稼げるし、そもそも、本当に使える情報なら、すぐに誰かが話題にするため、わざわざ調べるために時間を使う必要はない。

Xのインフルエンサーは、色々な場所に呼ばれ、常に新しい体験や写真を投稿できる立場にある。実際にインフルエンサー業で稼げている人は、記事を常に書き、1日中そのための調査や執筆をしている。

つまりこれは、努力というより、**AIが好きで好きで仕方ない人が、趣味として突き詰めている世界**だ。

表面上は「無職で自由」に見えても、その裏には相当な熱量と時間投資が隠れている。

“LLM無職”は、単なる気まぐれなライフスタイルではなく、情報発信とスキル活用を組み合わせた一種のビジネスモデルだ。見せ方と仕組みさえ構築できれば、これは一過性の流行ではなく、持続可能な働き方になり得るのかもしれない。
