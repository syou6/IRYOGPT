# 「noteを書くこと自体が資産になる」noteを100日間毎日投稿することで私に起きた大きな変化

私はかつて、自分の考えを言葉にすることをどこか恐れていた。

「誰も興味を持たないかもしれない」
「誰が読むのだろうか？」
「徒労に終わるのではないか？」

そんな思いが私をためらわせていた。

私は既にブログ運営は３回失敗している。３回とも、10本くらい投稿しては、ほとんど見てもらえない現実に嫌になり、辞めてしまった。

しかし、ちょうど１００日前、本の出版を1月14日に控えた今年の1月1日、noteを始めることにした。

出版社に宣伝を頑張って欲しいと言われて、１本出したのがきっかけであったけれど、何本か投稿していたら、周りの期待もあり、noteの連続投稿に挑戦することになってしまった。

しかし、その成果はあまりにも大きなものだった。著書のタイトルに「人生が変わった」と書いてあるのに、それが霞んでしまうほど大きな変貌を遂げた。

この100日間で得られたものは、ただ100本の記事が存在するだけではない。

この予想外の成果を、ここで共有したいと思う。

## 文章を書く速度と質の劇的な向上

この挑戦によって、私がまず得た最大の収穫は「スピードと質」の劇的な向上だ。

noteを毎日更新するためには、当然ながら毎日のように新しいテーマを設定する必要がある。それも、他の仕事もたくさんあるため、1時間半〜２時間という限られた時間で文章を書く必要がある。

私はこれに対応するため、大量のメモや日記を用意した。毎日メモを1~2時間おきに書き込み、色々な出来事や、思いついたネタを書いていた。これにより、編集にかかる時間を大きく削減する戦略を取ったのだ。

毎日のこの繰り返しによって、考えを整理し文章化するまでの時間が格段に短縮された。さらに、「人に見せる」というプレッシャーによって、編集能力、つまり、表現の明瞭さや読みやすさも飛躍的に向上した。

結果、現在では何か新しい出来事があった時も、すぐにそれを文章にまとめられるようになった。そのため、今後も、実績になりそうなことは、片っ端から文章に直していきたいと思う。

## 単月10万PVのオウンドメディアの構築

著名な人々の多くは、自らの経験や考えを発信する際、インタビュアーの手を借りたり、社員に記事を書かせたりしていることがほとんどだ。日本の教育では、実用的な文章を書くような練習はほとんどしないため、そのような文章は社会人になってから書く方がほとんどだ。

しかし私は、この100日の連続投稿によって、自分自身の手で体験や考えを直接発信する経験を積むことができた。そして、その結果、10万PVを達成するまでにブログが成長した！

このnoteのおかげで、メディア出演依頼が来た際も、相手側が現実的な企画やインタビュー案を作ってくれる。メディア出演依頼が現在多く舞い込んでいることもあり、この能力はコンテンツを陳腐化させないために、今後ますます重要な武器になるだろう。

これだけ色々なことがあったけれど、私の物語はまだ序章に過ぎないと思っている。実は、メディア出演や講演依頼の数は現状決まっている分だけでもまだまだ増える予定だ。ゴールデンウィークと６月中旬にはヨーロッパ出張もする。

私はこれからも、次々と新しい経験をすることになる。そして、ここで私がどのような行動を取るかによって、自らの人生を更に大きく変えるものだと認識している。

ブログを自ら運営することで、その場に行ったという経験を私が得られる、教訓にできるというだけではなく、それ自体を成果として報告できる。

そうした体験をすべて自らの言葉で伝えられるというのは大きな強みであり、私の人生において重要な資産となるだろう。

## noteが私にもたらしたもの

noteの連続投稿を始める前、私の知名度はほぼ皆無に近かった。

しかし、この100日の挑戦を通じて、多くのメディア取材や講演依頼なども相まって私の社会的知名度は急激に上がった。

特にnoteを毎日投稿し続けたことはライターさんや編集者に特に評価して貰っている。文章を書くのは、簡単に見えて、結構めんどくさい。一般的に、Webメディアの運営をしている会社の若手は、１日１本記事を書くのがノルマだ。だから、私と同じことをしていることになる（ただし私に土日という概念はない）。

私は大量の文章を投稿しているからか、私に取材してくれる方は結構読んでおいてくれる。これは取材してもらう際に、的確な質問が飛んでくるのでよく分かる。

ライターさんが書いてくれる記事は、私が記事にしてもらいたいことを概ね汲んでくれている。私が書くものに困った経験を何度もしているので、書きやすい内容を答えることを自然と心がけられるというのもあるかもしれないけれど。

そんなメディア対応が上手く行っているからか、講演や対談の仕事がたくさんくる上、私の著書がAmazonのプログラミング分野でずっと1位をキープし、全体でも安定して3桁台のランキングを維持している。

## 今後について

noteのおかげで上手く行ったという話を散々書いて来たのに恐縮ではあるけれど、今後は、投稿頻度を週に2～3回に調整する予定だ。

これは決して意欲が衰えたわけではなく、むしろ新たな挑戦に向けて、少し頻度を落とした方が良いと思ったからだ。

現在、多くのメディア露出があり、note以外にも取り組まなくてはならないことが山積している。私の本業は決して「エンジニア系タレント」ではないし、Webライターでもない。あくまでこちらは副産物だ。

もうそろそろ、文章を大量に投稿するような場面ではないかな、と感じている。ある程度ライティングスキルは習得したし、これ以上やってもこの１００日以上の技能の習得は見込めないからだ。ここからはもう少しゆっくりやってもいいと思っている。

実際、ここまで読者の皆様に見せているのは、「私の成長過程」だけであり、そこから生まれた成果物はまだお見せできていない。今はBtoBの仕事ばかりをしているからというのもあるけれど、できるだけ早く、BtoCの仕事を作っていきたいと思っている。

私の物語はここで終わるわけではない。むしろ、ここからが本番だ。これからは頻度を調整しつつも、引き続き価値ある情報をお届けし、さらに多様な分野へ挑戦していきたいと思っている。

いつかまた、この場で新たな成果を報告する日が訪れることを楽しみにしている。その時まで、私はさらなる挑戦を続け、自分自身を磨き続けることを約束する。

これからの私の歩みや挑戦も、温かく見守っていただければ幸いです。
