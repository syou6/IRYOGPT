かつて「ものづくりで食べていく」のは、一部の天才か、資産のある人の特権だった。

作品が売れるまではお金が入らない。材料費も時間も先にかかる。

つまり、完成品を出すまでの間は“無収入の下積み時代”だった。だから昔のクリエイターは、スポンサーやパトロンを必要とした。言い換えれば、ものづくりは「金持ちの道楽」だったのだ。

しかし、今は違う。作っている「途中」そのものに価値が生まれる。動画やSNSで発信しながら、その過程を見せるだけで、収益を得られるようになった。

「プロセスエコノミー」。それは、“過程をコンテンツ化することで、生きていける”という新しい経済の形。

いま、個人のクリエイターが「ものづくりだけで生活する」現実的なチャンスを手にしている。

私自身も、この考え方を意識的に実践している。この記事では、その実践を通して見えてきた「生き方としてのプロセスエコノミー」を説明してみたい。

# 人は“過程”に共感する

世の中の99%の人は、驚くほど優しい。人は、他人を応援したい生き物だ。誰かが何かに挑戦している姿や、少しずつ形になっていく過程を見ると、自然と心を動かされる。これはアニメや漫画にも共通している。主人公が努力して強くなっていく姿に、多くの人が感情移入するのは、「結果」よりも「過程」に共感しているからだ。

だからこそ、あなたが作品を作っていく過程を発信すれば、その道のりを“一緒に体験したい人”が集まる。応援している側も、自分が物語の一部になったような感覚を味わえるのだ。これは「ファンを作る」というより、「仲間を作る」行為に近い。単なる完成品の売買ではなく、「過程の共有」という関係性の経済が生まれる。

とはいえ、「応援してくれる人はいるけれど、どう収益につなげるか分からない」という声も多い。ここで重要なのは、「寄付」ではなく「商品化」だ。人は、ただお金を渡すより、「何かを受け取る」形の方が心理的ハードルが低い。つまり、「過程を見られる権利」「裏側を覗ける体験」などを商品化するのだ。これが、プロセスエコノミーの第一歩になる。

# まず目指すのは月10万円

では、実際に「生活できる」レベルのプロセスエコノミーを築くには、どのくらいの規模を目指せばいいのか？私は、まず「月10万円」を目標にするのが現実的だと考えている。

理由はシンプルで、10万円あれば多くの人にとって家賃を払える金額だからだ。家賃が払える＝露頭に迷う心配がなくなる。つまり、精神的な安定を確保できる。経営の基本は「固定費を減らす」ことと言われる。これは、固定費を払えなくなると破産するからだ。

だから最初に目指すべきは、「固定収入 >= 固定費用」にすること。生活のベースが整えば、創作活動を継続できる。

実際私は、固定収入があることで、スポットの仕事であっても、かなり強気に交渉をすることができ、それによりブランド力を保っている部分もある。

（お金がない→　仕事を選べない→　交渉力が低くなる→　ブランド力が低下する　→　お金がない と、負のループになるため）

# コンテンツの作り方

では、その10万円をどう稼ぐか。方法は大きく2つ――動画か文章だ。私が今、YouTubeで動画を作っているのもこの理由だ。動画は時間がかかるが、視聴者数が圧倒的に多い。

一方、文章コンテンツ（noteやブログ）は、読者層の質が高く、落ち着いたコミュニティを作りやすい。

ここで重要なことは、どちらもスキルが必要であることだ。

noteを書くのも、動画を作るのも、手間がかかるし、最初は上手く書けない。

そこで、私が提案したいのは、100日チャレンジのように「毎日1つのアウトプットをする習慣化」だ。

実際私は、有料noteを100本以上書くなど、膨大な数の練習をした。初めの方の文章と今の文章を比較してみて欲しい。１００本書くと、それだけ上達するということが分かるだろう。

ここで、重要なのは、世の中に公開する、ということだ。腕がある人にお願いして添削してもらうとお金がかかるので、練習する機会がかなり制限されてしまう。一方で、読んだり見たりするだけなら、一般人でもできる。良い悪いは、感性による部分が大きいからだ。

そして、こういう技術というのは、どんなに高い情報教材を買っても、どれだけ周りの人と交流を重ねても、絶対に上手くはならない。だから、何度も繰り返し練習を重ねるしかないのだ。

実際私も、100日チャレンジのday1~30くらいの投稿も、noteの過去の記事も、最近の動画も、消そうと考えたことは凄く多い。こんな品質の低いものを出してもいいのか、周りから馬鹿にされないか、と考えてしまうのは、おそらく誰もが同じだ。

しかし、そういう屍を乗り越えなければ、絶対に上手くならない。まずは１つ出してみて、周りの反応を試しながら、次を作る。どんなことをするにも、それを意識していくといいだろう。

# 過程を生きるという選択

かつての時代は、「完成品」だけが価値だった。だが今は、「作っている途中」も価値になる。挑戦している姿そのものが、人を動かし、物語を生む。

ものづくりとは、作品を作ることだけではない。それを「どう見せるか」「どう関わってもらうか」までを含めた、一連のプロセスのデザインだ。

あなたが今、机に向かって何かを作っているその姿こそが、最高のコンテンツであり、最も人を惹きつける瞬間なのだろう。

プロセスを発信し、共に歩む仲間を増やそう。その先に、“ものづくりだけで生活できる未来”がきっとある。
