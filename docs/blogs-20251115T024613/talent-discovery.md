SNSやYouTubeを開くと、「才能を開花させた人」たちが次々と目に飛び込んでくる。

起業家、投資家、Youtuber、noter、作家、プログラマー

ネット上には、わかりやすい成功例が散らばりすぎている。

「あんな風に、自分も輝けたらいいのに」
「自分にだって、まだ見つかっていない才能があるんじゃないか」。

嫉妬心からか、人には直接言わないものの、どこか、憧れを感じている自分に気づくことがある。

しかし、才能は突然開花するなんてことはない。才能は、自ら見つけ出さないといけないし、見つけ出すことができる。

今回は、その探し方について、実体験をもとに、導いていきたい。

# 「好きなことをやれ」と言われても、わからない

私も、もともとは、なんの取り柄もない大学生だった。コミュ力も高くなくていつも一人だったし、プレゼンも下手。特別なコネもない。奨学金も借りていた。

一般的な「普通」のレースに乗せられ、先生に「就職活動しろ」と言われ、嫌々エントリーシートを書き、内定をもらい、なんとなく会社員になった。

あの頃の私は、本気で退屈していた。いや、将来に絶望していた、という表現の方が正しい。

ネット広告で、「エンジニアになれば誰でも簡単に月収100万円！」という広告を見たので、先生に、「エンジニアになって、月収100万稼ぎたいです！」とか言った。先生は、「君ならすぐにそのくらいもらえるようになるよ」とか言ってくれた。

他にも、学会発表した後の懇談会でも、「君は将来大物になるよ」多くの先生から応援の言葉をもらっていた。

でも、実際は、全然違った。

会社に勤めた際の初任給は、月収22万5000円。同僚はプログラミングは触ったことはなく、会社の先輩や上司は、「オブジェクト指向はオワコンだから覚えなくていい」とか言っていた。JavaScriptはオブジェクト指向言語なのに。

- 給与テーブルを見ても、40代後半で月収50万になれば良い方。
- 会社の研修室で、無味乾燥に入門書をを読み上げる講師。

だから、毎日、を東京湾を眺めながら考えていた。

何か、私に眠る「才能」が、私に自由な翼を授けてほしい。

「私の才能って、どこにあるんだろう？」

しかし、“一般的なアドバイス”は、抽象的だ。

「好きなことをやれ」「情熱を追え」。いや、その「好き」を定義できていないから、困っているのだ。私は、何が得意で、何が続かないのか。それを自分で観測し続けた。外から突然、才能は降ってこない。どうすればいいのだろうか。

# 好きだけじゃダメ、得意だけでもダメ

２ヶ月くらい、会社で毎日窓の外を眺めていると、私は、一つの原則にたどり着いた。

それは

「得意なこと」で、「継続できる程度に嫌いではないこと」を探すことだ。

この条件は、２つ揃って初めて“才能”になる。

逆に「好きだけど全然得意じゃない」ものは、趣味で終わる。
「得意だけど死ぬほど嫌い」なものは、続かず折れる。

才能とは、気合や努力の量ではなく“継続できる斜面の角度”で決まる。一定の傾斜があるから、勝手に上達していく。

そして、結果として他人より上に行く。この“自然勾配”の見つけ方こそが、才能の発見だ。ここでの本質は、「天才性は、後から観測される」ということだ。

あとから振り返ってみたときに、“あの分野に触れたときだけ、やたら加速度がついていた”という瞬間がある。

それが“才能の根”だ。だから「才能」を外側のラベルで探索するのではない。内側の“加速度”を観測する。そしてその条件を、言語化し、特定する。このプロセスは“自己の計測”だ。才能探しとは、心理でも哲学でもなく、科学的な観測行動と定義づけられると予測できる。

# 才能の手応えを感じ取れる瞬間は必ずある

私の場合は、プログラミングと執筆だった。

プログラムを書いていたら、あっという間にオセロを作れるようになった。
ロジックを組むと「結果」が目の前で変わる快感があった。

その後、案件（PoCレベル）を頼まれた際、Next.jsは使ったことがなかったけれど、数日で作って提出できた。それで、あっさりと受託開発で月収100万円に届いた。これは私が「プログラミングを続けられる角度」の証明だった。

執筆も同じ。７ヶ月の間、毎日、朝から晩まで本を考えていた。会社では退屈な研修室で、ずっと、帰ってから本をどのようなストーリーで書くかを考えていた。

それを飲み会で一部話して笑いを取って検証したこともあった。結果的には、一心不乱にストーリーをたくさん作る練習をしていた。

この２つは、まるで「ログインボーナス０円」「課金要素なし」でも続いた。“やらないと苦しい”のではなく、“やっていると勝手に進む”。その感覚を、私は横軸を時間、縦軸を習熟度とした時に線形的に習熟度が伸びていく様から、「才能の勾配」と呼んでいる。

才能は、必ず"発掘"ことができる。

こうした現象を、なぜ“観測”できたか？

理由はとてもシンプルで、私にも好き嫌いがあったからだ。

私は受験やテストのための勉強は集中できない。だが、ゲームはストラテジ系なら集中できる。（プレイヤースキルが必要で練習が必要な、スポーツのようなゲームはすぐに飽きる）。

つまり、私は「システムを理解して勝ち筋を見つける領域」に強く勾配が出る。そして、プログラミングと執筆は、その性質に完全にハマった。好きなところがピタッと合うようなものがあれば、夢中になってできる。そして、夢中になっていることのうち、仕事になりそうなものに、膨大な時間を割いたことで、芽が出た。

だから、才能は“自分の内部を計測することで、徐々に輪郭が浮かび上がってくる”。

決して、天から降ってくるものではない。自分自身を観察し、得意なことや夢中になれることを一つ一つ見つけ出し、その条件を認識し、それに合うものを全力で進める。

この地味な観測行動が、究極的には、才能の発見を一番近道にするはずだ。
