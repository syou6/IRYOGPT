# Zettelkastenとは？レポートが書けない原因を解決する最強のアイデア整理術

「レポートのテーマは決まっているのに、文章が進まない…。」
ーーレポートを書こうと机に向かった瞬間、頭が真っ白になる。何から書けばいいか分からない。必要な情報がどこにあるのか分からない。

そんな状況には何度も陥った。締め切り間際にレポートの書き方の本を買いに行っても、ブレストなどのアイデアの整理術の説明ばかり出てきて、何の役にも立たない。

しかし、その根本的な原因は、情報を体系的に整理できていないことにあった。そんな悩みをスッキリ解消するのが、アイデア管理術「Zettelkasten（ツェッテルカステン）」だった。

## Zettelkastenとは？

Zettelkasten（ツェッテルカステン）は、ドイツ語で「メモ箱」や「ノートの箱」を意味し、情報管理や知識創造を効果的に行うためのメモ整理法・知識管理システムのことを指す。ドイツの社会学者ニクラス・ルーマン（Niklas Luhmann）が用いていたことで特に知られている。

Zettelkastenでは、アイデアや情報をカードや小さな紙片（Zettel）に記録し、それらをテーマやキーワードごとに箱（Kasten）に収納する。各カードは独立した知識単位として扱われ、相互に関連づけられる。

私はGoogleドキュメントにメモを書いて、それをカテゴリーごとにフォルダに分けて保存している。（このZettelkasten自体、５０年以上前の、パソコンがない時代の考え方であり、現在はデジタルが主流）。

## Zettelkastenで用いるメモの種類

Zettelkastenでは、メモを明確な役割ごとに分類することで知識を効率的に管理する。以下の3種類のメモを用いる。

### 1. 一時的メモ (Fleeting Notes)

-

**特徴**

-

思いついたことをその場でさっと書き留めたメモ。

-

あとで見返した時に意味がわかるよう、最低限の情報を書いておく。

-

手帳、付箋、スマホのメモアプリなどに一時的に記録する。

-

**例**

-

「ノートを取ることは思考を外在化すること」(p.25)

-

「紙よりもデジタルの方が検索性高い?」

### 2. 文献メモ:(Literature Notes)

-

**特徴**

-

本や論文、記事などを読んだ際、その内容を自分の言葉で短くまとめたメモ。

-

出典（タイトル、著者名、ページ番号）を記録しておく。

-

**例**

-

『take notes!』(p.47)：ノートを取ることで記憶力が向上し、抽象的な概念を具体化して理解しやすくなる。

### 3. 永久メモ(Permanent Notes)

-

**特徴**

-

一つのメモが一つのテーマやアイデアを完結して扱う（アトミックノート）。

-

自分の考えを明確に記述し、関連する他の永続的メモとリンクで繋げる。

-

通常は整理されたZettelkastenのシステム内に保管される。

-

一時的メモをもとに、２〜３行程度のメモを作成する。

-

**例**

-

ノートを取ることは、思考を明確化し、記憶を補強する役割を果たす。そのため、記録したノートを適切に整理・リンクさせることが重要となる。

### 4. インデックスノート（Index Note）

-

特徴

-

テーマやキーワードごとに作成される目次的ノート。

-

関連する永続的メモへのリンク（番号やタイトル）を一覧でまとめる。

-

永続的メモが増えるたびに追記・更新される。

-

例

-

「ノート術」インデックス

-

「ノートの外在化と記憶への効果」

-

「ノート整理と知識の体系化」

（この２つは永久メモがあるものと仮定）

### 5. 構造ノート（Structure Note）

### 特徴

-

あるテーマに関するメモを構造化・体系化する。

-

知識の関連性や階層性を明確に示す。

-

Zettelkastenの中のリンク集として機能する。

-

例

-

『効果的なノート術』構造ノート：

-

ノートの目的：

-

記憶の定着

-

思考の外在化と明確化

-

ノートの管理法：

-

紙ノートとデジタルノートの比較

-

永久メモの作成と整理法

（それぞれ永久メモがあるものと仮定）

## Zettelkastenの運用プロセス

### Step 1. メモの収集

最初は「フリーティングノート（Fleeting Notes）」を集める段階です。

-

思いついたアイデア、読書中に気になった箇所、会話中の発見などを書き留める。

-

メモ帳やスマートフォンなどに素早くメモを残す。

### Step 2. メモの理解・整理

次はフリーティングノートの内容を整理し、「文献メモ（Literature Notes）」を作る。

-

本や記事、論文などを読み、そこから得た情報を自分の言葉で要約する。

-

出典（著者、タイトル、ページ番号など）を記録する。

-

フリーティングノートの中で価値の低いメモを捨て、価値が高いものを整理する。

### Step 3. 永続的メモの作成

この段階で文献メモや整理されたフリーティングノートをもとに、最終的に「永続的メモ（Permanent Notes）」を作成する。

-

各メモは **1つのアイデアやテーマ** を扱う（アトミック）。

-

完結した文章で書き、明確なメッセージや意見を記入する。

-

他の永続的メモとリンクやタグをつけて、関連付けする。

### Step 4. メモの構造化

永続的メモが増えてきたら、「構造ノート（Structure Note）」や「インデックスノート（Index Note）」を作成し、知識を体系的に構造化する。

-

特定のテーマや関心領域ごとに、関連する永続的メモを整理しまとめる。

-

全体の理解や関係性が深まり、新しい気づきやアイデアが生まれる。

-

文章をフォルダ構造に分けるのと同じ

###  Step 5. メモの活用

作成したZettelkastenを実際に活用する。

-

論文、レポート、書籍、ブログなどを書く際、Zettelkasten内のノートを検索・引用する。

-

作成したメモが再利用可能な知識ベースとなり、繰り返し活用する。

-

新たなメモを作成することで、知識体系が常に進化・成長していく。

## おわりに

今回Zettelkastenの使い方についてまとめるにあたり、２年前に読んだ「[Take notes!](https://amzn.to/3XLhN1H)」をもう一度読み直した。この本は学者が書いたからか、正直抽象的で、どう活用するのかが分かりづらかった。実際、これについてまともに、適切に使っている人すら見つからなかった。

Zettelkastenを行う上で一番の問題点は、めんどくさいことだ。Zettelkastenの生みの親であるニクラス・ルーマン氏は、生涯にわたって約90,000枚ものカードを作成したと言われている。日本語の文字数に表せば、600万文字。書籍にして60冊分くらいだろう。

Zettelkastenでは定期的にノートを見直してリンクを再構築するべきと書いてある。しかし、そもそも2~3行の情報を整理することは非常に困難。そのため、個人的には、これをそのまま適応するのは難しいのではないかと思っている。

もし、この教科書通りのやり方をしている人がいたら教えて欲しい。

私は、メモを作るにあたり、一番難しいのは一時メモの作り方だと思っている（残りの工程は生成AIでもできるため）。そのため、一時メモを作成するためのノウハウをたくさん蓄積した。

私は元々はZettelkastenをベースにしたやり方をしていたが、あまりの手間と時代の違いから、少しずつ独自のやり方を作ってきたという経緯がある。

このZettelkastenという知識管理術をベースに、読者の皆様も、アイデアや情報が体系的に整理する方法を見つけてみて欲しい。
