# 「なぜこの2人が？」三宅香帆さんと私の対談が生まれた理由

「なぜ、この2人の組み合わせなのか？」

SNSでそんな声を見たとき、私は思わず笑ってしまった。実際、収録が決まったときも同じ感覚を抱いていたからだ。ソフトウェア開発やエンジニアとして活動する私と、文学で活動する三宅さん。同じ番組に並ぶのは、確かに妙な組み合わせに見える。

肩書きは同じ“作家”でも、歩んできた道はまるで違う。だから、なぜ同じ番組で対談することになったのかと、不思議に思われても仕方がない。

でも、だからこそ成立した対話があった。普段なら絶対に交わらないジャンルがぶつかり合い、そこから新しい気づきが生まれた。

作家の出自、言葉の扱い方、AIへの距離感——すべてが対照的でありながら、不思議な共鳴があった。

この「交わらないはずの線」が交わったとき、どんな図像が浮かび上がったのだろうか。

## 私は作家に会ったことがなかった

今回の収録はテレ東の「AIアカデミー」。
事前ミーティングでプロデューサーから「会いたい人いますか？」と聞かれたのがきっかけだった。

AI界隈って、有名な人は大体会ったし、残りは「驚き屋」か「情報商材屋」が多い。正直、ブランド毀損リスクが高すぎる。だから特にいなかったし、どうしようかと悩んでいた。

そんな私を見ていたプロデューサーが、「AIじゃなくてもいいですよ」言ったことで、ふと思いついたのが三宅さんだった。

私は作家になりたくて本を書いたわけではなく、周囲から書くことを強く勧められて始めた。だから、私が作家を志したのは、本の書き始めてから2ヶ月後だ。そのため、「作家になるために本を書き、夢を叶えてベストセラー作家になった人」という存在に、むしろ興味があった。

メディア関係の知人が「『なぜ働いていると本を読めなくなるのか』の著者の取材に行った（同行した？）」という話をしていて、印象に残っていたのもある。
「じゃあ三宅さんでお願いします」と言ったら、本当に対談が決まってしまった。テレ東、すごい。

## 三宅さん、AI使いまくってて私困惑

そして当日。
一番驚いたのは、三宅さんがAIをめちゃくちゃ使っていたことだ。
ChatGPT、Claude、Manus…複数の生成AIをガッツリ使って、しかも作家活動に組み込んでいる。

私は収録前、生成AIを使っていない相手に話すことを想定し、「AIと一緒に水着屋に言ったら店員に着せ替え人形にされて7万円くらい払った」みたいなしょうもない話をしようと思っていた（実際に話したけど、まあ当然カットされた）。

でも、作家という畑違いの立場から、きちんとAIを理解し、活用している姿を見て「これは説明することがないな」と思った。プロンプトの書き方も色々話そうとしたけれど、書き方がかなり似ていたし、文章を書くための生成AIの使い方について説明することがなかった。むしろ私が学ばされる感じだった。

普通、異業種の人は「AI？なんかよく分からないけど怖いやつでしょ？」という反応をしがちだ。でも三宅さんはむしろ前のめりに試していて、その柔軟さにやっぱ凄いな、と感心した。

## 『ノイズ』という言葉に込められた意味

収録前に三宅さんの著書『なぜ働いていると本が読めなくなるのか』を読んでいた。
感想を聞かれたら「私はノイズの世界に生きてるので、わりと本は読めますよ」と答えようと思っていたけど、その機会はなかった。

[https://amzn.to/4fL86Ii](https://amzn.to/4fL86Ii)

ただ読みながら気づいたのは、自分が経営者の自伝や専門書は山ほど読んできたけど、文芸系の本はほとんど読んでいないという事実だ。
一方で三宅さんの話し方や語彙には、積み重ねてきた教養が滲み出ていた。
ああ、これが「文化人」なのか、と。

ちなみに本の中で繰り返し出てくる「ノイズ」という言葉は、ただの雑音じゃなくて「読書を邪魔する余計な刺激や情報」のこと。インターネットはノイズを排除して効率化するけど、読書はあえてノイズを含むからこそ偶然の出会いがある。その余裕を労働が奪う、という話だ。なるほど、と思った。

私は学生時代に真面目に勉強してこなかったので、教養系は正直かなり弱い。何かきっかけがないと勉強しないタイプだ。だから余計に、教養が自然ににじみ出る人に憧れを覚えた。

## 収録以上に、学びの多い時間だった

振り返ると、この収録は「仕事」以上に私にとって刺激的な体験だった。普段なら交わらない領域の人と出会い、AIの話をし、互いの本を読んで感想を持ち寄る。そんな時間は滅多にない。

何より面白かったのは、作家である三宅さんが思った以上にAIを使いこなしていたことだ。三宅さんは文学の土俵にいながら最新の生成AIを実践的に取り入れ、創作の一部として消化している。その姿勢は私にとって衝撃だった。
