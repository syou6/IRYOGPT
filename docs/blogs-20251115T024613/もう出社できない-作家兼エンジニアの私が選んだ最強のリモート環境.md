# 【もう出社できない！】作家兼エンジニアの私が選んだ最強のリモート環境

気付けばスマホを手に取り、SNSや動画に吸い込まれる。

リモートワークはサボりがちだ。通勤も上司の目もない環境は、自由であると同時に、自制心を試される場でもある。

座っていて疲れて、ベッドに寝転がり、スマホを見ていると、いつの間にか夕方。何をしても続かないのは、自分の怠け癖だと諦めかけていた。

以前の私はそれを「やる気がないせい」と思い込み、自己嫌悪に陥っていた。

しかし原因は単純だった。家具と環境を整えれば、自然と集中できるようになる。そんな、私の、リモート環境を紹介しよう。

## 足元から整えるリモート環境

リモートワークを快適にする最大の要因は、間違いなく家具だ。特に椅子とデスク。この2つは、ケチった瞬間に腰痛や肩こりが襲ってくる。

まず椅子は、[Dowinxのゲーミングチェア](https://hb.afl.rakuten.co.jp/ichiba/4b4a01cf.9bd0b1de.4b4a01d0.8bb0c74e/_RTLink114915?pc=https%3A%2F%2Fitem.rakuten.co.jp%2Frt-dowinx%2F6658%2F&link_type=hybrid_url&ut=eyJwYWdlIjoiaXRlbSIsInR5cGUiOiJoeWJyaWRfdXJsIiwic2l6ZSI6IjI0MHgyNDAiLCJuYW0iOjEsIm5hbXAiOiJyaWdodCIsImNvbSI6MSwiY29tcCI6ImRvd24iLCJwcmljZSI6MSwiYm9yIjoxLCJjb2wiOjEsImJidG4iOjEsInByb2QiOjAsImFtcCI6ZmFsc2V9)を愛用。ゲーミングチェアは長時間座っても疲れにくく、リクライニングしながら構想を練ることもできる。

デスクはニトリで買った白いシンプルなモデル（約2万円）。１年前に買ったからもう多分見つからない。ただ、狭い部屋にはいるような奥行きと幅だったから買っただけだ。私は喫茶店で仕事をする際に、椅子の質を見ることは多いけれど、デスクは高さと広さくらいしか見ないから、多分それでいいんじゃないかと思う。

さらに秘密兵器が[フットマッサージャー](https://amzn.to/45MKxLH)だ。これがあるだけで午後の集中力が段違いに変わる。足先が温まると全身の血流が良くなり、長時間の執筆やコーディングでも疲れが蓄積しにくい。

また、ベッドは良いものを買ったほうが良い。私は、平日の閑散としたニトリで店員さんを呼び出し、許可をもらった上で全種類マットレスの上に寝転がり、すべての寝心地を試したうえで、一番合っているものを選んだ。マットレスに５万円、フレームに５万円払った。

コーヒーメーカーやアロマ？そんなものはない。むしろ「飲みたくなったら休憩のサイン」と割り切り、外に出てカフェで飲むようにしている。冷蔵庫すら置いていない我が家に、そんな「贅沢品」は存在しない。

## 最小構成で最大効率を生むツール群

仕事で欠かせないのは、開発用の**Visual Studio Code**とターミナル、執筆用の**Googleドキュメント**。この2つさえあれば、ほぼすべての作業が完結する。

メモはGoogle ドキュメントの他には、紙のメモも使う。100均で買ったキャンバス地のノートに、100均のゲルインキボールペンで書く。理由はシンプルで、「なくしても困らないから」。高級なノートやペンは所有欲は満たされるかもしれないけれど、どこかに置き忘れた瞬間に後悔が押し寄せる。だいたい、価値があるのは私が書いた文章や設計書であり、市販品のペンなんて、１００円だろうと１万円だろうと、それと比べたら価値の違いは些細なものだ。

通信環境は、自宅では光回線を利用。外出先では、常にスマホのテザリングを使用し、公共Wi-Fiには極力接続しない。セキュリティと安定性の両方を守るため、という体裁だけど、実際には、Wifiの接続設定をするのは面倒だからだ。スマホは2台持ちで、一台がだめならもう一台を使うというワークフローだ。

## 生産性を保つための5つの習慣

快適な家具とツールを揃えても、生活習慣が崩れれば生産性は落ちる。そこで私が意識している習慣は以下の通り。

-

休憩はスマホではなく、外の空気を吸いに出る。

-

家とカフェを交互に移動しながら作業する。場合によってはコワーキングも挟む。

-

整体やパーソナルジムに通う。最近は整体に週6ペース。ただ、２か月続けてだいぶ良くなってきたので、そろそろ半分に減らす予定。

-

朝からコーヒーを飲まない。カフェインの過剰摂取を避けるため、15時までは控える。

-

食事はほぼ外食。外に出る口実にもなり、気分転換になる。

この「適度に外に出る」習慣が、心の健康にも大きく貢献している。家にこもりきりだと、どれだけ理想的なデスク環境でも心が摩耗してしまうのだ。

## 本当の改善ポイント

リモート環境の話をすると、多くの人は椅子やデスク、最新ガジェットに注目しがちだ。もちろんそれらも重要だが、実際のところ、道具の差が生産性を劇的に変えることは少ない。

本当に大きく変わるのは、身体のパフォーマンスだ。頭がぼんやりしたままでは、高級チェアに座っても、最新PCを使っても、いい仕事はできない。だからこそ、まずは睡眠をしっかりとること。そして適度な運動を続けることだ。もし運動が面倒なら、整体に通うのも立派な選択肢になる。

家具やツールは「環境の快適さ」を整えるものだが、睡眠と運動は「自分自身の性能」を底上げしてくれる。

リモートワークを本当に整備するべきなのは、道具ではなく、自分自身なのだ。
