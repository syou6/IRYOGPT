# 「やる気」に意志はいらない！私の人生を変えた“行動思考”

「努力は裏切らない」
「成功する人は感情を抑えてコツコツ努力している」
――世の中には、そんな美しい言葉があふれている。

やりたくないことにも耐え、計画通りに自分を律し続けた人間だけが夢を叶えられる。まるで“感情”はノイズで、“理性”に従うことこそが正義であるかのように。

でも、私はこの常識にずっと違和感を抱いてきた。実際、どれだけ理性的な計画を立てても、気持ちが動かないと一歩も前に進めないことが多い。逆に、「やるべきじゃない」と思いながらも、気づいたら無駄なことに時間を使っていた経験は何度もある。

では、なぜ理性だけでは動けないのか？

なぜ、"完璧な計画"は成功しないのか？

そんな疑問を考え尽くし、辿り着いた“行動思考”の秘密について、解き明かしてみたい。

## ふたつの自分――理性と感情

「こうすれば良い」「今これをやるべきだ」
――理性はいつも正しい答えを持っている。「やるべきことリスト」を作って、そこに書いたことを淡々と消化すれば人生は上手くいく。そう信じがちだ。

だが、現実はそんなに単純ではない。

実際に手を動かすとき、心の中で“もう一人の自分”が顔を出す。どんなに合理的に考えても、「今日はなんとなくやる気が出ない」「気分が乗らない」といった感情が、理性の指示を簡単に無効化してしまう。

「今日この仕事を終わらせた方がいい」と分かっていながらも、なんとなく先延ばししてしまったり、「行かなくてもいい飲み会」に流されて無駄な時間を過ごしてしまうのは、頭の中で理性と感情が分離しているからだ。

私たちは“考える自分（理性）”と“感じる自分（感情）”が常に共存している。そしてこの2つはしばしば対立する。理性はゴールに向かって最短距離を提案するが、感情は時にブレーキをかけ、時にハンドルを奪う。つまり、「正しい行動」を実行するかどうかの鍵は、理性と感情の力関係にあるのだ。

## 行動の主導権を握るのは感情

多くの人は、「理性でしっかり計画を立てて、あとは自分を律するだけ」と考える。しかし、どれだけ完璧な計画でも、実際に動くのは“感情が納得した時”だけだ。

過去を振り返っても、「絶対にやった方が良い」と分かっていたタスクを後回しにし続けたり、逆に「行かない方が良い」と理性的には思っていた所に足を運んでしまったりした経験が山ほどある。学習や運動といった良い生活習慣はなかなか身につかず、寝転がってYoutubeを見たり飲みに行ったら4次会まで居て気づいたら3時、みたいなことが度々起きる。

それは、行動の主導権が感情にある証拠だ。

結局のところ、人は「やる気が出たから動いた」のであって、「todoリストを書いたからやる気が出た」のではない。

それは、結局「行動の主導権は感情にある」という事実を物語っている。

人は「やる気が出たから動いた」のであって、「todoリストを書いたからできた」わけじゃない。どんなに理想的な計画を立てても、それに心が伴わなければ、実際には一歩も動けない。逆に、どんなに無意味だと思っていても、感情が動けば体は勝手にそちらへ流れていく。

## 感情を軸に“行動”を設計する

理性が正しい、感情はノイズ――そんな思い込みはもうやめよう。むしろ「感情こそが行動のエンジン」であると捉え、うまく活用することが大切だ。

私自身、「感情が強く動いたとき」にこそ、本気で行動できた経験がいくつもある。

たとえば、フリマアプリで理不尽なクレーマーに出会い、ものすごくイラッとした。その瞬間、「絶対に損をしたくない！」という気持ちが湧いてきて、人生で一番集中して法律の本を読み込み、最終的には相手の要求を切り捨てた――そんなことがあった。

また、毎日提出する日報作成が面倒で仕方なかったとき、「どうにかして楽したい」「バレずにサボりたい」という感情が爆発。そこから何百時間もかけて生成AIを使いこなし、pdfをぶん投げたら自動で作られる仕組みを作り上げてしまった。

「正しいからやる」ではなく、「強い感情が湧いたからこそ、圧倒的な行動力が生まれる」ことを、何度も体感している。

つまり、無理に感情を押し殺すのではなく、「今どんな感情が動いているのか」を観察し、それを行動の起爆剤にする。イラっとした、不満を感じた、ワクワクした――どんな感情も原動力になりうる。

そして、その感情に理性を組み合わせ、「じゃあ具体的にどう行動すればいいか？」「そのエネルギーをどこにぶつけ、出口をどこにもっていくか」「結果としてどのようなことを得られるか」と考えることで、自分なりの納得感のあるアクションが生まれる。

## 行動と結果の正しい関係

「努力は裏切らない」という言葉の裏側には、しばしば「感情を抑え、理性に従え」という無意識の前提が隠れている。

しかし本当の意味で行動を続けられる人は、自分の感情を無視したり押し殺したりしているわけではない。むしろ、感情というエネルギーをうまく使いこなし、理性と掛け合わせて“行動しやすい環境”を作っている。

結局、人の行動を突き動かすのは「感情の波」だ。理性だけで自分をコントロールしようとするのではなく、「どうすれば自分の中に行動のエネルギーとなる感情の波を起こせるか」を意識してみる。そして、その波が起きた瞬間をキャッチし、迷わず行動に移す――この2つの意識があれば、努力はずっと楽になる。

例えば、ストレスを感じた時にそれをあえて徹底して言語化する、影響されたい人と積極的に関わるなどをすることで、感情の波を自分で作り出せる。

そして、その波が高まったときには、「今だ！」と一気に乗ってみる。

無理に感情を抑えるのではなく、“波を起こして波に乗る”こと。これこそが、無理なく動き続けるためのコツだと思う。
