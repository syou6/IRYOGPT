# 【サボり×ミニマリスト】ミニマリストの本を読んだ。どこか違和感を感じた

片付けたくない、動きたくない、でも部屋は綺麗に保ちたい
――こんな矛盾した願望を抱えているのは私だけじゃないはずだ。

片付けや掃除ほど面倒なものはないけれど、散らかった空間で落ち着く人はあまりいない。私自身、「最小限の手間」で「最大限の快適さ」を手に入れたいと常々考えている。

そんな折、『ぼくたちに、もうモノは必要ない。』というミニマリストの本を手に取った。読み始めた瞬間、どこか自分に似た“サボりの精神”を感じ、共感が芽生えた。

しかし、ページをめくるにつれて、私の生活とは全く異なる著者の極端な生き方に、徐々に違和感を覚え始めた。

## 学べること：モノを減らすと人生が楽になる理由

モノを減らすことの最大のメリットは、「管理の手間がなくなる」ことだ。部屋が片付く、余計な買い物をしなくなるというのは多くの人が知っているけれど、本書はさらに一歩踏み込んでいる。

著者曰く、人がモノを持つ最大の理由は**「自分の価値を他人に伝えるため」**だという。本当は内面を伝えるのが難しいから、外見的に分かりやすくモノで表現してしまうのだ。しかし、初めからモノに頼らなければ、自分自身と他者との関係性はもっとシンプルでストレスフリーになると説く。

ここまでは私も納得できるし、実際に実践している。ただ、著者と私は決定的に異なる点があった。

著者は食器や調理器具のみを残し、デスクやベッド、本棚などの家具を一切持っていない。一方私は全く逆で、21食すべてを外食で済ませることで、食器や調理器具を手放したのだ（もちろん家具は持っている）。

どちらもモノを減らすという目的は同じだけれど、アプローチが大きく異なり、そこに非常に興味深さを感じた。

## 著者は病んでいるのではないか？

しかし、本書を読みながら徐々に疑問が湧いてきた。著者は35歳独身の男性。最近、中目黒の家賃8万7千円のアパートから最近品川区の家賃6万7千円のアパートに引っ越したらしい。収入もあまり多くなく、貯金は引越しに全て消えてしまったという。現在の著者の楽しみは「図書館で本を読む、公園を散歩する、自炊する」ということだけだ。

正直、この人は「足るを知る」というレベルを超えて、本人も語っているように、まるで修行僧のような生活を送っている。

**ーーなんのために、生きているんだろう？**

ミニマリストというよりも、生活そのものが質素すぎて、何を楽しみに生きているのか心配になった。そもそも、貯金までミニマリストにする必要はないのに、と思ってしまう。

## 経験までミニマリストになる必要はあるのか？

著者は手取り10万円でも生きていけると主張し、テレビやSNSに映る華やかな生活を目指さないと言う。それ自体は間違っていないと思うけれど、私は経験や楽しみまでミニマルにすることには疑問を感じる。人生の喜びや充実感というのは、経験や交流、あるいは新しい挑戦から生まれるものではないか。

実際、私は最近屋内スカイダイビングを体験してきた。風だけで空を飛ぶという新しい経験は非常に楽しく、心が豊かになった気がした。こうした体験や新しい挑戦をするためには、時にはお金やモノを増やす必要があるのではないだろうか。

モノを減らすことは良いことだと思うけれど、経験や知識、楽しみまでも減らしてしまうのは逆効果だと思う。

## おわりに

『ぼくたちに、もうモノは必要ない。』を読んで感じたのは、**「適度なミニマリズム」と「豊かな経験」のバランスこそが人生を楽しく生きる鍵だということ**だ。

モノを減らして管理のストレスをなくすのは素晴らしいけれど、それが目的となってしまい、自らの人生の喜びや欲望、可能性を狭めるのは違うと思う。

人生とは、自分の欲望や動機と上手く付き合っていくことにある。適度に遊び、楽しみ、時には新しいことに挑戦する。そんな不確実性を受け入れながらもゆっくりと前に進んでいく必要がある。

ーーその先に、自分だけの本当の豊かさが待っているのではないか。
