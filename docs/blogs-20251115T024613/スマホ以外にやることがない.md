# スマホ以外にやることがない

今日もまた、スマホの中で時間が溶けていく。

動画やSNSを眺めている間は、退屈という感覚すら忘れさせてくれる。日常の中で感じる小さなストレスや退屈さが、スマホを開けば一瞬で消える。ほんの数分のつもりが、気がつくと数時間が経っている。

けれど、スマホを置くと、現実の静けさや物足りなさが押し寄せてくる。

気がつけば、スマホ以外のやることが見当たらなくなっていることに少し驚く。でも、それは多分、私だけではないのだろう。

## 現実世界の退屈さ

現実は地味だ。スマホを置いて周囲を見渡せば、そこには昨日とほとんど同じ景色が広がっている。部屋の壁、窓の外の景色、街を歩く人々。何も特別なことは起きていない。音も動きもほとんどない日常の中では、感覚が鈍くなってしまいそうだ。

この穏やかさを退屈と感じるのは自然なことかもしれない。アニメや映画、小説のような刺激的な世界と比べて、毎日は退屈だ。

人間はそもそも刺激を求める生き物だ。何かが起こること、心が動くことを常に望んでいる。だが、現実はそんなに頻繁には刺激を与えてくれない。だからこそ、スマホの中に私たちは夢中になる。

## 毎日やっていること

毎日の生活は、仕事、飲食、そして「名前の付いた娯楽」で埋め尽くされている。映画を観る、本を読む、友達と会う、どこかに遊びに行く、など。

それ以外の時間はどうだろうか? 意識しないまま手に取ってしまうスマホがその隙間をすべて埋めているだろう。

何をするでもない微妙な空白の時間をスマホは上手に埋めてくれる。スマホの中のコンテンツは、「毎日の生活」を誰かが変わりにやってくれているだけだ。

気づけば1日の中で最も触れているものは、スマホの画面かもしれない。

起きては動画を開き、暇な時間もとりあえず動画を流す。夜寝る前も動画に寝かしつけられる。書いていて馬鹿みたいだけど、紛れもなく事実だ。今このnoteを書いている瞬間もYoutubeの曲メドレーを再生している。
（集中力がないから、私は暗記や試験勉強が苦手なんだと思う。）

## スマホの中の「仕事」

ただ、ここで面白いのは、スマホやゲームの中でやっていることは、「仕事」ということだ。ゲームの中では、そのゲームのルールに則って、本気で取り組む人が大半であり、そうしない人は他のユーザーから嫌がられる。アクションゲームでもシミュレーションゲームでも、人は最も賢く、最も優秀であろうと努力する。SNSやニュースのポータルサイトでは、自分に役に立つ情報を一生懸命集めようとする。

私自身もメディアから取材を受ける際に、「仕事や勉強なんてしたくない！」とよく言っているけれど、どう考えても取材を受けるという行為は仕事そのものだ。なにせ、「やりたくない」理由や、他に何をするかなど、取材があるたびに新しい言い回しを考え、自分がいかに怠惰であるかを説明しようとするのだから。私自身、この矛盾に悩み、次回取材が来たときにはどのように説明しようかと困っている。

## スマホは一瞬の快感を大量に摂取させてくれる

スマホがここまで魅力的なのは、ほんの一瞬の快感を大量に提供してくれるからだ。SNSでのいいね、動画の面白いシーン、ゲームのクリア時の爽快感――それらの快感は非常に小さく短い。

しかし、**その「小さな快感」が無限に続くのがスマホの世界だ。**次々と新しい動画や投稿が流れてきて、一瞬ごとに楽しさを感じさせてくれる。

現実世界では、そのような短時間で簡単に得られる快感はほとんど存在しない。日常の小さな喜びは確かにあるが、スマホの中のような強烈な刺激や連続性は持たない。

実際、私もそういうものを供給している側にいると思う。色々な文章をnoteに書いたり、寄稿・監修したりしている。

キラキラしたツイートはいつも探している。でも、そういうものはなかなか見つからない。朝から晩までパソコンの前にいる以上当然かもしれないけれど…。

## スマホ以上に面白いことはそうそうない

「スマホ以外にやることがない」という感覚の裏側には、スマホ以上に面白く手軽に楽しめることが現実にはなかなか存在しないという現実がある。

現実の世界で面白さを得ようとすると、多くの労力や時間が必要になる。外出するにも準備が要る。新しい趣味を始めるにもハードルがある。その一方でスマホは何も準備せず、何の努力も必要なく、ただ指先一つで楽しめる。

実際、スマホ以上の快楽を得られる遊びはいつも探している。中でも一番手軽なものは、**お酒を飲んで判断能力を落としてから子どもがやっている遊びをすること**だ。不健全なことばかりしているような気がするけど、そういうものなのかな。

## おわりに

今日もまた、スマホの中で一日が終わっていく。振り返ると特別なことは何もなかったけれど、退屈を感じる時間もなかったように思う。社内ニートを辞めさせる切り札が「退屈」を感じさせることだと言われているので、ある意味幸せなことかもしれない。

スマホを通じて得られる刺激は心地よく、そして手軽だ。

時間を無駄にしているようで少し罪悪感も覚えるけど、こんなふうに過ごす日々が意外と普通になっているのかもしれない。いや、中学生くらいの頃から何も変わっていない。

スマホが手放せないのは、きっと私だけではないはずだ。皆、同じような小さな楽しみに包まれて、今日を終えているのかもしれない。
