# パワポ作りに虚無感を感じるあなたへ

パワポ作りって、なんであんなに疲れるんだろう…？

学会や講演の資料を作ろうとすると、平気で丸一日が消える。1本仕上げるのに8時間。しかも、一度きりで使い捨てられる紙芝居。そのためだけに一日が溶ける。この虚無感はなかなかのものだ。

でも、もっと壮絶なのは私の周囲のエリート会社員たちだ。外資コンサル、大手IT、メガバンク…どこも「業務の8割がパワポ」「1週間で20本スライド」「会議一発で全部使い捨て」みたいな修羅場話がゴロゴロ転がっている。

アイディアを練って、Excelでグラフをつくり、スライドごとに小数点や図の位置を調整して…。仕上げた渾身の資料も、会議が終わればゴミ箱行き。パワポって本当に“消耗品”なんだと思う。

## AIが“エリートの仕事”を一発で肩代わり

最初はどうせ「AIで作ったパワポなんて無機質で使い物になるわけがない」と思っていた。

でも、試しに使ってみて衝撃を受けた。
私は普段、月に一度くらいしかプレゼン資料を作らない。だから、パワポ作りが日常な中大の後輩を呼び出して、渋谷でskywork.aiを体験してもらった。

後輩君は、「テスト週間なんだけど…」と呟きながらも、「日米関税交渉妥結の概要と、その後の米EUとの妥結や、妥結していない国への米国の対応をまとめたスライドを作成してください」と打ち込んでいた。

結果には驚いた。私なら情報収集から図解まで、丸1日かかるものが、数分で“8割方使えるスライド”になって現れる。しかも発表用のカンペまで自動生成。

実際、真面目で勤勉な後輩君からは、「ほとんど訂正いらない」と太鼓判だった。正直普段ニュースに出ることはあっても見ることはあんまりない私にとっては何のプレゼンなのかよく分からなかったけど、凄く良くできているらしい。

（後輩君からは、「本当に経済学部出身？」と笑われた。テスト週間にテスト勉強してる後輩君とは違って、私は授業中でさえゲームしかしてない大学生活だったから…。）



## PPTXでダウンロードできる!

色々なAI資料ツールを触ってきた中で、skywork.aiが一番凄いと思ったのは「pptxファイルでダウンロードできる」ことだ。

世の中のAI資料作成ツールは「HTMLしか出せません」「編集できない画像です」「PDFしか吐けません」みたいな仕様が多い。それだと“仕事では絶対に使えない”。HTMLをCSSごと直したりなんて、エンジニアでもやりたくない。

だったら最初から自分でパワポ作った方が早い、という壁を超えられなかった。でも、skywork.aiは違う。
「AIに投げる→pptxでDL→PowerPointで開いてちょっとだけ手直し」
この流れが本当に完璧に実現されている。

他のAIツールを知っている人なら、この“pptx直DL”の有り難さは語るまでもない。少し気になるところがあっても、pptxなら即修正できる。
AIに大枠を作らせて、人間が微調整して仕上げる——ようやく現実になった。



## Webページも数分で生産できる

skywork.aiの本気度はパワポだけにとどまらない。 ホームページも一瞬で作ることができる。

「『#100日チャレンジ』著者・大塚あみのオフ会イベント告知ページを作って」と頼むと、プロフィール掲載付きのページが数分で完成した。しかもzipファイルでDLできるから、そのままサーバーに上げて即リリースできる。

正直、今まではオフ会イベントなんてサブスクユーザー限定でしかやってなかったけど、これは普通に募集をかけるときも使えるかもしれない。

エンジニアじゃなくても、数クリックで「イベントページ完成→公開」までたどり着く。
AI生成物＝微妙、みたいな時代は完全に終わったと実感する。
アイディアが浮かんだその日に、パワポもWebページも「自分で何も作らずに」完成する時代が、想像よりもずっと早く来るのかもしれない。

## 仕事観が根本から変わる時代へ

skywork.aiを使ってみて思うのは、単なる作業時間の短縮以上に「人間の仕事観そのものが変わる時代が来た」ということだ。

パワポに人生を溶かしてきたエリートたちの「頭のいい作業」すら、どんどんAIに奪われていく。今まで「人間しかできない」と思っていた仕事が、容赦なく自動化されていく。そのとき、人間にしかできないこと＝ゼロからの発想・問いを立てる力、リアルな場で語る熱量、そういう部分だけが残っていく。
