# 「ファクトチェック」がAIで瞬時に可能に？Xで注目の「@Grok」の実力とは

「@grok ファクトチェック」と一言書くだけで、人工知能の「Grok」が情報の真偽を瞬時に教えてくれる──こんな機能が今、X（旧Twitter）で急速に広まりつつある。

「真実か、嘘か」、疑問に思ったことをすぐに解決できるという手軽さから、幅広いユーザーの心をつかんでいる。

一体、この流行はどのような経緯で広まったのか？実際の効果や評判について気になり、詳しく調べてみた。

## 最近の流行

この数週間、Xユーザー間で急速に広まったトレンドがある。それは、X社が開発した生成AI「Grok」を使って手軽にファクトチェックを行うことだ。

具体的には、怪しい情報や気になる投稿に対して「@Grok ファクトチェック」と一言コメントするだけ。すると、Grokが即座に返信し、その情報の真偽や根拠を示してくれる。
私も実際に試してみた。

手軽さやスピード感がXユーザーに受けており、政治的発言から日常の些細な疑問まで、あらゆる場面でこの機能が使われ始めている。

最初の頃は、思想の強い政治家や陰謀論者、宗教信奉者などが「ファクトチェック」に自身の意見を否定されて怒っていたことで話題になったが、最近はバズったツイートに誰かが何も考えずにやっているような気がする。

## ファクトチェックの本来のやり方

そもそも、ファクトチェックとは本来どのように行うものなのだろうか。従来のファクトチェックは非常に手間がかかる作業だ。

まず、情報の出所を特定し、その情報の正確性を公的文書や専門家の意見、あるいは一次資料に基づいて検証する。その後、複数の独立した情報源から確認を取り、客観性を確保する。そして、最後に明確な結論を出す。

非常に手間暇がかかり、専門家が慎重な調査を要する作業だ。世の中が嘘だらけなのか、真実が相対的なのかは分からないけれど、簡単なものではないだけに、本来のファクトチェックは時間と労力を必要とする。

私の元にもたまに、意見を求めるようなメールが来ることがある。

## Grokは荒っぽい？

この新しいファクトチェック機能の主役であるGrokの性能はどうだろうか。

AIとしての性能自体は非常に高く、会話調でフレンドリーな文体を持つ。この点はChatGPTと比べて、ビジネスっぽさがなくて、YouTubeなどの音声コンテンツの読み上げ原稿には適しているかもしれない。

ただ、文面からは「男子中学生っぽい」軽さや荒っぽさを感じる。Grokの文章を見る度に気になってしまうこの文体であるが、これは仕様として意図して行っているものなのだろうか？

一体、Grokはどの層に向けて作ったのだろうか？

## 下馬評：否定的なコメントで溢れている

こうした人気にもかかわらず、ネット上の評価を見ると、実は否定的なコメントが目立つ。「AIが真実を決めるのは危険だ」「単なる遊び道具」「使い物にならない」など、手厳しい意見も多い。特に、専門的な情報の真偽をAIが簡単に判断できるわけがないという指摘は多く見られる。

さらに、一部ユーザーはこの機能を単なるネタやジョークとして扱っている。

中でも、**「******[Grokにファクトチェックを求める層は、使用者よりもGrokの方がIQが高いので、鵜呑みにした方がマシという残酷な現実](https://x.com/HayakawaHayat/status/1909045960006877689)**」**というツイートは、多くのXユーザーに世間の不条理さを嘆かせた。

しかし、そういう５行以上の文章を読めない層は、Grokが回答した文章は長すぎて読めないので、鵜呑みにすることも出来ないのではないだろうか？

## 「ファクトチェック」はただの便利な合言葉?

個人的には、「ファクトチェック」はとても便利な機能だと思う。ただし、本当の意味でのファクトチェックというよりは、「自分が気になったことをAIに尋ねている」だけにすぎない気もする。

実際に使っている人の多くは、本当に真偽を確認したいというより、新しい機能を使いたいだけ、「コマンドを打ちたい」だけではないだろうか。私自身も同じような理由で今日始めて使ってみた。

実際に使ってみると、良いおもちゃになりそうな気がする。何か良い使い方はないかを今後考えてみたい。
