## 4. 私が大事にしていること

### 1. 効率化（＝手を抜くために全力を尽くす）

「いかにサボるか」「いかに手を抜くか」という観点で物事を考えるのが大好きです。でも、実はこれには大きなエネルギーが要ります。新しいツールや仕組みを試すときは、大抵その導入に手間暇がかかるもの。だからこそ、手を抜くために全力で調べたり、人に話を聞いたり、実際に試したりする。この“遠回りのようで最短ルート”を探すのが私の原動力です。

### 2. コンテンツの創造

私の頭の中には、常にいろいろなアイデアが浮かんでいます。ゲームやツールを作ってみたり、文章や本を通して発信したり。自分の内面にある世界を、現実の形として生み出すことに喜びを感じます。

### 3. 色々な場所へ行くこと

国内外を問わず、思い立ったらすぐに旅へ出かけます。2024年だけでスペイン、スウェーデン、デンマークなど6カ国を訪問しました。旅や遊びを通して得られる新鮮な刺激が、私の創造力とエネルギーの源になっています。世の中の人は無駄な仕事に全力を尽くし時間をたっぷりとかける一方で、旅行は2泊3日とかで手短に効率的に回ろうとします。私は逆のタイプです。仕事は上手く効率化しサボって、遊びに全力を尽くす。十数泊するとか、周りにバカだと思われるくらい派手に羽を伸ばすのが私の流儀です。
