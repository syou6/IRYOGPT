# メール作成から始めるな！初心者向けChatGPTスタートガイド【5000時間使って分かった！】

**「何をChatGPTに任せたらいいかわからない…」**

ニュースでは連日のようにChatGPTが話題になり、「未来を変えるAI」「仕事の効率化に必須！」なんて煽るけれど、具体的にどうすごいのかはよくわからない。

ーーそのようなことを考えている方は多いのではないだろうか。

私は週に２，３回取材を受けたり講演をしたりしているけれど、こういう悩みはどこに行っても聞く。どこに行っても同じことを聞いてくれるので、他社で話していないことをいつも考えている。

しかし、実際にChatGPTを上手く使えない人に聞いてみると分かるのは、**まともに使っていない筆者が書いた、デタラメな記事を真に受けて、難しいところから始めようとしている**、ということだ。

## 「メール作成」は上級者向け？

巷では「ChatGPTはメール作成に便利」とよく言われている。実際、ChatGPTに直接聞いてもそう答える。

しかし、実際に初心者が最初からメール作成に手を出すのは意外と難しい。メールというのは相手との関係性、細かいニュアンス、状況に応じた配慮が求められ、初心者がChatGPTに適切な指示を出すのは案外ハードルが高い。

私はたまに文面を作るときに使うけれど、背景知識を片っ端からChatGPTに教えないと、まともな文章ができない（ブログ記事も同じ）。

だからこそ、初心者がまず取り組むべきなのは、もっとシンプルで具体的なこと。例えば、普段Google検索をするように、何か調べたいことや人にちょっと聞きたいことをChatGPTに任せてみるのが良い。

## 身近なことから始めよう

私がChatGPTを初めて使ったきっかけは、**「宿題をサボりたい」**というごく身近で小さな理由だった。

[https://note.com/amiotsuka/n/nda94c4fa32cf](https://note.com/amiotsuka/n/nda94c4fa32cf)

学生時代、レポート課題がたくさん出てきた。私はレポートを書くのが嫌いだったので、ChatGPTに頼ってみた。すると、300時間くらい練習すると、レポート課題を半自動生成することに成功したのだ！

当時海外のあらゆる論文で提唱されていた、ChatGPTで書かれた文章を検出する方法を掻い潜り、自分で書いたものとして出しても怒られないレベルのものを容易に作る、強力なメソッドができた。

[https://note.com/amiotsuka/n/n504ed2864143](https://note.com/amiotsuka/n/n504ed2864143)

それからは、プログラミングのコード作成を頼んだり、こうしてnoteの執筆を手伝ってもらったりと、気付けばChatGPTを自然に活用するようになっていた。

何気ない日常のちょっとした問題解決から始めることが、ChatGPTを身近に感じられる一番の近道だと思う。

ただし、ChatGPTがどんなに優秀であっても、絶対に忘れてはいけないことがある。

それは、**「ChatGPTはあなたの能力以上のことは絶対にできない」**ということだ。

ChatGPTはあくまであなたの「助手」であり、魔法ではない。あなた自身が理解していることや、できることしかChatGPTには頼めないのだ。

これを忘れると、「なんだ、思ったより使えない」と勘違いしてしまう。例えば、Excelの関数やプログラミングコードの作成も、元々自分でできることを説明すれば簡単にやってくれる。問題は自分でやり方を学習しようとせず、ChatGPTに丸投げしようとするから起きる。

## なぜプロンプト集は役に立たないのか

最近、書店やネットで「ChatGPT究極のプロンプト集」のような本やネット記事を見かけることが増えてきた。

しかし、実際に使ってみると、思ったほど効果が感じられず戸惑う人も多いのではないだろうか。

私はあのようなプロンプト集は使っていない。その理由は簡単で、「プロンプト集」は結局、**「上司に好かれる返答集」や「部下に嫌われない話し方」のような胡散臭い記事と何ら変わらない**からだ。

人間関係がマニュアル通りにはいかないように、ChatGPTもマニュアルだけで完璧には扱えない。

重要なのは、「何をして欲しいのか」を素直にChatGPTに伝えること。Excel関数を作ったり、簡単なプログラムを書くことだって、実は難しくない。ただ丁寧に「こうしてほしい」と説明するだけで、ほぼ確実にChatGPTはその通りに動いてくれる。

ChatGPTが役に立たないと言っている人のプロンプトをよく見てみると、**「上司からされたら最悪の指示」**が多いのも頷けるだろう。

## めんどくさいことは、ChatGPTに全部任せよう

最後に一番重要なポイントを伝えたい。ChatGPTを使う最大のコツは、**「自分ができるけど面倒なこと」**を任せるということだ。

私はアイデアを出すことは得意だけれど、文法の細かなチェックや表現の微調整のような単純作業は嫌いだった。アイデア出しは平日の渋谷をふらふらと歩いているだけでできるお気楽な仕事に対して、文章の調整は机に向かわないといけないからだ。

ChatGPTに校正を任せると、面倒な編集の時間を節約でき、創造的な部分に集中できるようになる。

面倒なことをChatGPTに任せることで、自分が本当にやりたいことにエネルギーを注ぐ。

ーーこれこそが、ChatGPTを使いこなす一番のコツだ。
