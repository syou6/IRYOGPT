# 【リンク集】#100日チャレンジ 全作品リスト

「私のプログラミング学習に付き合って♡」
忘れもしない、2023年10月27日。私は一つのツイートを投稿した。

正直、深く考えずに投稿した。周りが「いいねした数だけ勉強する」という企画をやっていたのを見て、始めただけだ。

周りからの注目を集めようと、1つ目のプログラムとして、オセロを投稿した。オセロにはどんなに少なく見積もっても20時間はかかった。

この時はまだ知らなかった。毎日10時間以上かけ、ソフトウェア工学を勉強しながらプログラミングを学習することになるということをーー。

## ステップ0：プロローグ

大学4年生の春。私はどこにでもいる、怠惰な学生だった。朝10時くらいに起きては遅刻して大学に行き、家でも大学でもスマホやPCでゲームをしたり動画を見たりしては毎日を過ごしていた。

就職活動なんてしていないし、だいたい卒業しても働きたくない。将来を半ば諦めていた。希望を持てなかったのかな。

ある日、授業中に「ChatGPT」を知り、課題を楽に済ませるために活用を試みた。300時間かけて試行錯誤し、当時海外論文で提唱されていたChatGPT検出の回避をできる、[宿題をサボるためのメソッド](https://note.com/amiotsuka/n/n504ed2864143)を生み出した。

ChatGPTをサボる方法を探る中で、プログラミングにも応用できることを発見する。色々なことが起きた後、​毎日1本のアプリを作成し、SNSに投稿する「#100日チャレンジ」を開始した。​
