# 人生に真摯に向き合ったら幸せになれるのか？―虚無感とその先にあるもの

**「自分の人生にちゃんと向き合いなさい」**
――この言葉を受験や就職のたびに繰り返し聞かされてきた。

とても美しく、崇高に響く言葉だ。

しかし、私はずっと疑問に思っていた。そもそも、**「真摯に向き合う」とは一体何をすることなのか？**

具体的な方法は誰も教えてくれなかった。社会が示してきたのはいつも「自己成長」と称する、競争社会で勝つ方法ばかりだった。

ずっと疑問だった、「真摯に向き合う」って、一体何を指しているのだろう？私は改めてその本質を考えることにした。

## 人生に向き合うって？

一般的には「自分の人生と向き合う」とは、自己理解、自己受容、主体的選択、自己成長を指すらしい。

-

**自己理解**: 自分が何を望み、何が好きで嫌いかを知ること

-

**自己受容**: 自分の良い面も悪い面も認めて受け入れること

-

**主体的選択**: 自分で決断し、その結果を引き受けること

-

**自己成長**: 能力や価値観を高め、理想的な自分に近づく努力

しかし実際に社会が私に明確に求めてきたのは「自己成長」のみだった。

就職活動では成長志向が重視され、「ライフワークバランス」という美辞麗句はあれど、具体的な道筋は誰も示さない。むしろ社会は**「周囲のために自己犠牲をしろ」**と無言の圧力をかけてくる。

そのため、私は人生に向き合う本当の意味を見失ってしまった。

## ネットで調べてみると

インターネットで「人生に向き合うメリット」を調べてみると、後悔の少ない人生を送れる、幸福感が高まる、人生の自由度が増えるなど、抽象的で掴みどころのない回答ばかりだった。

私は社会に出た際、「生活のために働くしかない」ことを不本意ながら受け入れて就職を決めた。しかし就職後に待っていたのは人生の収束感や停滞感だった。自分の人生が見えてしまい、夢を持てなくなったのだ。

社会で成功を掴む夢を本気で叶えようとすれば、それはとてつもない努力と熾烈な競争を伴う。エンジニアになろうと思えば、数千時間の学習を求められ、作家になろうと思えば企画を通す難関を越えて、売れ続ける作品を生み出さなければならない。起業家なら、10年後に生き残れるのはわずか1割という現実だ。

私はこの競争に比較的勝ってきたほうだと思う。実際、多くの人が私のブログを見てくれていることが、その証拠だろう。でも競争に勝つことと幸せになることはまったくの別物だ。むしろ、常に何か満たされない気持ちを抱え、noteを含めて、その虚無感を埋めるために色々なことをしている私がいる。

## 私たちが消費に逃げる理由

現代社会には、モノやお金を過剰に消費することで心の隙間を埋めようとする人が多い。

最近は何百万円も推しに貢いだり、下心からお金を貸したりし、殺人事件まで起きるような酷い世の中になった。

そこまでいかなくても、SNSでは、ブランド物や推し活に異常なまでに散財している人の話題が溢れているし、実際に収入が増えるにつれ金遣いが荒くなる人が大半だ。

お金を使うことで、その瞬間は不安や虚無感を一時的に忘れることができる。飲みに行けば一時的に満たされるし、何かを買えばその瞬間だけ気分が良くなる。

しかしそれはまるで自販機の飲み物のようだ。その場の渇きを潤すだけで、根本的な解決にはならない。

なぜ、そんな虚しいことを繰り返してしまうのだろうか？

## 人生と向き合う真の意味

私はようやく気づき始めている。「人生と向き合う」とは、**一つのことへの依存度を下げること**なのかもしれない、と。

しかし、だからといって、完全に満たされるわけではない。私たちは、きっと永久に何か満たされない気持ちを持ち続けることになる。それを受け入れるのは簡単ではない。だから多くの人は、消費や浪費に逃げ道を求めてしまうのだろう。

完璧な対応策など、そもそも存在しないのかもしれない。それでも、複数の居場所や依存先を持つことで、自分自身を守ることができる可能性はある。

**人生と真摯に向き合ったら、本当に幸せになれるのだろうか？**

その問いへの答えは、誰にもわからない。

結果が明らかになるその日まで、私たちは自分の人生を生き続け、問い続けるしかないのだから。
