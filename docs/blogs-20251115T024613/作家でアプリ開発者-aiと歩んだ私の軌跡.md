# 作家でアプリ開発者──AIと歩んだ私の軌跡

こんにちは、作家でアプリ開発者の大塚あみです。

「生成AIに育てられた第一世代」として、AIと共に生き、創り、語る活動をしています。
アプリを作ったり、本を書いたり、講演で全国を回ったり──そんなふうに、“ひとりの人間としての好奇心”と“テクノロジーの可能性”の間を行き来しながら、生きています。

もともとはただの「AIで遊んでた学生」でした。
でも、気づけばAIが私の人生を大きく変えてくれました。
この数年間で、学会発表、受賞、書籍出版、会社設立、そしてメディア出演まで。

まるでジェットコースターのような時間を駆け抜けながら、
「AIと人の新しい関係」を自分自身の生き方で実験してきました。

## いま取り組んでいること

いま私は、自社「合同会社Hundreds」を立ち上げ、
アプリ開発・受託開発・執筆・講演・メディア出演など、多方面で活動しています。

### 🧭 アプリ開発者として

私は現在、「**メモるん**」というメモアプリを開発しています。
“思考をメモし、整理してくれる”というコンセプトで、
誰でも気軽に「自分の頭の中」を整理できるように設計しています。

[https://apps.apple.com/jp/app/%E3%83%A1%E3%83%A2%E3%82%8B%E3%82%93-%E3%83%81%E3%83%A3%E3%83%83%E3%83%88%E3%81%A7%E6%9B%B8%E3%81%91%E3%82%8B-%E3%81%97%E3%82%83%E3%81%B9%E3%82%8C%E3%82%8B-%E6%92%AE%E3%82%8C%E3%82%8B%E3%83%A1%E3%83%A2%E3%82%A2%E3%83%97%E3%83%AA/id6752797808](https://apps.apple.com/jp/app/%E3%83%A1%E3%83%A2%E3%82%8B%E3%82%93-%E3%83%81%E3%83%A3%E3%83%83%E3%83%88%E3%81%A7%E6%9B%B8%E3%81%91%E3%82%8B-%E3%81%97%E3%82%83%E3%81%B9%E3%82%8C%E3%82%8B-%E6%92%AE%E3%82%8C%E3%82%8B%E3%83%A1%E3%83%A2%E3%82%A2%E3%83%97%E3%83%AA/id6752797808)

人は思考を外に出さないと迷子になる。
でも、整理する時間はなかなか取れない。
そんな現代人のために、“書くことそのものを助けるAI”を作りたいと思ったのがきっかけです。
メモるんは、単なるメモアプリではなく、「思考を成長させる道具」です。
個人開発としての挑戦ですが、私はこれを“第二の自分”だと思っています。

将来的には、AIを用いて、私が普段やっていることが全部自動で行われるようにしたいと思っています。

### 🧑‍💻 受託開発業

一方で、企業・大学・教育機関からの依頼を受けて、業務用アプリや学習支援ツールの開発も行っています。
モバイルとサーバーサイドを組み合わせた開発を得意とし、
特に「生成AIを活用した業務効率化」や「教育分野でのAI導入」に力を入れています。

海外企業とのリモート契約も多く、ヨーロッパの教育機関との共同開発も行ってきました。
コードだけでなく、プロダクトの“哲学”や“物語”を設計することを大事にしています。

### ✍️ 執筆活動

執筆は、私のもう一つのライフワークです。
noteでは、「生成AIと生きる」をテーマに、創作・開発・思考の記録を綴っています。
#100日チャレンジ の裏話から、アプリ開発のリアル、個人的な哲学まで、幅広く発信しています。

読者の多くは、AI時代をどう生きるかに悩むクリエイターやエンジニア、そして学生たちです。
彼らにとって、私の発信が“自分の生き方を設計するヒント”になれば嬉しいと思っています。

### 🎤 講演・メディア出演

海外登壇も含め、AIの社会実装と教育の橋渡しを目指しています。

「AIに奪われる側から、AIを操る側へ」

## これから

これからの目標は、「**個人の知性を拡張するアプリ**」を作ることです。

“AIが人間を評価する”のではなく、
“AIが人間の可能性を広げる”ようなツールを作りたい。
メモるんを中心に、感情や習慣、資産、創作など、
人の生き方そのものを記録・分析・可視化するアプリを構想しています。

## 結びに

私がこの2年間で学んだのは、
「人生はデバッグの連続」ということです。

バグを恐れず、仮説を立て、修正を繰り返す。
それこそが、AI時代の「生きる力」だと思っています。

私の物語は、まだ途中です。AIという鏡を通して、これからも“自らの可能性”を見つめ続けたいと思います。
