# 「価値あるアイデアなんてない」と思っているあなたへ

**私には人に語れるような特別な経験や素晴らしい発想はない**
――3年前の私は、そう思い込んでいた。

毎日、朝起きて、大学に行く。それ以外はスマホを見るかゲームをしているかの私に、面白いことが起こるわけがない。特別な経験や素晴らしい発想は、選ばれた人にしか持ち得ない。

しかし、今ではその考えが間違いだったと自信を持って言える。なぜなら私は、日常の小さな出来事に新しい価値を見出し、それを活用する方法を知ったからだ。

ある日、本書を読んだ私は、半信半疑でメモを取り始めた。最初の1年は何も起きなかった。「やっぱり無駄か」、そう感じていたものの、サンクコストからか続けていた。

しかし、1年以上日常を細かく記録を続けてから、私の日常は変わり始めた。ーー平凡だと思っていた出来事が、時間を置いて他の記録と結びつくことで、役に立つ瞬間を何度も経験した。

最初はレポートをサボるための方法ばかりだったけれど、次第に賞を取れるような論文を書いたり、プログラミングに応用したり、更には著書を書くための題材にもなった。

平凡で退屈だと思っていた毎日が、実はアイデアの宝庫だったと気付く瞬間が増えてきた。

なぜ、平凡だと思っていた日常に、そんなに価値を生むようになったのだろう？

## 本の概要

『TAKE NOTES!――メモで、あなただけのアウトプットが自然にできるようになる』は、ドイツの社会学者ニクラス・ルーマンが考案した「ツェッテルカステン」を基礎にしたメモ術を解説し、私たちの情報整理能力や思考の生産性を高める具体的な方法を提示している。

この本の主題である「ツェッテルカステン」とは、アイデアを個別のカード（またはメモ）に書き込み、それらを相互にリンクしていくシステムである。

この手法の革新的な点は、**情報をカテゴリーごとに縦割りで整理するのではなく、個々のアイデア同士を相互に繋げてネットワーク化することで、新しい発想が自然と生まれやすくなること**だ。

著者はツェッテルカステンの活用法として、メモを「走り書き」「文献メモ」「永久保存版メモ」の3種類に分類し、それぞれの特徴や効果的な使い方について具体的に説明している。

-

「走り書き」:一時的なアイデアや思いつきを素早く記録する

-

「文献メモ」:読書や研究から得た知識を記録する

-

「永久保存版メモ」:自分自身の言葉で深く考察し、再構築した情報を保存する

これらを適切に組み合わせることで、情報の整理だけでなく、新しいアイデアを生むことが促進される。

## 「自分の言葉で書く」重要性

本書が特に重要視しているのは、メモを「自分の言葉」で表現するという点だ。

単なる情報のコピペではなく、自らの言語で再解釈することで、情報は記憶に深く定着し、より有効な知識として活用できるようになる。

このプロセスを実践することで、メモが単なる記録から自分自身の思考や理解を深めるツールへと進化し、情報をより効果的に活用できるようになることが学べる。

## 批判点：学者の悪い所が出ている?

本書に対して感じた最大の課題は、**「価値あるアイデアが存在すること」を前提としている**点だ。

著者はアイデアをメモとして保存し、相互に関連付けていくことで新しい洞察を得られると主張している。

しかし、**実際には、何が価値あるアイデアなのかを判断するのは非常に難しい。**

そもそも、何か思いついたアイデアが将来役立つかは予測がつかない。例えば、飲みの席での笑い話は、将来本を書くかもしれない私にとっては例として役に立つかもしれないけれど、多くの人には役に立たないだろう。

私は人からイジられたり批判されたりしたら、すべて情報として記録してある。しかし、この情報を使う機会は非常に限られており、文章にまとめる手間をかけるのは、心理的にも負担をかけるし、そもそも時間の無駄ではないか、と思っている。

著者が示すツェッテルカステンという手法は、明確な目的を持った論文作成やレポート作成などには有効だが、目的が曖昧な状態で運用すると、単に文書管理の手間が増えるだけになりかねないと思う。

さらに時代が変化し、検索性が飛躍的に向上した現代では、**細かいキーワード管理自体がほぼ無意味**になっている。

私自身も、日々の思考をそのままGoogleドキュメントに書き込んでおくだけで十分と感じている。必要な情報は後から検索すれば、何百万文字もあるクラウドの中からでもあっという間に見つけられるため、本書が提示するような複雑な情報管理システムを構築する必要性は低下しているのではないだろうか。

## 思ったこと――現代に合ったメモ術の再考

私の情報管理は非常にシンプルで、Googleドキュメントに思いついたことをそのまま書き出している。Notionのような多機能ツールも試したけれど、動作の重さがストレスになるため、文字だけを扱う場合には適さないと感じた（Googleドライブを普段遣いするため、複数のサービスを利用するのに手間がかかるのを嫌った節もある）。

さらに現代では、思ったことをそのままタイピングし、後からChatGPTのようなツールで要約させる方が効率的だ。推敲しながら書くよりも、まずは考えたことをすべて書き出し、その後で整理した方が思考漏れも少なく速い。私自身、取材対応などでも数ヶ月から数年前の自分の考えを再利用している。世の中の多くの人がその場限りのアイデアで対応するのに対して、私は過去の自分の知識を"同期"し、大量のアイデアを即座に引き出して効率よく対応できている。

本書の提案する方法はすべてが現代に適合するとは限らないけれど、情報をネットワーク化して再利用するという基本コンセプトは今も有効だと思う。

私は常に「いかにサボるか」を考えているが、"アイデアの再利用"のを行う上で本書の考え方はとても重要だと思う。

思考過程を記録することで、後にそれを何度も活用できるという考え方は、生成AI時代には最も重要な考え方だろう。

そのため、この"Zettelkasten"の元祖である本書は、時代に即してはいないものの、理論の面では非常に洗練されており、古典的名著であると思う。
