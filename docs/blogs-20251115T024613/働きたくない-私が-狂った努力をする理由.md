# 『働きたくない』私が、狂った努力をする理由

「働きたくない」

多くの人が冗談半分に口にするこの言葉を、私は本気で信じている。子どもの頃から、周りの大人のように毎朝満員電車に揺られ、やりたくもない仕事に何十年も人生を費やすことが、どうしても私の未来像として描けなかった。「いかにして働かずに生きていけるか」ばかりを考えてきたのが、正直なところだ。

しかし、そんな私の行動を知る人はみんな、そんな私の行動に、疑問に思うらしい。

宿題をサボるためにAIに300時間没頭したり、世界中の論文を漁ったり、#100日チャレンジに挑戦し、noteや講演、ソフトウェア開発業務に毎日膨大な時間を費やす日々。客観的に見れば、まるで「努力家」のように見えるかもしれない。

「働きたくない」と言いながら、なぜそんなに一生懸命になれるのか？
この矛盾のように見える生き方には、実は私なりの確固たる理由がある。

今日はその“理由”について、少しだけ本音を話してみたい。

## やりたくないことから逃げ続ける覚悟

会社員は「安定」だと言われる。でも私に言わせれば、これは大きな誤解だ。毎日朝9時から夕方5時まで、40年も一日も欠かさず働き続けることが“生存の条件”になっているシステム。これが本当に安定だろうか？

正直、宿題を忘れたり、遅刻したりして先生に叱られてばかりだった私には、そんな生活はどう考えても無理だった。週40時間、場合によっては残業で50時間も「やりたくないこと」をやらされ続けるなんて、想像するだけで息が詰まる。

ただ、ここから逃げるのは現代社会では本当に難しい。
逆説的だけど、「働きたくない」と心から思うからこそ、“自由になるための仕組みづくり”や“自分の能力を磨くための努力”には、むしろためらいなく全力を注ぐことができた。それは、他の人から見れば「変わってる」かもしれないけど、私にとってはとても自然なことだった。

## 子どもがポケモンを覚えるように

でも、私のエネルギーは「逃げるための努力」だけでは生まれない。本当に何かに没頭するのは、突如としてやってくる“衝動”に突き動かされた時だ。まるで、子どもが興味を持ったことに一瞬で夢中になるように。

例えば、ChatGPTが登場した時。

先生が「宿題をサボるために使ってはいけません」と言ったの効いた私は、「絶対にサボってやる!」とムキになって取り組み、気づけば数百時間も技術を試し、海外論文を漁りまくり、自作のAIアプリを次々と作り出していた。

誰かに「やれ」と言われたわけでも、報酬が約束されていたわけでもない。ただ、「もっと知りたい」「自分の手で試してみたい」という気持ちが溢れて止まらなかった。

この“導かれる”感覚は、説明しようと思ってもなかなか伝わらない。でも、きっと誰しも一度は体験しているはずだ。子どもの頃、好きなゲームや漫画のキャラクターを自然と全部覚えてしまったり、興味のあることに没頭して気づいたら夜になっていたり。

今でも、朝からやり続けていたら、お日様を一度も見ることがないまま１日が終わったり、逆に、夜からやっていたら周囲が明るくなっていた、なんてことは、月に２回くらいある。こういう、どこから生まれたのか分からない、膨大なエネルギーが、私の人生を大きく動かしてきた。

## 夢中のきっかけは、高尚なものじゃない

こういう話をすると、よく「すごく高尚な動機があるのでは」と思われることが多い。でも、実際は全然そんなことはない。むしろ、9割は「お金や商品、食べ物で釣られる」とか、「誰かに負けたくなくて競争心に火がつく」とか、正直あまり褒められた理由じゃない。

でも、それでも“没頭できるもの”を見つけられるのは、すごく幸せなことだと思う。自分でも些末で役に立たなさそうなことに熱中していたら、なぜかそれが人より詳しくなっていたり、いつの間にかお金になっていたりする。

「好きなことを仕事に」なんて大げさな話じゃない。
ただ自分が熱中していた、小さな衝動やしょうもない興味が、ふと気付くと“仕事”になっている。そういう偶然に人生を預けてみるのも、悪くないと思っている。

## 生産性よりも大事なこと

大人になると、「生産性」とか「効率」とか、他人の物差しで語られることがどんどん増えていく。
「そんなことに本気になるなんて子どもっぽい」「一人だけ熱くなってダサい」と、冷笑したり遠巻きに見たりする人もいる。

でも私は、自分の人生を“他人の言葉”で定義されたくない。
夢中になることを笑われても、「やりたくないこと」から必死で逃げてもいい。
ゲームだって、人生だって、時には感情的になったっていい。
むしろ、そのくらい本気にならなければ面白くないし、自分ごとにはならない。

自分の衝動を信じて、他人の言葉に惑わされず、“大人げない”くらい夢中になってみる。

それが、私が「働きたくない」と言いながら、”努力しているように見える"理由なのかもしれない。
