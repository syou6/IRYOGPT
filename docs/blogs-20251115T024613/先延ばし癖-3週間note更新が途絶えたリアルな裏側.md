# 【先延ばし癖】３週間note更新が途絶えたリアルな裏側

**「note100日チャレンジで毎日更新していたあの人、どこに行ったのかな」**

ーーそんな噂がタイムラインを駆け抜けたかどうかは分からない。でも、確かに三週間くらい更新していなかったのは事実だ。

そんな、周りからの心配のDMが届き始めた頃、私は時差ボケと戦いながらヘルシンキ空港で次の乗り継ぎを待っていた。

時間が全然なかったのは事実だけど、その理由はモチベーション低下でもネタ切れでもない。手元には大量のネタのメモがあるし、それを書きたいとも思っている。

しかし、多くの仕事のオファーを頂き、急激に広がった活動領域。それをどう収拾するか、これからの活動をどう広げていくのか。その設計図を引くことに全精力を注いでいた。

私はこの3週間、何をしていたのだろう？

## １．ヨーロッパでの14日間

今月、私のパスポートには１つ、国章が増えた。スペイン、台湾、ドイツに続く4つめの国章、フィンランドだ。私は先週まで、ウィーンに居た。大学の授業に呼んで貰えたためだ。

その他、仕事を貰っている所に挨拶をした。日中は登壇とミーティング、夜は資料作成に追われた。慣れない中欧での生活と時差ボケで毎日の生活はめちゃくちゃになっていた。

ヨーロッパに３回も行くと、さすがにもう、海外への憧れというのはなくなってしまう。現地の食べ物は脂っこくてメキシカンかヴィーガン料理しかまともに食べられないし、英語が通じない所が割とあって、現地の知り合いに通訳してもらわないとできない場面も多い。一人で何でもできる東京はは、他人と関わらなくても生きていけるという面で、とても良い都市だ。この２週間の出張は、文化体験という面ではいいかもしれないけれど、正直つらい面も多かった。

その一番の理由は知り合いにアパートを借りて住んでいたからだと思う。海外旅行への憧れは、高級ホテルを筆頭とした、外国人向けサービスの快適さなのだろう。東京に帰ってから、デパートでの普通の食事を食べたり、いつも行っている整体を受けたり、スパでのんびりと過ごしたりして、私がいかに東京で快適な生活をしているのかを思い知らされた。

## ２．登壇機会やメディア露出の拡大

最近、私はメディア露出を多くしている。先週はTBS Cross Digに掲載されるほどだ。

[https://www.youtube.com/watch?v=iD1sVwUwgrg](https://www.youtube.com/watch?v=iD1sVwUwgrg)

私自身の知名度拡大に伴い、基調講演や企業・大学講演などのオファーが次々と舞い込んでいる。現時点で既に6つもの講演が決定し、その準備に追われている。

１ヶ月前と比べ、遥かに格式高い所から呼ばれるようになった。それに、明らかに待遇が変わった。私の要望や事情をかなり真剣に加味してくれる。

講演をするのであれば、雑に適当に受けるのではなく、ちゃんとしたコンテンツを作っていきたい。私は授業だろうと講演だろうと、ちゃんと３０分真面目に聞いた経験は一度もないと思う。なにか別のことをしていた記憶しかない。３０分退屈な講演を聞くことが、どれだけ苦痛なことであるかは私が一番よく理解している。

私は必ず何かしらのオリジナルコンテンツを入れている。もちろん同じ部分もあるけれど、相手方の要望に応じて各講演で異なるテーマと内容を準備する必要があり、時間は一瞬にして消えていった。

時間を探しては案をいくつか作り、それに基づきスライドを作る。私は講演や登壇を、メディア部門として、こちらでも「成功」させ、「１００日チャレンジ」で成功したという印象を決定的なものにしたいと思っている。普段なら合間を見て記事を書くのだが、今回はそんな余裕を作れないほど追い込まれていた。

## #100日チャレンジ をしても、先延ばし癖は変わらない

「100日チャレンジを完遂できる人に先延ばし癖なんて？」とよく聞かれる。

実際、私は「サボり癖」を全面に出しているのに、１００日チャレンジという過酷な挑戦をした。

強力なインセンティブや外圧がなくなれば、人は“目先の快”を優先しがちだ。前に呼んだ[行動経済学の本](https://amzn.to/4mjr6As)でも、この傾向を“将来価値の割引”として説明されていた。忙しいときほど人は“簡単タスクを潰して働いた気になる”ことで長期的パフォーマンスを損なうらしい。

[https://note.com/amiotsuka/n/nfde90d632537](https://note.com/amiotsuka/n/nfde90d632537)

今回の私がまさにそれだった。目の前のことに忙殺され、普段のように、「７日後の私が今後１０年を見たときに最も優れた戦略を取れるような状況にする」という戦略を取れなくなっていた。

実際、noteを書くことはとてもめんどくさい。これを書くには、最近の出来事を整理して、ネタを集めて編集しないといけないから。ただ、これのおかげでどこに行ってもネタが豊富だという側面があるから、重要なことであることは認識しているのだけどね。

## おわりに

今の私は仕事がとても順調に推移している。毎月新しい機会に恵まれ、「#100日チャレンジをしたことで人生が変わった」というタイトル通りのことが本を出した後でさえ続いている。

この3週間という空白期間を経て、私自身が見つめ直したことは多い。自分がどんな状況であればnoteを書き続けられるのか、逆に何が障害となるのかもよくわかった。このnoteを書くことで、また新しい気持ちでリスタートを切ることができそうだ。

今後はもう少し、緩急のある更新を心がけたいと思う。これからも、変わらずお付き合いいただけたら嬉しいです。
