# “副業で300万円”は夢か現実か？大手メーカー30歳エリート会社員のリアルな悩み

**「副業で年間300万円を稼いで、年収を1000万円に乗せたいんですよ」**

そう胸を張って語るのは、大手医療機器メーカーの生産技術部門に勤める坂本さん（仮名・30歳）。国立大学の修士課程を修了後、いわゆる“エリート街道”を歩んできた坂本さんは、今や誰もが知る上場企業で働き、年収は700万円に到達。周囲からすれば十分に恵まれた環境に見える。
しかしそんな坂本さんが、昨年わざわざブートキャンプに1年も通い、さらには「年収＋300万円の副業」を狙っているというのだ。一見順風満帆に思えるキャリアの裏側には、果たしてどんな不安や葛藤が潜んでいるのか。今回はそんな坂本さんにお話を伺い、その本音に迫ってみた。

### ◆お仕事と現状の悩みについて

**──まずは現在のお仕事について教えていただけますか？**
「大手の医療機器メーカーで生産技術を担当しています。具体的には、製造ラインの設計や設備の導入などが中心です。」

**──実は私、最初にDMでお話を伺ったとき、20代前半のフリーターや駆け出しのエンジニアが通うようなブートキャンプなのかなと勝手に想像していました。でも蓋を開けてみたら、国立大修士で、大手上場企業勤務のエリートじゃないですか。失礼ながら、想像と違って驚きました。今の年収も悪くないですよね？**
「はい、エリートかはわかりませんが、年収は700万円くらいと、研究室時代の同期と比べても、人並みに貰っているとは思います。」

「しかし、実際に働いてみると自分に専門性があるかといわれると微妙で……。大学時代に所属していた研究室ではロボット技術を扱っていて、C言語を使う場面が多かったんですが、“これを一生極める自信はないな”と思ってしまって。そのまま修士を出て今の会社に入り、気づけば生産技術の部署に落ち着いた形です」

**──一見、順風満帆に見えるキャリアですが、何か不満があるのでしょうか。読者の中には憧れを抱く方も多いのではないかと思います。**

「そうですね。“不満”というと贅沢に聞こえるかもしれないですが、まずは“自分の専門スキル”がほとんど身についていないことが気になっています。」

「ライン設計や設備導入のような大がかりなプロジェクトは実は、年に数回しかありません。普段は保守管理やメンテナンス業務が多く、どうしてもルーティン寄りになってしまうんですよね。」

「また、収入においても“キャリアの天井”が見えてしまっているような気がしています。以前は年功序列で給料も上がっていましたが、今後は社内選考制度の影響で役職やポスト次第になりそうなんです。自分がどう評価されるかを考えると不安で……。」

**──なるほど。では、坂本さんが求める理想の収入はどのくらいですか？**
「とりあえず、副業で300万円くらい上乗せして、年収1000万円にしたいんです。今は700万円ほどあるので、プラス300万円というイメージですね」

### ◆副業のプランは？

**──副業で300万円となると、なかなかインパクトがある数字ですね。もともと副業に関心があったのでしょうか？**

「副業に強い関心があったというよりは、“このままだと頭打ちだな”と思ったのが大きいです。別に会社として副業を禁止しているわけでもないし、いざ何かに挑戦したいときに備えて副業という選択肢を用意しておきたい、と。ただ、私自身、まだ“何をやればいいのか”は全然固まっていないんですよね」
**──あまりはっきりイメージがない感じですか？**
「そうなんです。例えばWebプログラミングを学んでみたものの、どう使っていけばいいのか分からなくて。でも今はエンジニアとして案件を受けてお金を貰えるようになれるほど、自分に自信はないですね」
**──そこはFake it till you make it の精神で頑張るしかないですね。**
「そうなんですかね……。」

### ◆ブートキャンプ参加の経緯

**──ブートキャンプではどんなことをなさいましたか？**
「カリキュラムとしては、データ活用や業務改善のためのプログラミングを学ぶ内容でした。Pythonを使ってデータ分析を行ったり、Node.JSを使った簡単なWebアプリをチームで作ったりして、実際に“ビジネスの課題をどうやってテクノロジーで解決するか”を体験しました。」
さらにビジネススキルを高めるために、ビジネスモデルの立案やプレゼンの実践なども組み込まれていて、いわゆる“ビジテック人材”を育成するプログラムでしたね。参加者全員でアイデアを出し合って、仮想のサービスを立ち上げるプロジェクトを回す経験はとても勉強になりました。」

**──ブートキャンプに参加したのは、会社が推奨したわけではなく、ご自身で行きたいと思って？**
「そうです。会社は特に関与していません。もともとDX推進室へ行きたいと思っていたので、“いきなり自社の選考に応募する前に、実務で使うスキルを身につけたい”と考えてブートキャンプに行ったんです。ビジテック人材っていうんですかね、ビジネスもテクノロジーも理解できる人材を目指したいと思ったのがきっかけです」

**──DX推進室には最終的に配属されました？**
「いえ、駄目でした……。タイミングの問題が大きかったです。希望を出していたのに、いざ枠ができたときには他の部署の人が行くことになってしまって。」



### ◆“強み”をどう活かすか

**──武器となるものが一貫していないのが状況を厳しくさせていますね。修士時代の研究、会社での業務、ブートキャンプで得たこと。全部違うことをしています。**
言われてみればそうですね。修士時代はロボット技術を研究していてC言語を扱っていました。今の会社では生産技術のライン設計とメンテナンス、そしてブートキャンプではWeb系やデータ分析の基礎を学んでいます。

**──ソフトウェア開発能力ではなく、生産技術の知識など、業務知識を武器にしたいですね。**
「そうですね。確かに、自分がすぐに実践的なエンジニアとして通用するかというと不安が大きいです。そうなると“生産技術の現場を知っている”とか“大企業の管理経験がある”とか、そういう強みを活かせる方向性で探すのがいいんですかね」

**──私から見ると、坂本さんは“自分の強みが分からないまま、何となく勉強している”印象です。本業では学歴と大企業の後ろ盾があったから収入が入ってくるけれど、副業ではそうはいかない。まずは副業の市場や案件の相場、そして企業や個人がどんな人材を求めているのかを徹底的に調べてみるのがいいと思います。**
「おっしゃる通りですね。今思うと、何も分からないまま“とりあえずWebプログラミング”の練習をしていたような気がします。ネットでまずは案件を探し、案件で求められていることを先に調べてみることにします。」

## おわりに

一見すると何も不自由のないエリートサラリーマンに映る坂本さん。しかし実際には「専門性を身につけるチャンスが少ない」「会社の人事制度が変わって将来のポジションが不透明」など、大手企業ならではの悩みが色濃く存在した。その結果として「副業で＋300万円、年収1000万円へ」という目標を掲げているけれど、現状では具体的なプランやリサーチはこれからというのが正直なところだろう。
副業で成功するためには、やはり市場の動向を知り、案件ごとの報酬相場や必要スキルを把握したうえで自分の強みを掛け合わせていくことが重要だ。
大切なのは「何となくの不安」や「稼ぎたい」という気持ちを、早めに行動へと変えること。小さくてもいいから具体的な案件を探してみる、コミュニティに顔を出して話を聞いてみる……そうした積み重ねで未来は大きく変わるだろう。坂本さんが近い未来、良い副業を見つけ、稼げるようになることを願ってやまない。
