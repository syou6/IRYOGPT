# 大切なものは、お金では買えない

**「もったいない」は美徳ではない。呪いだ。**

ある日、部屋を見渡すと、そこには積み上がったモノの山があった。

目に入るのは乱雑に積まれた書類、本棚から溢れた本、そして行き場を失った無数のペン。これらが私の頭の中まで散らかしている気がした。

**「いつか使うかもしれない」「売れば少しはお金になる」**そんな気持ちが私を支配していた。

しかし、その日、私は気づいたのだ。モノが溜まるほど、失われているのは「お金」ではなく「時間と心の余裕」だと。

私は、散らかった部屋を片付けることにした。

## 片付けのきっかけ

片付けを始めた理由は寝不足で頭が動かないことだった。頭がぼんやりして仕事がまったく捗らず、何か別のことをしようと思っていた。

部屋を見渡すと、散らかった部屋が広がっていた。普段は見て見ぬふりをしていたけれど、この時は数ヶ月ぶりに、「片付けようかな」という感情になった。

加えて近々引っ越すつもりでいたので、余計なものは少しでも減らしておきたいという気持ちも背中を押した。

## 「お金で解決できるものはすべて捨てる」

片付けを進めるにあたり、私は1つの、単純なルールを決めた。

それは**「今使わないもので、お金さえ出せばいつでも手に入るものは、すべて処分する」**というものだ。

部屋に落ちているものは、書類や文房具、本など多彩なものがある。「フリマで売ったらいくらか」なんて考えていたら進まない。できるだけ早く片付けを済ませたい。

潔く捨てるために、明確な基準が必要。今回は、このシンプルなルールを守って整理を進めてみた。

## たくさん捨ててわかった、不要だったもの

実際に捨て始めると、予想以上に不要なものがあふれていた。使わないボールペンが約100本、名刺が山ほど、そして放置されていた古いスマートウォッチやパソコン、ディスプレイ、印刷機、スーツケースも迷わず処分した。

本棚におさまらない分の本やゲームソフト、貰い物のブランド服などは処分に困ったのですべてブックオフに持っていった。本やゲームソフトは合わせて1万円ほどになり、意外と良心的なんだと関心したくらいだ。しかし、この1万円は粗大ゴミと家電リサイクルの費用で全部無くなった。

結局処分することになったのは、9袋の燃えるゴミと、ペットボトルが2袋、段ボールが10個くらい、粗大ごみが５つ、家電リサイクル対象が２つ。

大量のゴミが1Kの狭い部屋から出ていった。

無駄な買い物ばっかしてるなぁ、と思ったけれど、処分してしまえば何とも思わない。むしろ捨てることで気分が軽くなった。

## 残ったのは、目に見えない経験や思い出だった

意外にも、片付けをしていて唯一手が止まったのは、私が手書きで書いたメモや日記、手帳、1年前に書いていた原稿や出版前のゲラなのだった。

懐かしさとともに、かつて夢中でプログラムや本を作っていた当時の感情が蘇った。モノ自体には執着がなくても、そのモノに結びついた記憶や感情には強い意味を感じるのだと気づいた。

本当に大切なものは、やはり目に見えないものなのかもしれない。
本を書くためのメモなど
## これから…

かつては「もったいない」という考えに囚われていた。私の世代は、モノを買ったら、フリマアプリで売ることは当たり前だ。本や家電、服など、新品でも中古でも、購入するときには常に「リセールバリュー」を考え、不要になったら速やかに売却するというライフスタイルだ。

年上世代からしたらニュースで見ただけかもしれないけれど、大学生の大半はこういう生活をしている。実際、多くの学生が教科書をフリマで買って、それをフリマで売却している。先輩から貰うなんて慣習はもはや消えかけている。

大学生だった1年前の私は、この考え方に忠実で、フリマアプリを駆使し、次第に仕入れまで行い、毎月5万円ほど稼いでいた。おかげでアルバイトをする必要もなく、奨学金とこの収入だけで生活が成り立っていたほどだ。これは私がChatGPTで宿題をサボるための方法を開発するために熱中するまで続く。

しかし、この習慣は徐々に部屋を散らかす要因となっていた。わずかな価値があると思えば、つい手元に残してしまう。結果としてモノが積み上がるものの、それを管理するだけの時間がないため、部屋にゴミが積み上がっていくようになってしまった。

今の私は、そんなことはする必要がない。今は仕事が次々と舞い込む。そもそも今回ブックオフに持っていったときに受け取った額も、対応しきれていないほど来る仕事をこなすだけで簡単に稼げてしまう。

一般に「節約」と言われる行為は、自分の貴重な時間と引き換えに些細なお金を得る行為だ。もはやそのような節約は、私にとって本質的な意味を持たない。顕示的支出ではない、一般的な消費活動でそこまで気を使う必要はないし、世の中に流される必要もない。

今の私はエンジニア・作家・研究者として、やっと評価され始めてきた。今、この機会を確実に掴みたい。そのために、私が本当に守り、大切にすべきものは、自分のアイデンティティと限られた時間、この２つだ。

私には、今、この大切な1年で、やらないといけないことがたくさんある。

これからは、本当に価値のあること、本質的で重要なことにこそ、私の時間とエネルギーを注いでいこう
ーー私はそう強く決心した
