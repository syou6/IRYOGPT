# 【就活生必見！】&quot;サボり界の風雲児&quot; が教える、サボってスマホを見られる仕事の探し方

満員電車に乗って通勤し、会社では常に周りから監視されて、スマホを見ようものなら怒られる。

毎日同じようなタスクに追われて、上司にこき使われ、やりたくもない仕事を淡々とこなす日々――そんな現状にイライラするのは当然のこと。

現代はワークライフバランスが叫ばれる時代。とはいえ現場ではまだまだ厳しい監視が根強く、胸の内側に秘めた「サボりたい」という気持ちを抑え込んで苦しんでいる人も多いでしょう。

本記事では、あらゆる人にサボりの武勇伝を聞いてきた、“いかにバレずに、精巧なサボり方をするか”を研究し続けてきた筆者が、こっそり掴んだノウハウを一挙公開します。

## サボりやすさの評価ポイント

まずは「サボれるかどうか」を見極める際のポイントを整理してみましょう。大きくは「プラス要因」と「マイナス要因」があります。プラス要因とは、「サボりやすさが高まる要素」であり、マイナス要因とは「サボりづらくなる要素」を指します。

### プラス要因

-

**物理的な監視がゆるい
・**人数の多い大企業やオフィスが広い会社など、常時誰かに見られているわけではない環境。
・倉庫管理や外回り営業など、業務エリアが分散しやすい仕事も該当。

-

**成果や指標があいまい
・**個人目標が明確に設定されていない、もしくは形骸化されている場合、日々の行動管理が曖昧になりがち。
・年功序列が強い企業などは、短期的な成果を厳密に追わないケースが多い。
・目標は設定するときにできるだけ低くすることが重要。

-

**リモートワークやフレックス制度**
・上司や同僚からの視線がない自宅やカフェなどで仕事をする分、自由に使える時間をコントロールしやすい。
・ただし、常時オンラインミーティングなどで監視される環境だと逆効果なので注意。
・リモートワークだと良くも悪くも成果物を作った証拠が残るため、リアルの、証拠の残らない手渡しの方が良い人もいる。

-

**組織のルールや文化がアナログ
・**デジタル化が遅れていたり、監視ツールが導入されていない企業は、個々の動きを追いにくい。
・作業報告を紙ベースで行うような場合も、人の目が届かない時間帯が生まれやすい。
・社内の効率化を進めると労働者が不利になるため、効率化施策は骨抜きにし、自分の仕事のみ効率化し、提出期限直前に提出するのが有効。（早めに出すと次の仕事が来る）

-

**「何をやっているのか」が分かりにくい**
・何をやっているのか分かる仕事だと、監理されやすい。
・「直接的に人の役に立つ」仕事ほどサボりづらい。
・エッセンシャルワーカーのような重要な仕事は避けるべき。

### マイナス要因

-

**厳格な数値管理・監視システム**

-

GPS付きの社用スマホやパソコンの使用ログ監視システムなど、細かいデータを取得される会社。

-

倉庫管理でもバーコードスキャンでピッキング回数を常時記録するなど、サボる隙がないケース。

-

**小規模なベンチャーやスタートアップ**

-

従業員数が少ないと、誰が何をしているかが一目瞭然。

-

成果や行動がダイレクトに経営層に伝わるため、サボるとすぐに目立つ。

-

ただし、本当に仕事がないケースもあるので、会社によるとしか言えない。

-

**常時応対が発生する仕事**

-

電話営業や受付など、電話や来客がひっきりなしに来る業務。

-

サボる余裕がまったくない上に、離席も難しい。

-

**目標管理制度が徹底**

-

個人の進捗が細かく可視化される。

-

レビューの頻度が高く、上司からの干渉も多い。

-

小さなタスクを大きく見せかけるような工夫が必要となるため、サボる難易度が上がってしまう。

-

**完全オープンなオフィスレイアウト**

-

仕切りがないワンフロアや、常に背後にも人がいるようなデスク配置だと、スマホを見るのも難しい。

-

働いている姿が丸見えなので、どうしても常に「周りの目」を意識せざるを得ない。

-

逆に雑談をして時間を潰す猛者もいる。

## サボれる仕事トップ５

ここからは、筆者が実際に**各業界のサボり経験者に取材**した結果から、「サボれる度」が比較的高いと感じた仕事トップ５を紹介します。もちろん、企業や部署によって状況が異なるため、あくまで参考程度に捉えてください。

### 5. 倉庫管理

-

**ポイント**: 倉庫は広い空間で働くため、全員を常時監視するのが難しい。また、在庫チェックやピッキングなど、作業の合間が生まれやすい。

-

**注意点**: 監視カメラやバーコード管理が厳しく、作業ノルマがリアルタイム管理される倉庫はサボりづらい。

### 4. コールセンター（待機時間が多い部署）

-

**ポイント**: コールセンターは常に忙しいイメージがあるが、夜間やオフピーク時間帯は意外と待機が長いことがある。SV（スーパーバイザー）の目がゆるい環境ならさらにサボりやすい。

-

**注意点**: インバウンドで問い合わせが絶えないセンターや、営業要素が強いアウトバウンド（企業発の営業）はサボる余裕がない。監視システムが発達しているセンターも多い。

### 3. IT系ホワイトカラー（保守・運用）

-

**ポイント**: システムが安定稼働していれば、障害対応までの間に空き時間ができる。リモートワーク環境が整備されている企業では特にサボりやすい。

-

**注意点**: アジャイル開発などタスク管理が厳密で、常時進捗を報告するような環境は逆にサボれない。小規模スタートアップのエンジニアも、存在感が大きいためサボりづらい。

### 2. 営業（外回り）

-

**ポイント**: 社外に出てしまえば上司の目が届きにくく、移動時間を利用してサボることが可能。顧客訪問を理由にスマホをいじる余裕もある。

-

**注意点**: GPS付き端末で行動を監視されたり、訪問件数を厳しく管理される会社だと厳しい。また、新規開拓やテレアポ中心だと常に社内にいるためサボりにくい。

### 1. 事務（定型業務中心）

-

**ポイント**: 大企業や官公庁系の事務は、承認プロセスが多段階で、待ち時間が発生しやすい。定型的な書類チェックやデータ入力を手早く終わらせれば、ある程度の余裕がつく。多くの社内ニートはこの層だと思われる。

-

**注意点**: 受付や来客対応を兼務する事務は常に人前に立つのでサボれない。人手不足で常にバタバタしている部署だと、スマホを見ている余裕はない。

## サボれる会社を選ぶコツ

上記の「サボれる仕事」に就くためには、会社選びの段階からサボりやすい環境の特徴を把握しておくことが大切です。以下は、サボれる会社を見つけるうえで注目すべきポイントです。

-

**大組織・大企業**

-

従業員数が多いほど、一人ひとりの動向が細かく把握されにくい。部署間の仕事が複雑に分業されているため、責任の所在も曖昧になりがち。

-

**業績や成果を曖昧に評価する風土**

-

いわゆる年功序列型で、日常的な行動管理がそこまで厳しくない会社。

-

売上やKPIに過度に縛られず、「どれだけ会社にいるか」で評価されるような環境は、逆手に取ればサボりやすい。

-

**リモートワークやフレックス制度の取り入れ方**

-

制度としてはあっても「結局毎日出社しないといけない」「オンラインで常に監視される」ような会社もある。

-

実際に活用されていて、自由度が高いかどうかを見極めるのが肝心。

-

**部署やプロジェクトの曖昧さ**

-

やることが明確に決まっていない部署や、役割がはっきりしていない仕事は、サボっていても目立ちにくい。

-

組織改編を頻繁に行う企業では、タスクの境界が流動的で、誰の作業か分からない部分が多い。

-

**アナログ体質・保守的な企業**

-

新しい管理ツールやクラウドサービス導入が遅れている企業では、リアルタイムで行動を追跡されることが少ない。

-

複雑なフローを経て承認を取るため、待ち時間が多いケースもある。

## おわりに

サボりは、ひと昔前なら「怠惰」「不真面目」と見なされがちでした。

しかし、現代の働き方改革やメンタルヘルスの重要性が叫ばれる今、実は「うまくサボる」ことが心身の健康やモチベーション維持において大切な意味を持ち始めています。

会社の仕事はあくまで生活するための小銭を稼ぐために仕方がなく行うものです。仕事のためにプライベートを犠牲にするのは本末転倒です。

サボる技術を身につけると、あなたのワークライフバランスが、もっと軽やかで快適なものになるはずです。
