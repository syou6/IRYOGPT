# 引越し業者を使わずに一人で引っ越した話

「引っ越し代に10万円以上かかる？」

業者の見積もりを目にした瞬間、私は目を疑った。

当時大学生だった私の月の生活費は家賃を除いてもその程度だった。10万円もあれば、贅沢をしなければ1ヶ月間はゆうに暮らせる。そんな大金を「ただ荷物を運ぶだけ」で支払うことには到底納得できなかった。

そもそも、一人暮らしの荷物なんて大した量じゃない。10万円も取るなんて、とり過ぎだと思う。

**「そんなにお金を使うなら家具も家電も全部捨ててやる。」**

私は引っ越し業者を使わずに自力で引っ越すことを決めた。

## 不本意な引っ越し

そもそも私が引っ越すことになった理由は、嫌々ながらも就職を余儀なくされたからだった。

私は就職したくなかった。いや、働きたくなかった。

朝８時に起きて、満員電車に揺られて会社に行き、夜遅くまで上司に扱かれ、帰ってきたら夜10時。倒れ込むように寝て気づいたら明日になり、また仕事の繰り返し。そうして命の代償として稼いだ額も日々の生活に消えていく。そんな毎日を送るのが耐えられなかったけれど、生活費のために泣く泣く会社員になった。

最初の3ヶ月間だけは出社が必須だったため、会社から数駅の場所に8万円という高額な家賃を払い住むことになった。手取りは22万円程度で、学生時代より生活水準は変わらなかったけれど、あの頃の私は、「働く」ということへの絶望感でいっぱいだった。

## 業者なし、一人で挑んだ引っ越し

引っ越しのタイミングも悪かった。2月下旬に学会でスペインへ行き、3月上旬は台湾に滞在していたため、引っ越しの準備も業者手配も後回しになった。

3月中旬の繁忙期に業者を使えばいくらかかるか予想もできない状況だった。
そこで私は、引越し業者を使わずに引っ越すことにした。

家具家電は既に4年間使っている上、全部安いものなので思い入れもない。良い買い替えタイミングだと思ったからだ。

私はモノをあまり持たない。服は全部で10着くらいしかもってないし、そんなにモノは持たない。そもそも外食しないからキッチン道具が何もなく、フライパンや炊飯器すら持っていないのも大きいと思う。

なにせ私はベッドとパソコンさえあれば生きていける人で、私が片付けができない人なのでものを買わないようにしている所が大きいと思う。

結局、家具家電は全て粗大ゴミに出し、必要なものは段ボール6箱にまとめてヤマトの宅配便で送った。驚くほど簡単に引っ越しは済み、費用も粗大ゴミと宅配便代を含めてわずか15,000円ほどだった。

## 引っ越しの買い出し

３月上旬、家具・家電を買うためにぐりーんうぉーく多摩に行った。

まずはニトリでベッドやゲーミングチェア、デスク、本棚などを購入。15万円くらいかな（ベッドだけで10万円使っている）。

次に訪れたヤマダデンキで洗濯機、冷蔵庫、電子レンジを購入したけれど、ネットの契約をすることで5万円引きとなり、結局家電の費用は1万円程度で済んだ。最近の電気屋はこういうのがあるからいいよね。

正直買い物が楽しいという感情はなく、面倒な手続きを淡々と済ませた。

携帯とインターネットの契約で時間がかかったためすでに日が暮れており、休日のインターネットの工事は休日料金がかかると言われて落胆した。

## 自力引っ越しは断然おすすめ

こうして業者を使わずに一人で引っ越した私は、自力引っ越しは断然おすすめだと感じている。4年間使った安価な家具や家電はすでにボロボロだったし、プラス7万円程度の出費で新品に買い替えられるならむしろお得だった。

また、引っ越しを機に荷物が大幅に減り、生活の整理や断捨離の良い機会にもなった。

今も住所は変えていないけれど、次回の引っ越しもきっと業者を使わずに、自力で挑戦するだろう。引っ越し代に悩んでいる人はぜひ試してみて欲しい。
