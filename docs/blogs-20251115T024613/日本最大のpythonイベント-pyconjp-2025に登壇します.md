# 日本最大のPythonイベント、PyConJP 2025に登壇します！

2023年4月22日。日付まで鮮明に覚えているほど、私にとって特別な日。

この日、中央大学の教室で、私は人生で初めて「Python（パイソン）」というプログラミング言語に触れた。

Python、と聞いて「ヘビ？」と思う人も多いかもしれない。実際、私も最初は「ふぃそん？」とか読んでいるような状態だった。
Pythonのロゴ。ヘビがモチーフらしい。

でもこの出会いが、今の私を作り、人生を大きく変えてしまったのだ。
そして、まさか2年半後に「日本最大級のPythonイベント」で登壇することになるなんて、当時の私には想像もできなかった──。

## “宿題をサボりたい”から始まったストーリー

「プログラミング」と聞くと、難しそう、理系っぽい、というイメージがあるかもしれない。
私もそうだった。でも、最初の動機は至って不純。「宿題、サボりたいなあ……」という気持ちだった。

Pythonは、魔法みたいだった。授業中に先生に隠れてChatGPTにプログラムを書かせるだけで、あっという間に答えを出してくれる。

スプレッドシートの処理、ファイル操作、図の描画。「これは使える！」と、私はどんどんPythonにハマっていった。

家に帰ってからも、ネットで調べては小さなプログラムを書き、「こんなこともできるんだ！」と夜な夜な感動していた。オセロ、将棋、テトリス、ぷよぷよ…色々なゲームを作った。

授業の課題をAIで手伝ってもらいながら、私は“ラクをする方法”を全力で探していた。

## 100日間、毎日アプリを作り続けたワケ

そんな私が次に始めたのが、「100日チャレンジ」だった。
毎日何かしらの小さなアプリやプログラムを作って、SNSに投稿し続けるというもの。

正直、最初は「ネタが切れたらどうしよう」と思っていた。
だけど、これが予想以上に面白かった。
思い付きでゲームを作ったり、必要なツールを作りながら、ソフトウェア設計を学んでいった。

最初は「誰かに見てもらえたらいいな」くらいの気持ちだったけれど、
回数を重ねるごとに、Twitterやブログのフォロワーさんからコメントをもらえるようになった。毎日少しずつ上達していき、10日ごとに、別人のように成長していた。

この100日チャレンジは、私の人生のターニングポイントだった。この挑戦を終えた後、論文を書き、多くの学会で評価され賞を取る一方、メディアに紹介されたことをきっかけに本を書き、その本がベストセラーになった。

それをきっかけに、お仕事をたくさん貰えるようになった。講演や受注開発だけで十分生計を立てられるようになって、会社を設立した。拙書のタイトル通り、「人生が変わった」のだ。

##  “PyCon JP”登壇決定！——私が伝えたいこと

そして、2025年。ついに「PyCon JP」という、日本最大級のPythonカンファレンスで登壇することが決まった。

2年前の私には、想像もできなかった未来だ。

「PyCon JPって、どんなイベント？」
一言で言えば、全国からPython好きが集まる“お祭り”のような場所だ。プロから初心者、学生、エンジニア、主婦、デザイナー……ありとあらゆるバックグラウンドの人たちが、「Pythonって面白い！」を共有し合う場所。
登壇する人も聞く人も、“好き”と“情熱”を全力でぶつけ合う——そんな熱量に溢れている。

……と、私を招待してくれた、PyConJP 2025座長の西本卓也さんに教えて貰った。

西本さんから、以下のようなことを言われている。

>

「特に、これまでPyCon JPに関心を持たなかった方々や、生成AI時代になってPythonと出会った新しい層の方々に、大塚さんのお話を通じて勇気を与えられたらと思っています。
そして「つまらない」「苦しい」というプログラミング学習のイメージを変え、楽しく創造的な学びの可能性を示せたらと考えています。」

3月に、実際に届いたメールから引用

私はこのイベントで、おそらくそういうことを話すのだと思う。

私が歩んできた、新しい「エンジニアへの道のり」の体験や、Pythonを通じたプログラミング学習などについて、面白くてためになる話をしたい。そして、誰かの背中をそっと押せたら——そんな気持ちで登壇に臨みたい。

##  「プログラマー気質」は"ズル"から動き出す

振り返ってみると、私の人生は“ズルしたい”“ラクしたい”という、なんとも不純な動機から始まった。いや、　そんな気持ちだけで今も成り立っている。

でも、その一歩が、思いもよらない景色につながっていた。
プログラムを書く際にバグに悩まされたり、20コミットくらいが無駄になったり、環境構築に失敗したり、何度も「もうやめようかな」と思う瞬間もあった。

だけど、そんな一つ一つの経験が、今の私に結びついている。ChatBotを作ったり、Webアプリやサイトを作ったり、研究用・業務用ソフトウェアを作ったり、色々なことをしたけれど、いかに手を抜くかという視点で、積極的に新しい技術を試し、ソフトウェア設計をたくさん学んできた。

Pythonは簡単でわかりやすい一方で、なんでもできる。

今はもちろん他の言語も使うけれど、かつてPythonを数千時間使って学習したこの経験や、エンジニアとしての「考え方」は、一生物の財産だと思っている。あのとき、Pythonに出会えて、本当に良かった。

私にとってPythonは、「始まりの言語」だ。私のエンジニアとしてのキャリアは、Pythonから始まった。
