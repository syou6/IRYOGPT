# 日記を書いていたら、作家になっていた

**文章を書くのは苦手だ。書かなくていいような人生を送りたい。**

大学3年生の私は、そのように考えていた。

思えば、文章を書くことでろくなことがあった試しがない。

あらかじめ答えが決まった「自分の意見」を書く課題。

「おりこうさん」が書いた面白くない本を読まされる読書感想文。

先生が喜びそうなことから少しでも外れたら、「不正解」とされる。

そんな窮屈で監視された世界が、心底嫌いだった。

しかし、そんな私は２年後には”作家”と呼ばれるようになった。

そのきっかけは、ごく平凡な**“日記”**だった。
Googleでエゴサすると、Googleさんは私を作家と言っている。

## 1. 人生の閉塞感と退屈な毎日

2023年、私は大学3年生だった。

その頃の私は特にやることがなく、自堕落な大学生活を送っていた。

授業が退屈で、講義中はゲームや動画で時間をつぶし、テストは運頼み。

そんな中で、私は時間がある日は近所のブックカフェに足を運んでいた。

小説やラノベなどの物語は現実離れしているか主人公がアホでイライラするので好きではない。だからビジネス書や自己啓発本をよく読んでいた。

どの自己啓発本も、「日記を書け」「メモを取れ」と私に勧めてくる。

でも、そんな几帳面なことを私ができるわけがない。そもそもそういう生真面目さがあるのであればもっとテスト勉強とかしているはずだ。

当時の私は自らの人生への希望を完全に失っていた。

とはいえ、そういった本を当時数百冊と読んでいることを考えると、人生を変えたいという思いがあったのかもしれない。

なんだかんだあったのか、日記やメモを書いてみようと思った。

しかし、当時の私は文章にする能力もなければ、そのような出来事もない。

そのため、当時の日記は、時間とその時の行動を記録する、「行動ログ」のようなものだった。

## 2. ChatGPTとの出会い

翌年の2024年4月。大学4年生になった私は、授業の一環で「ChatGPT」という対話型AIツールと出会った。

今でこそソフトウェア開発で使っている。

しかし、当時は正直なところ、「宿題を楽にできないか？」という不純な動機で始めた。

当時、ネットニュースでは、ChatGPTを使ってレポートを提出し、単位を剥奪された学生が話題になっていた。

それを見た私は、「バレたらまずい！」と思い、「どうやってバレずにうまく活用できるか？」を必死に模索。毎日ネットで調べ、思いつく方法を片っ端から試した。

その過程で、ChatGPTと対話しながらプロンプトや使い方のコツを日記に詳しく書き記すようになった。

もともと行動ログだけだった日記は、ChatGPTによるやり取りやプロンプトの試行錯誤の記録でどんどん分量が増えていく。記録しておくことで、昨日使ったプロンプトを再利用できるからだ。

私は単に宿題をサボるために毎日プロンプトを作り、その結果を記録していった。

単に“サボる方法”を考えていたはずなのに、この過程で物事の説明能力が飛躍的に上昇。なんやかんやあって論文を書き、賞を取ったり招待講演に呼ばれたりと、目に見えて才能が開花しだした。

## 3. #100日チャレンジ

2024年10月28日。私は「100日チャレンジ」を始めた。

なんやかんやあって学会で上手く行ってしまった私は、次の発表のために、ソフトウェア開発を勉強する必要があったからだ。

このとき私は、1日10時間ほど没頭してアプリを作っていたため、メモの量が激増していた。

そのため、一部を文書化し、行動ログとメモだけの日記からリンクで飛べるようにした。

更に、**「自分への引き継ぎ文章」**という概念を追加した。

この考え方は画期的だった。日記に目を通せば昨日の続きをスムーズに再開できる。これにより前日何をしていたかを忘れることが全くなくなったため、作業効率は飛躍的に上昇。

過去の私との連携が効率的にでき、「車輪の再開発」がほとんどなくなった。

## 4. 日記から生まれた“作家”への道

2025年3月、情報処理学会の学会誌の取材を受けた際、「これだけの経験と記録があるなら、本にまとめてみたらどうか？」と提案された。

文章を書くのは嫌いだからと戸惑ったけれど、振り返れば日記には約300ページ分の膨大な記録がぎっしりとあった。

日々の行動、思考、プログラミングで悩んだことなど、書くための素材は山ほどあった。

そこで意を決し、日経BPさんで企画書を通したあと、約半年かけて執筆に取り組んだ。

その結果、2025年1月14日に『****[#100](https://bookplus.nikkei.com/atcl/catalog/24/12/05/01757/)****[日チャレンジ毎日連続](https://bookplus.nikkei.com/atcl/catalog/24/12/05/01757/)****[100](https://bookplus.nikkei.com/atcl/catalog/24/12/05/01757/)****[本アプリを作ったら人生が変わった](https://bookplus.nikkei.com/atcl/catalog/24/12/05/01757/)』を無事出版する運びとなった。

思いがけず多くの方に手に取っていただき、発売前から増刷が決定。2月23日現在、4刷まで到達し、各メディアで紹介されるとともにGoogleさんからも作家として認めていただいた。

学生時代、「書くことは嫌いだ」と言っていた私は、いつの間にか“作家”として歩み始めていた。

## おわりに

「日記を書いていたら、作家になっていた」

――最初はただの「現実逃避」や「宿題回避」の手段だったものが、ここまで私の人生を変えるとは夢にも思わなかった。

日々の記録や思考を言語化することで、自分の進む道を客観視できるようになり、知識や経験を次の行動につなげやすくなる。

嫌いだったはずの執筆は、いまでは私の強力な武器になっている。

人生はどうなるか分からない。
