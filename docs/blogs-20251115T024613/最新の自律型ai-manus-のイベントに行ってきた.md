# 最新の自律型AI「Manus」のイベントに行ってきた！

「本をタダで欲しいからイベントに来てほしい」

こんなユニークな誘い方をされたのは初めてだった。

**「経費使える立場なんだから、自分で買えばいいのに」**
**「前々日に告知して人を募集するなんて、変な人だなぁ」**

## 驚異的な性能を持つManas

このイベントでは、「Manas」を初めて体験した。

実際に触れてみて驚いたのはその高度な機能性。CodexやDevinなど、他の自律型AIエージェントをいくつか試してきましたけれど、Manasは明らかに他のサービスよりも数か月先を行っている印象を受けた。具体的な詳細については別の記事でしっかりと書こうと思うけれど、直感的な操作性と素早い反応速度は間違いなく業界をリードしていると感じた。

## なぜかManusを語らないGoromanさん

イベントはManusのためのものはずだが、なぜか主催のGoromanさんはManusの説明を全くしなかった。Goromanさんのプレゼンはどちらかというと自由な雰囲気で進められ、技術的な深掘りよりも「とにかく新しいツールで遊んでみよう！」という気軽なスタンスだった。

確か、Goromanさんって、幼い頃から最新技術を色々触ってきた、叩き上げの技術者だったと思うのだけど、多分こんな感じで技術に触れてきたのだろうな。

## AIコミュニティとの新たなつながり

イベント全体を通して感じたのは、AIコミュニティの熱量と新たな人とのつながりだ。今、生成AI関連の業界はとにかく景気が良い。個人や中小企業にも、何百万、何千万というお金が動いている。「LLM無職」と言われる、仕事を辞めてLLMにフルコミットする人も増えている。そのためか、とにかく参加者同士の交流が非常に活発だった。

今回のイベントは単なる技術体験にとどまらず、新たなコミュニティや人脈形成の場としても非常に有意義な時間を過ごすことができた。次回もまたぜひ参加したいと思える、魅力的なイベントでした!
