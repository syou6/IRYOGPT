# 朝からスマホに時間を奪われる皆さんへ：人生が変わる習慣設計術

「また朝からスマホに時間を奪われてしまった」

私は、そんな後悔を抱えながら一日をスタートさせることに嫌気がさしていた。

目覚めてすぐスマホに手を伸ばす習慣は、自分の時間や集中力を奪い、充実感よりも虚しさだけが残る。

そうした失敗を繰り返して気づいたのは、**「自分の意思に頼ってはいけない」ということ**だ。

そもそもスマホに無駄に触れてしまうのは、やるべきことが面倒だったり明確ではないから。そこで私は、「スマホを使ってしまう＝習慣設計が間違っている」という逆説的な考え方を取り入れ、習慣作りを根本から見直すことにした。

## スマホ時間は習慣設計のバロメーター

まず最初に私が着目したのは、自分が1日の中でどれだけスマホを無意味に使っているか、ということだった。

意識的にスマホを見る時間を記録してみると、ほとんどが「暇だから」や「他のやるべきことをやりたくないから」という動機だった。

これに気づいた私は、自分がスマホを触るときは習慣の仕組みがうまく機能していない時だ、と考えた。例えば、朝起きてすぐスマホを触ってしまうのは、起床後にやるべきことが設定されていないからだ。

そこで私は朝一番にnoteを書くことにした。従来は18時投稿だったけど、朝9時〜10時に投稿するという新しい習慣を導入した。

この時間に投稿すると、通勤時間中か会社についてからのニュースを見る時間だからか、Xの反応が凄く良い。

ネタについては毎日１時間くらいかけてまとめてあるので、いざ書くとなれば１時間くらいで書ける。これにより、朝の最初の行動をスマホからnoteの執筆に置き換えることに成功した。

実際に成果は出ていて、今月のnoteのPV数は8万を超え、なぜかブロガーとしても成功の部類に入ってきた。

## 意思よりも報酬

次に私が導入したのは、「意思には頼らない」というルールだ。#100日チャレンジ以前、私は「毎日1時間勉強する！」などと決めては失敗してきた。

意思というものは基本的に長続きしない。強い衝動や動機はある程度自分を動かせるけれど、少なくとも私は義務感で真面目に取り組むような「おりこうさん」ではなかった。

そこで、意思ではなく、適切な報酬と緩い締切を設けることで、自分自身を無理なく動かす仕組みを作るようにした。

たとえば、このnoteを書いているときも、「早く終わらせて何か食べに行こう」と思ってやっている。事実上の緩い時間設定と報酬設定をすることで、集中力を自然に引き出してくれている。

## 大切なのは「1つのゴール」だけ

また、私が大切にしているもう一つのルールは、**「1つのゴールだけを設定し、それ以外は細かく気にしない」**ということだ。

たとえば「仕事を3時間やる」という目標を立てると、実際にはぼーっとする時間も含まれてしまい、自己嫌悪に陥りやすい。

そこで私は、**「とにかくこのタスクを終わらせる」という明確なゴールだけを設定する**ようにしている。そもそも、途中でスマホを見たくなるのは人間の構造上仕方のないことだ。

ゴールだけ決めて、そのための進捗が出ているのであれば、それ以上に自分を責める必要はない。実際には3時間の計画が５時間になることもあるけど、それもだいたいは見積もりが甘かっただけ。

１日ですべてを変える必要はない。1日1%でも昨日よりも良くなれば十分だ。

## 習慣が人生をゆったりと整えていく

現在の私は、8時ごろに起床するとすぐにnoteを書くことから1日を始める。その後、近くのカフェで朝食を食べ、10時からソフトウェア開発の仕事に集中。昼食後はどうせ捗らないので自由時間にして読書をしたり、駅ビルをふらふらと散策してゆったりと過ごす。夕方4時くらいに再び仕事を再開し、夜は適度に事務的な作業を行った後、23時には仕事を完全に終了している。

こうした生活を送るようになって、自分自身が感じるストレスも減り、生活全体が自然なリズムで整うようになった。

習慣構築において重要なのは、**「自分の弱さを認め、弱さをカバーする設計をすること」**だ。意思の力ではなく、スマホの時間をバロメーターとして設計を見直し、適切な報酬を設定し、目標をシンプルにすること。この習慣構築法が、私の人生を豊かに、そしてゆったりと支えてくれるだろう。
