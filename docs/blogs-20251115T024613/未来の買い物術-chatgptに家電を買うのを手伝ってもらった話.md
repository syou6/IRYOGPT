# 【未来の買い物術!?】ChatGPTに家電を買うのを手伝ってもらった話

「掃除機欲しいけど、家電量販店に行くのめんどくさいなぁ。」

壊れた掃除機を処分してもらってから1ヶ月。そろそろ新しい掃除機を買わなくてはと思いつつも、忙しくて放置していた。

そもそも掃除機には種類が多すぎて、自分ひとりで選ぶのは非常に困難だ。分厚いカタログなんて見たくもないから、店員さんに説明を聞かないと絶対に分からない。だから、買い物そのものがとても面倒くさかった。

「AIに手伝ってもらえば楽なのでは？」

そう思いたち、ChatGPTに頼ることにした。

## ChatGPTに助けを求める

とはいえ、掃除機は多くのメーカーが出しているし、どのモデルを買えばいいのか分からない。

「掃除機が欲しい。何を買えばいい？」そう聞くと、ChatGPTは人によるとか言ってきた。仕方ないので、私の情報を色々と書いた。一人暮らしで、6畳の1Kに住んでいる。そう書くと、ある程度絞って教えてくれた。

「あなたのように怠惰な方は、フィルターのメンテナンスが不要なDysonの掃除機を買うと良いでしょう。」

要約すると、このような文章が帰ってきた。確かに自炊が嫌で毎日外食するような私にはフィルター掃除なんてできないけれど、どうしてChatGPTはこんなに私について知っているのだろう？

## どれを買えばいいのか分からない！

とはいえ、Dysonも多くの製品を出している。ネットで調べると、通販サイトがたくさん出てくる。ここからどれがいいのかを調べるのもめんどくさい。そこで色々な情報を入れた。

「6畳のワンルームに住んでいる」
「セミロングの髪が落ちている」
「タイルカーペットを敷いている」

するとChatGPTは、製品を色々と比較し、V12 Detect Slim Absoluteというモデルを提示してくれた。

[https://amzn.to/43HU4Tn](https://amzn.to/43HU4Tn)

## どこで買えるか聞いてみると…

「最安値をお調べしましょうか？」
親切なChatGPTの提案を見て、私はお願いすることにした。しかし、そこから問題が起き始めた。ChatGPTが提示するリンクを開くと、売り切れだったり、微妙に違うモデルだったりした。

それに気づいた私は、ChatGPTに、その製品が本当に正しいか聞いた。すると、２回に１回は間違っていた！

私は何度もこの作業を繰り返した。10回くらいやったと思う。毎回2分ほどかけてChatGPTに確認してもらう作業を繰り返すことになった。

## 購入できたけど…？

先に音を上げたのは私だった。ChatGPTは何回やっても答えてくれる優秀なアシスタントだ。しかし、短気な私には、この細かな管理作業には耐えられなかった。

「もういいや。」

うんざりとした私は、Amazonの検索欄にモデル名を入力した。結果として、ChatGPTが提示してくれた最安値より5千円くらい高くついたけれど、売っているかどうかすら分からないリンクを何時間も探すストレスから解放されるならその方がずっと良いと思えた。

## AIにも分からないことがある

今回の買い物を通じて、「AIにも分からないことがある」ということを痛感した。モデルや付属品の違い、リアルタイムの在庫状況など、複雑な情報を完璧に扱うのはまだ難しいようだ。未来の買い物術として期待したけれど、家電製品の複雑さを克服するには、もう少し時間がかかりそうだ。

…そもそも、家電のカタログは難しすぎる。絶対に、嫌になった顧客が何も考えずに最上位モデルを購入することを狙って作られているんじゃないの？

丸一日「勉強」すれば、５０００円くらい安くなるかもしれないので、興味のある方は頑張ってみて欲しい。大学生の頃、5000円のイヤホンを買うために１時間くらいネットで最安値を探していたことを思い出したよ。
