# 『欠乏の行動経済学』を読んだら、「サボりの天才」の私の計画性が判明した

「計画性」と聞くと、どのような人を想像するだろうか。

きっと、課題を渡された初日からコツコツと取り組む、真面目で几帳面な「おりこうさん」をイメージするだろう。

私はそれとは対極の存在だ。課題を渡させたらどうやったらそれをやらずに済むかにすべてを捧げる。レポート、課題、仕事の締め切り――私は昔から「どのように効率的にサボるか」を真剣に考えてきたタイプだった。

そんな私でも、最近は周りから色々な仕事を渡されて忙しくしている。

いつもギリギリまでタスクを放置する癖がある私にとって、「欠乏」という言葉は馴染み深くもあり、ちょっと苦々しい響きでもある。

『欠乏の行動経済学』は、私に武器を授けてくれた。それは、余力を確保することに命を捧げる私の生き方を、根底から支えるようなものだった。

## 概要：私が出会った「欠乏の本質」

この本の主張はシンプルだ。「欠乏」は人間の判断力を大きく狂わせる。お金、時間、食料など、何であれ足りなくなった状態は、人の視野を狭める。

著者は様々な事例を挙げ、「欠乏」が短期的な問題解決力を高める一方で、長期的な視点を失わせることを明らかにしていく。

特に衝撃的だったのは、**貧困層が場当たり的に借金をすることと、高所得者が時間に追われていつも忙しくすることが、本質的には同じ構造である**という指摘だ。

## サボり魔の私に計画性があったとは

読んでいて、著者が描く「普通の人」の行動パターンと、自分自身の行動が違うことに気付いた。一般的な人はタスクが与えられると、とりあえずすぐに取り掛かる。しかし、取り組んだのはいいものの、雑務や新しい仕事に邪魔され、最終的に締め切り直前で追い込まれるらしい。

一方で私は、タスクをもらった瞬間からまず**「いかにサボるか」を考えるタイプ**だ。書き取りの宿題ならカーボン紙を準備し、議事録作成なら[自動化アプリ](https://notta.cello.so/4O4ifbSJFwR)を探す。つまり、**私は「サボる計画」を立て、それに必要な道具を確保してから、締め切り直前まで放置するタイプ**だったのだ。

私は時間的な余力を確保して別のことを行っており、全体的な時間的余力を増やし、締め切りを守れないリスクを大幅に削減することができている。

皮肉にもこの本を読んで、**私が実は計画的な人間であったことに気づかされてしまった。**

## 「欠乏」の功罪：視野狭窄の罠

「欠乏」の功罪を考えるとき、私はまず自分の経験を振り返った。締め切りが近づくほど、私は超人的な集中力でレポートや仕事を片付けられる。まさに「欠乏」状態がもたらす瞬発力のおかげだ。

しかし、その一方で長期的な視点は完全に失われる。目の前の緊急な課題に意識が奪われ、それ以外のことはことごとく無視されてしまう。メールを放置してしまい、後日謝罪するはめになることもよくある。仕事で忙しくてポストの中を見ていなかった結果、請求書が何回も来てカード会社から怒りの電話が何回も届いて法律事務所からの手紙が来たこともある。
(請求額5万円に対して口座には50万円くらいあったのに！）

「欠乏」は今この瞬間にも影響している。この本を読んでいたとき、私はモスバーガーの騒がしい店内にいた。お昼過ぎから本に夢中になり、気づいたら夜になっていた。店内にいた時間は約5時間だったけれど、その間、自分が「読む」ことに完全に集中していたため、周囲の変化に全く気付かなかった。

この体験は、「欠乏」がいかに視野を狭め、集中と引き換えに周囲のことを見えなくしてしまうかを示している。

## この本から学べること：欠乏との付き合い方

読後、この本が教えてくれたことは多かった。特に印象に残ったのは「**判断力は有限の資産である**」という言葉だった。

日々の小さな決断で消耗してしまうと、本当に大切な判断ができなくなる。毎日の生活で意識的に「余力」を確保しないと、その余力は無意識に消費されてしまうらしい。

私のようなタイプが無意識に実践していた「いかにサボるか」という工夫は、実は著者が推奨する「重要であるが緊急ではないこと」の一種なのかもしれない。

集中して一つのことを片付ける時間を確保し、自らにとって重要なことはうまくサボる。ーーこうやって、時間の欠乏との上手な付き合い方を模索していこうと思う。
