# 次の目標は「新大陸」

本を出した。アプリもリリースした。

つい１年前まで、それは遠い夢のようなことだった。机に向かって、ひとりで生成AIと対話しながらアプリを作り続けた日々。周囲から見ればただの「実験」や「遊び」にしか映らなかったかもしれない。

けれど、その積み重ねが一冊の本になり、全国の書店に並び、多くの読者に手に取ってもらえた。そして、その物語から派生するようにして「メモるん」というアプリが生まれ、私はようやく「自分の創造した作品を世に送り出す」という経験をつかんだ。

講演会の壇上に立ち、数百人の前で話すようになったときも、不思議な気持ちがした。ほんの半年前まで、私は自らの物語を語るなんて想像すらしていなかった。気づけば「著者」として紹介され、いつの間にかアプリも作れるようになっていた。努力の足跡は、確かに形になった。

けれど、心のどこかがまだざわついている。達成すればするほど、次の目標が見えてくる。今の私は「本を書いた人」としては認知されているが、「アプリ開発者」としてはまだ途上にある。

視線は自然と海の向こうへと向かう。まだ踏み入れたことのない大地。憧れと不安が入り混じる、未知の領域。そう、私にとっての次の目標は「新大陸」に足あとを残すことだ。

## 本とアプリがくれた「出発点」

すべての始まりは、ほんの小さなきっかけだった。
大学時代、宿題をサボるためにChatGPTに没頭していた私を、先生は「君は生成AIの使い方が上手すぎる」と言って学会に連れて行ってしまった。その偶然が、私の人生を大きく変えた。研究発表で賞をもらい、そこから「#100日チャレンジ」を走り抜け、生成AIとともに毎日アプリを作り続ける日々が始まった。

あの100日間で積み上げた無数のメモや試行錯誤は、やがて一冊の本となり、多くの人に読まれることになった。発売から間もなく重版がかかり、「生成AIに育てられた第１世代」という言葉とともに、私は国内で知られる存在になっていった。講演や取材の依頼が次々と届き、かつての「宿題をサボりたかった学生」が、いつの間にか「著者」と呼ばれるようになった。

さらに、本の延長線上で生まれたのが、自作アプリ「メモるん」だった。チャットのように気軽にメモできるこのツールは、私が「ただの語り手」ではなく「つくり手」でもあることを示してくれた。リリースの瞬間、舞台で「Release Now!」の文字がスクリーンに浮かんだとき、胸の奥で確かな手応えを感じた。「これでやっと、私はエンジニアとして証明できた」と。

[https://note.com/amiotsuka/n/ne898ba3997d3](https://note.com/amiotsuka/n/ne898ba3997d3)

だが、それは同時に「出発点」に過ぎなかった。
本は私を著者にしてくれた。アプリは私を開発者に近づけてくれた。けれど、私の中でまだ定まらない問いが残っていた。

――「私が次にするべきことは、何なのか？」

## 太平洋を越えられる実力を求めて

国内での活動は、今のところ順調だ。
本のおかげで講演の依頼は途切れず、取材も舞い込む。北海道から九州まで、呼ばれればどこへでも行けるし、それを生活や仕事に結びつける基盤も整ってきた。正直、国内で生きていく分には、この流れだけで十分やっていける。5年は今の生活を続けられると思う。けれど、その安定感こそが、私にとって一番の危うさなのかもしれない。

私が目指すのは「ここで終わること」ではない。
今の私は「著者」としては知られている。だが「アプリ開発者」としての印象を強めるためには、もっと広い舞台で実績を残す必要がある。その舞台は国内ではなく、海の向こうにある
――そう、誰もが憧れてやまない「新大陸」だ。

もちろん、海外に行くだけなら簡単だ。航空券を買い、観光客として写真を撮ればいい。お金さえ払えば今すぐにでも飛べるだろう。だが、それでは「足あと」にはならない。私が欲しいのは、現地での「仕事」だ。講演でもいい、プロジェクトへの参加でもいい。何か一つでも成果を残し、その証拠を国内へ持ち帰ること。それこそが本当の意味での「上陸」なのだ。

その中でも特別な意味を持つのがアメリカだ。
生成AIの本場であり、数多くの研究者や開発者、そして次の時代を形づくるスタートアップが集まっている場所。そこに行き、彼らと同じ空気を吸い、自分自身の言葉と作品で勝負すること。それができなければ、どこか「置いていかれている」という感覚がつきまとう。

ただし、現実には大きな壁がある。日本を半月離れるだけでも、今の生活では大変だ。開発案件の対応や講演・メディアの予定もあり、国内の活動を止めてまで外に出るのは容易ではない。ヨーロッパには知り合いがいるので連れて行ってもらう形で訪問できるけれど、アメリカには自分の力で道を拓くしかない。そのことが、私の胸に重くのしかかっている。

だが同時に、その壁の向こうにこそ、私が求める景色がある。まだ踏み入れたことのない地平に、自分の最初の足あとを残したい。憧れと不安が入り混じる感情こそが、今の私を突き動かしている。

## 「依存」から「自由」へ

どうすればその「新大陸」にたどり着けるのだろうか。
私が考える答えは一つだ。

「アプリとnoteだけで生活を成り立たせること」

これまで、生活の安定を確保するために受託開発や依頼された仕事を請け負ってきた。報酬は確実で、成果も見えやすい。私が社会人２年目で月収が軽々と100万円を超えたのもこれが理由だ。

[https://note.com/amiotsuka/n/n4ac7f6908fa3](https://note.com/amiotsuka/n/n4ac7f6908fa3)

けれど、それは同時に「東京にいなければならない」という鎖を意味していた。打ち合わせ、契約、納品……そのすべてが物理的な場所に私を縛りつける。安全ではあるが、そこからは一歩も外に出られない。

私が欲しいのは安定ではなく「自由」だ。
私の創造から生まれた文章とアプリが生活を支える基盤となれば、私はどこにでも行けるし、何でもできる。私が持つすべての時間や労力を、私のアイデンティティを示せる成果物ーーアプリ、note、本など、にすべて投入することができれば、私はもっと大胆な成果を生み出せるようになる。

その体制を築くために、私はまず国内で「アプリ開発者」としての存在感を高めなければならない。アプリを改善し続け、ユーザーに愛されるものへと育て上げる。そして、それを収益につなげ、講演や執筆と並ぶもう一つの柱にする。そうすることで初めて、「受託に依存しない生き方」が可能になる。

もちろん、これは簡単ではない。収益を安定させるには時間がかかるし、改良を続けるための努力も欠かせない。それでも、私は確信している。この「依存から自由へ」の転換が、新大陸へと渡るための唯一の道なのだと。

米国のカンファレンスやミートアップでは、ほとんどの人が「自分のアプリ」を持っていることを前提に交流している。自分のアプリという「名刺」を持ってアメリカに行かないと、観光目的で“留学”を装う堕落した大学生と同じだ。

だからこそ、私は「自分の名刺」となるアプリを持って、新大陸に渡りたい。誰かの仕事に従属するのではなく、私自身の成果物を胸に、胸を張って挑戦する。そのとき初めて、米国の開発者と対等に向き合えるのだと思う。

## 新大陸に足あとを残す意味

私の目標は、アメリカに永住することではない。
国内での活動はこれからも続けていきたいし、ヨーロッパやアジアなど、他の場所にも今と同様、たくさん行くことになるだろう。ただ、その中で「アメリカに行ける力がある」という事実は、これからの活動において大きな意味を持つと思っている。

生成AIの本場であるアメリカで存在感を示せれば、国内外を問わず「どこででもやっていける」と証明された存在になる。逆に言えば、その力を持たずして世界を語ることはできない。だからこそ、私にとって「新大陸に足あとを残す」という挑戦は避けて通れないと思う。

本とアプリは、私に「著者」としての顔と「開発者」としての顔を与えてくれた。これからは、その二つを武器に、国境を越えて活動する力を身につけていく。アメリカに住みたいわけではない。でも、アメリカに行ける力があること。それを叶える頃には、どこに行っても使える大きな実績や名声を得て、私をさらに遠くへ導いてくれるはずだ。

「次の目標は新大陸」。

それは、未知の場所への憧れであると同時に、自分の未来を切り拓くための実践的な宣言でもある。これからの航路はまだ地図に描かれていない。けれど、その海を渡る準備は、すでに始まっている。
