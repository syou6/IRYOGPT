# 【毎日更新】Noteを70日続けるとどうなるのか？

**「毎日投稿を100日も続けたら、何か変わるのか？」**
――そんな好奇心から私は「note100日チャレンジ」を始めた。

現時点で70日間が経過したけれど、毎日欠かさずに記事を書き続けている。

過去の記事を眺めてみると、この2ヶ月余りで、私の文章力やnoteを使った発信の仕方が明らかに変化していることに気付かされる。

この記事では、70日間noteを毎日更新して分かったこと、起こった具体的な変化、そしてこれからの方向性についてまとめていく。

## note公式に取り上げてもらった！

3月7日、私が書いた記事がnote公式に取り上げられた。
取り上げられたことによってその記事の閲覧数は大幅に伸びた。
初めて掲載された記念に撮ったもの

公式掲載は偶然ではあるけれど、継続的に発信しなければ絶対に訪れなかったチャンスだと強く感じている。

また、公式での紹介をきっかけに、私の文章はnote公式が勧められるくらいの品質であると認識できたのも大きな収穫だった。noteの毎日投稿を続ける意義は、こうした**「予想外の出会い」にもある**と実感している。

おそらく、今まで私を知らなかった人も数多く見てくれたはずだ。

## ストーリー形式の記事

毎日投稿を続ける中で、最近特に力を入れているのがストーリー形式の記事だ。最初は**「面白そうだな」と軽い気持ちで始めた**けれど、書いているうちに日常のちょっとした出来事を物語として再構成する面白さにハマっていった。

ストーリー記事は通常の記事の倍以上の時間がかかるものの、文章で書くよりもより細かい情報を伝えられると思っている。文字での記憶よりも、状況の記憶の方が鮮明だからだ。

さらに、ストーリーを描くことで普段見逃していた日常の小さな変化や感情に敏感になり、生活そのものが豊かになった気がする。普段メモや日記で書いていることを、かなり忠実に伝えられるのはとても面白いと思う。

**書くことで人生がより鮮明に見えてくる**
――毎日書き続けることで得られた最大の恩恵は、実はこの感覚かもしれない。

## 確実に向上した文章力

70日間毎日記事を書き続けると、文章力が明らかに向上しているのが分かる。以前の自分の記事を読み返すと、「うわぁ…」と恥ずかしくなるほど読みづらいと感じる。

現在の記事は読みやすさが増し、論理的でスムーズな展開ができるようになった。実際、noteの記事の閲覧数は着実に増加しており、**記事1本あたりの平均閲覧数は初期の頃と比べて約4倍にまで伸びている。**

この数字が、私自身の文章が読者に受け入れられてきている証拠だ。Webライティングは初めてだったけれど、70日間で、より自然体で、そして自由に書けるようになってきたと思う。

基本的に、私はこの辺ならそれなりの文章を書けるらしい
・メモ・日記について
・プログラミング関係
・習慣化など
・「私」について

## 投稿時間がグダりかけている

しかし、毎日投稿が習慣化してくると、新しい課題も見えてきた。

それは**「投稿時間の乱れ」**だ。

最初は毎朝一定の時間に更新していたけれど、最近は他の仕事を優先するようになり、6時更新だったものが夜8時以降に更新することが増えてきた。

ただ、これは一概に悪いこととも言えない。というのも、「書こうと思えば必ず書ける」という自信がつき、noteを投稿する負担が減った結果、他の仕事に積極的に時間を割けるようになったためだ。

私は専業note投稿者と違って、noteにすべてを捧げているわけではない。せいぜい1日3時間程度だ。

それでもnoteを継続することが生活の中に完全に溶け込み、自然と優先順位が調整されている証拠でもある。

今後の課題は、投稿時間の管理をもう少し意識的に行い、生活リズムを整えていくことだろう。

## おわりに

70日という節目を迎えて改めて振り返ってみると、毎日の投稿を通じて得られたものは多かった。公式掲載、ストーリーを書く楽しさ、文章力の向上、そして投稿時間という新たな課題まで、多くの気づきがあった。

100日目まではあと少し。このチャレンジを通じて更に何が変化していくのか、自分自身でも楽しみにしながら書き続けたいと思っている。
