# 【毎日更新】Noteを80日続けるとどうなるのか？

「毎日noteを書き続けたら人生はどう変わるのか？」

ーー80日前、私はそんな漠然とした疑問を抱えてスタートラインに立ったのだろうか？

「100日間の連続投稿チャレンジ」なんて、勢いと好奇心だけで決めた軽い気持ちーーいや、最初はこんな挑戦やる気は全くなかったような気がする。

あまりにも多くの出来事、それも私の人生が一変するようなことが毎週のように起きているからか、既に1年くらい経っているような気持ちになる。

## 著書が新聞に!?

驚いたことに、**私の著書が新聞広告に掲載されることになった!**
実際に載った新聞

私も当日新聞を見てから知ったのだけど、発行元の日経BPさんが、2ヶ月で5刷という好調な売上を背景に広告を出してくれたらしい！

新聞に自分の本が大きく取り上げられているのを目にしたときは、思わず息を呑んだ。これまで地道に文章を書き続けてきた成果が、形になって現れた瞬間だった。

実はこのnoteは、出版社から「本の宣伝をして欲しい」という要望から始まったものだった。そのため、最初のnoteはそんなエピソードになっている。

[https://note.com/amiotsuka/n/nda94c4fa32cf](https://note.com/amiotsuka/n/nda94c4fa32cf)

しかし、**毎日投稿しているうちに投稿自体が当たり前になった**。

そもそも私には毎日日記を書く習慣があるので、これ自体は全く苦にならなかった。日記の一部をくり抜き、フォーマットを整えて投稿するだけだ。

次第に、noteは読者との交流するツールになった。おそらく、私がキーボードを叩ける限りは書いているんだろうなぁ。

## SNSがバズった日――1ツイートで20万PVの衝撃

80日間の中で、SNS上で大きな出来事が起こった。

noteの記事の一部を切り取って投稿した何気ないツイートが、気がつけば20万PVを突破するほどバズったのだ！

[https://twitter.com/AmiOtsuka_SE/status/1902135834796814575](https://twitter.com/AmiOtsuka_SE/status/1902135834796814575)

まさに予想外の結果で、**「一体どこがそんなに受けたんだろう？」**と自分でも驚いたほどだ。

しかし、この成功体験を再現できるかというと、まだ課題は多い。私はあくまでnoteの文面の一部を切り取っただけだからだ。

偶然に頼らず、「狙ってバズる」ことを可能にする再現性をどう確保するかが、今後の大きな課題として浮上している。

## 今私がやっていること

現在私がやっていることはこの４つに大別される。
4つの、今私が今やっていること

上の２つはなんとなく分かるだろう。私の主要コンテンツだ。問題は下の２つだ。

現在、私が取り組んでいる個人開発効率化は非常に順調だ。実際に日本だけではなくヨーロッパにもその名声が届いている。

しかし、その成果は私自身の学習能力と体力に大きく依存しており、そもそも一日12時間（週84時間）を超えてプログラムを書くのは現実的ではない。

また、効率化が進むにつれて判断すべき情報量も増えるけれど、[人間が一日に判断できる情報には限界がある](https://note.com/amiotsuka/n/nfde90d632537)らしく、いずれ限界が来ることを予想している。

そこで私は現在、自律型AI開発（Devin計画）に積極的に取り組んでいる。今のところその開発能力は私の20%ほどだけれど、近い将来、自分と同等の能力にまで成長させる計画だ。

さらに、それを今の2倍、最終的には100倍程度まで拡大し、SIer相手に一人でも立ち向かえるほどの体制を構築したいと考えている。

## 継続の先に見えてきた景色とこれから

80日間noteを毎日書き続けて実感したのは、[ここにも書いた](https://note.com/amiotsuka/n/nc62eee0e8191)けれど、「継続することでしか得られない成果」が確かに存在するということだった。

著書が新聞広告に掲載され、SNSがバズり、フォロワーが増えるという目に見える成果ももちろん嬉しい。しかし、**最も価値があったのは、自分自身の文章力や発信力の成長を実感できたこと**だ。

残り20日も、そしてそれ以降も、私は変わらず書き続ける。この旅路の先にどんな景色が広がっているのか、今から楽しみで仕方がない。
