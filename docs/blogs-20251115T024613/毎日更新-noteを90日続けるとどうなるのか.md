# 【毎日更新】Noteを90日続けるとどうなるのか？

**「noteを書いたところで、読んでくれる人なんているの？」**

3ヶ月前、私はnoteを書くことに疑問を抱いてきた。何を隠そう、私は既にブログを３回立ち上げた。毎回10本くらいの記事を書いては、そのすべてで１日10PV以下に終わった。海外からのスパムメールとメッセージだけが残る、ネットのゴミと化していた。

しかし、4度目の挑戦である、このnoteは結果が全く異なった。毎日noteを更新することで多数の読者に見てもらえた。毎日noteを更新することで得られた結果は、自分自身でも驚くほど大きな変化をもたらした。

90日という節目を迎え、具体的にどんな成果が生まれたのかを振り返ってみようと思う。

## 記事が毎日売れるように！

noteを毎日書き続けたことで、最初の大きな変化は「記事が毎日売れるようになった」ということだ。

初月はすべて無料記事にし、先月は記事が売れるのは月に数回程度だった。

しかし、毎日コンスタントに投稿を続けるうちに、確実に読者が増え、記事の売れ行きも比例して増えていった。

特に顕著だったのは、有料メンバーシップの人数の増加だ。メンバーシップを始めてから40日で21人にまで伸びた。

毎日の積み重ねが「信頼」と「関心」を生み出した証拠だと思っている。

## 単月10万PV突破！

90日間の毎日投稿によって、単月で10万PVを超えるという成果を達成することができた。最初は1万PVを達成するのも大変だったのに、**最近は1本で1万PVを叩き出す記事が毎週のように生まれている!**

今や記事を投稿すると必ず誰かがツイートで感想を伝えてくれるようになった。この反応が非常に面白く、書き手としてのモチベーションを高めてくれる。

実際に記事へのリアクションがあるのとないのとでは、モチベーションが大きく異なる。反応があることで、「次の記事ではもっといいものを書こう」という意欲が湧き、執筆の質が向上していく好循環が生まれた。私が継続して書いているように、読者も継続して読んでくれている。これはnoteを書く最大の喜びであり、成果の一つだと言える。

[https://twitter.com/AmiOtsuka_SE/status/1905042043858252195](https://twitter.com/AmiOtsuka_SE/status/1905042043858252195)

## 多くのメディア露出!

今週も多くのメディア露出が実現した。この10日間の間に、ラジオ番組（J-wave）への出演を果たしただけでなく、リアルサウンドテック、Mac Fan、日経クロステックなどの著名なWebメディアにも取り上げられるようになった。

最近では、読者だけでなくメディア関係者にも記事が注目され、口コミが広がっているような実感がある。私や出版社が今まで縁のなかったところにも私の記事やツイートが届いているようだ。

私がnoteを書き続けることによって、社会的な認知が広がり、今後の活動の可能性も大きく広がっていることを実感している。90日間の継続的な発信は、単なる個人のアウトプットにとどまらず、自分の影響力を広げていくための大きな手段となっている。

## 今後について

**このnoteの最終的な目標は、書籍化するためのメモの結晶体を作ることだ。**

著書「#100日チャレンジ」の続編は、まだ情報不足や守秘義務の問題があり実現は難しいけれど、noteベースの本をビジネス書的な立ち位置でまとめていけたらと考えている。そのためにも、引き続き良質なコンテンツを生み出していくことを意識したい。今週はこの２つに注力したい。

### 「メモ」や「日記」の活用

これまでは主にメモの作成方法に焦点を当てていたが、今後はメモの「使い方」にも焦点を当てていく予定だ。noteを大量に作成しながら様々なプロジェクトを同時進行できるのは、このメモ術のおかげだと言って間違いない。

この圧倒的なアウトプットを支える部分の説明を、100日目になる前にしたい。

### 5%の法則

この法則は一見適当に見えるかもしれない。しかし、私の根幹を支える重要な考え方であり、少ない単位で成果を上げ、リスクをうまく回避するための戦略だ。

この法則をもっと詳しく解説し、活用法を伝えていければと考えている。

## 毎日継続することで得られる「本当の価値」

noteを90日間毎日書き続けたことで見えてきた最大の価値は、ただ単にPVや収益が上がったことだけではない。それ以上に、読者との強い繋がりができ、さらにはメディアを通じて自分の言葉が広く届けられるという可能性を感じられたことが最も大きな収穫だった。

これからnoteを始めようとしている人や、毎日の継続をためらっている人に伝えたいのは、「継続は想像以上のチャンスをもたらしてくれる」ということだ。

ぜひ自分自身で、その可能性を実感してほしい。
