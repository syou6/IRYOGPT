# 渋谷に引っ越しました！

「住む場所を変えると、人生が変わる。」

たしか、そんなことが書いてある本を昔読んだことがある。

思えば私は、これまで週3回は渋谷に足を運んでいた。打ち合わせ、友人との食事、単にふらふらと出歩く――この街はいつも新しい刺激を与えてくれた。この街をふらふらと出歩いていると、色々なアイデアが生まれる。今まで書いた論文はほとんど、この街を散策しながら考えたものだ。

でも、ただ“通う場所”だった渋谷が、“住む場所”になったら、自分の毎日はどう変わるのだろう？

そんな小さな好奇心と、「今よりもっと成長したい」というシンプルな欲求。その2つが交わったとき、私は「渋谷に住もう」と決めた。

## 渋谷を選んだ理由

これまでの私は、渋谷から電車で片道45分ほどかかる場所ーー戸越の辺りに住んでいた。新卒1年目で借りたマンションは、家賃も安くてそれなりに快適だったけれど、正直「ベストな選択」ではなかった。どちらかというと、「とりあえず」で選んだ場所であり、「本当に住みたい街」ではなかったのだ。

そもそも社会人1年目は、生活を成り立たせることで精一杯。手取り22万9千円の私に、そんな余裕はなかった。

でも、今年になり、仕事も順調に回るようになってきたタイミングで、「毎日をもう少し快適に、効率的に過ごせないだろうか」と思うようになった。

きっかけは単純だ。「移動時間がもったいない」という思いだった。

私は取材や講演などで、よく渋谷や六本木、新宿などに来る。往復90分の移動は、それ自体は苦痛ではないけれど、実際のところ、その時間を有効活用できていたかといえば微妙だ。

しかも、時間的な関係でタクシーを使う機会が増えてきたり、パソコンを開きたいのに移動中は開けないといった小さなストレスを感じることが増えてきたりと、ライフスタイルに合わなくなっているという実感があった。

だったら、最初から渋谷に住んでしまえばいいじゃないか。そう思うようになったのは自然な流れだった。

## 引っ越しの大変さ

引っ越しは何度か経験してきたけれど、今回は正直、想像以上に大変だった。渋谷のマンションは家賃もそれなりに高いし、人気物件はすぐに埋まってしまう。そもそも渋谷という場所は、住む場所というより「遊ぶ場所」「働く場所」として選ぶ人のほうが多い街だ。

「住みたい部屋が見つからない」「内見の予約が取れない」と、いくつもの壁にぶつかった。理想と現実のギャップを何度も味わいながら、それでも「妥協はしたくない」という気持ちが勝った。

一番大変だったのは、部屋の埋まる速さだ。部屋が空いたと思ったら、その日のうちに内見に行き、その日のうちに決定しないと、別の人にその部屋を取られてしまう。結局、１ヶ月くらいかけて、今のマンションを見つけて、契約を締結した。

## 渋谷での新生活にかける期待

新しい部屋に落ち着いて、少しずつ日常が戻ってきた今。これからの生活について、いくつかの楽しみと目標がある。

まずは、「街が私の生活圏になる」という感覚だ。これまでの私は、「渋谷に行く」たびにどこか「遠出」の気分があった。でも、今はヒカリエも、109も、PARCOも、すべてが徒歩圏内。

そして、移動にかける時間が格段に減ることだ。知り合いと会うときに、「戸越まで来て」と書くとワガママ感があるが、「渋谷で待ち合わせよう」と書くと、至って普通に聞こえる。別の場所だとしても、電車は10線も乗り入れているため、どこに行くにも一本で行ける。その分、別のことに時間を使えるようになるはずだ。

## おわりに

「住む場所を変えると、人生が変わる。」

この言葉が本当かどうか、今の私にはまだ分からない。

たしかに、渋谷という新しい街に引っ越してきて、日々の景色も、出会う人も、生活のリズムも、少しずつ変わり始めている。でも、それが「人生が変わる」ほどのインパクトになるのかどうかは、これからの日々の中で、私自身が確かめていくしかない。

もしかしたら、劇的な変化は起きないかもしれない。けれど、この街で暮らすことで、小さな選択や出会いが積み重なって、少しずつ私自身も変わっていく——そんな予感がしている。

「住む場所を変えると、人生が変わる。」

この言葉の本当の意味を、これから渋谷での日々を通して、自分なりに探していきたい。

——渋谷での新生活、始まりました。
