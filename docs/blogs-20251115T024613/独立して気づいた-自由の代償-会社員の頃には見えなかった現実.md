# 独立して気づいた「自由の代償」──会社員の頃には見えなかった現実

「独立すれば、もっと自由に働けるはずだ」

大学生の頃から、私はずっと、そう思っていた。

会社という枠に縛られず、やりたい仕事を選び、自分のペースで生きる。
朝の満員電車も、上司の顔色もうかがう必要もない。
自分の実力だけで生きていく――それが“理想の働き方”だと思っていた。

実際、ネット上では、会社員は損だというツイートで溢れている。収入が低い、税金が高い、社会保障料でごっそりと持っていかれる。実際に、１年前の私はそう思っていた。だから、会社を設立し、今のような生活をしている。

そして、実際に、独立してからは、そんな誰もが想像するようなことをしている。

自分で案件を選び、自分のブランドをつくり、時間の使い方も完全にコントロールできる。
誰かの許可を待つ必要もなく、すべての判断を自分で下せる。

でもその自由には、ふとした時に、気付かされることがある。

自由には、必ず代償がある、ということを。

## 「働ける限り働いてしまう」問題

「収入が安定しない」には、大きな問題がある。まずは、以下のストーリーを見てもらいたい。

>

会社員時代は、勤務時間が終われば自然と仕事が終わる。
けれど、独立すると“終わり”がない。
働けば働くほど収入が増えるし、休めばその分だけ減る。
気づけば、「もう少しやっておこう」と毎晩のように深夜まで作業していた。

このような話は、よく聞くだろう。私もそうだった。

実際こういう話を会社員の知り合いに言うと、「羨ましい」とか、「休みたいときに休めるならそっちの方がいい」とか言われる。

表面的にはそのとおりだ。私も独立当初は「自由になったから、たくさん遊ぼう」と思っていた。でも、それは本質的ではない。

１番の問題は、**「仕事を断るのは簡単だが、獲得するのは難しい」**ということだ。

そのため、「稼げるときに稼ぐ」というのが基本的な戦術になる。断ると次にもらえなくなる可能性があるため、多少無理してでも取りに行く。そのようなやり方が常態化する。

そのため、表面的には自由そうに見えるけれど、実際には、「仕事の来ない閑散期に頑張って時間を作って休暇にする」ことになる。

実際私も、お盆の仕事が全く来ないときに旅行や海外出張を入れていた。

## 収入の増減の現実："人脈"がすべてを左右する！

実は、独立して最初に痛感したのは、「人脈がすべてを左右する」という現実だった。

「独立したら人間関係なんて気にする必要がない」などと、ネットで副業を勧める人は主張しているけれど、それは完全に間違いだ。むしろ、人間関係がすべてだ。会社員の時よりも、人間関係が重要だ。

どれだけ技術力があっても、どれだけ時間を注ぎ込んでも、そもそも「誰かが自分を知っている」状態でなければ、仕事は発生しない。会社員時代は、会社という看板が営業をしてくれていたし、用事があれば相手から声をかけられる。だが、独立すると、その看板は消える。信用も、実績も、すべてゼロから積み上げなければならない。

最初の数ヶ月は特にそれを痛感した。仕事ができても、仕事がなければ意味がないからだ。

だから実際、私が最初に受けた案件のほとんどは、知り合いからの紹介だった。

知り合いが仕事をくれ、その方の知り合いがくれ、といったように、仕事をくれる人が増えていった。

だからこそ今では、「スキルより人脈」という言葉を、皮肉ではなく現実として受け止めている。

個人で生きるということは、社会という大きな生態系の中で、自分という“小さなブランド”を育てることだ。
そして、そのブランドを信じてくれる人がいる限り、仕事は続く。

独立の世界では、結局のところ「誰とつながっているか」が、すべてを左右した。

## 一人会社」の現実: 必要な知識が異様に多い

独立して最も大変だったのは、技術でも営業でもなく、「知識の幅」だった。
会社員時代、私はエンジニアとしてコードを書くだけでよかった。
でも今は、法務・経理・営業・マーケティング――すべて自分でやる必要がある。

契約書を読めなければ、トラブルが起きたときに自分が損をする。
請求書の書き方を間違えれば、入金が遅れる。
税金の仕組みを理解していなければ、確定申告で大混乱する。

さらに、自分の収入を上げるためには「働く量」ではなく「単価」を上げる戦略が必要だ。
そのためには、ブランディングやマーケティングの知識も欠かせない。
どんなに技術があっても、「知られていなければ存在しない」のと同じ。
自分をどう見せるか、どこに立つかを考え続けなければ、すぐに埋もれてしまう。

つまり、独立とは「一人で会社を運営する」ことそのものだ。
自分が社長であり、営業であり、経理であり、現場担当でもある。
何か一つ欠けても回らない。

実際、最初はそれが途方もなく大変だったけれど、周りから教えてもらいながら、少しずつ覚えていった。

私はまだ年齢的に周りが気を使って教えてくれる。この点では、早くに起業して良かったのかな、と感じる。

## おわりに

独立してみて、ようやくわかった。
「自由に働く」という言葉は、想像以上に複雑で、重たい意味を持っている。

よく、フリーランスエンジニアが、30~35歳くらいで会社員に戻るという話を聞くけれど、実際に１年近くやってきて、少しずつ理解してきたような気がする。

ソフトウェア開発技術を一人で継続的に習得し、繁忙期は多少無理してでも仕事を取っていくような生活は、そう長くは続けられないのかもしれない。

今の私は1日13〜14時間を4ヶ月くらい続けられるし、このようなことをできる人があまりいないからか、もしくは技術力が上がったからか、単価をかなり上げられた。

実際、同世代と比べても遥かに収入がいいし、予定が入っていなければ気軽に世界中を飛び回って遊べる。仕事もプライベートもかなり充実している部類に入ると思う。この1年間、たくさんの思い出が、私の写真フォルダに詰まっている。

でも、この生活は、10年後はできないのかもしれない。無理できるのは20代まで、と、知り合いの"人生の先輩"からは、何度も忠告されている。

**それでも、私はこの生き方を選んでよかったと思っている。**

独立とは、自由を得る代わりに、すべての責任を背負うことだ。
「やるか、やらないか」も、「成功か、失敗か」も、すべて自分次第。
だからこそ、ひとつひとつの選択が、人生そのものを形づくっていく。

「自由に働く」とは、決して“楽をすること”ではない。

むしろそれは、資本主義という熾烈な競争の中で、「自由」という美しい言葉の裏に隠された現実と向き合い、自分にしかない武器を見つけ、生き延びることだった。
