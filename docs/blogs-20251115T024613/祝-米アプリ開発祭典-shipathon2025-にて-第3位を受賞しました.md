## 「メモるん」とは

**メモるん** は、音声・画像・テキストといった複数の情報モダリティを、チャットのような直感的な UI で記録・整理できるメモアプリです。

-

**主な特徴**
 ・音声録音 → 自動文字起こし
 ・画像添付／カメラ撮影で視覚情報も記録
 ・テキスト編集・タグ・検索機能
 ・チャット風 UI による直感的な操作感
 ・過去のノートを “会話形式” で辿れるリコール性

-

**開発背景／意図**
 アイデアを “言葉にせず消えていく断片” にせず、その場で自然に残せるようにしたい、という思いから始まりました。
 シンプルながらリッチな記録体験を提供することで、ユーザーの思考・インプットを逃さない「第二の記憶領域」を目指しています。

・基本的には、OCR、音声文字起こしをより使いやすくする方向性での機能の改善を進めていく予定です。
