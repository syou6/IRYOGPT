# 私がもし、もう一度大学生をやり直すなら

「普通にはなりたくない」

そんな言葉を口にする大学生をよく見かける。

実際、努力の有無に関わらず、あらゆる学生が何か特別な存在になろうと願っているだろう。それは社会的成功かもしれないし、それ以外かもしれない。

しかし、なにか１つくらいは、自分が特別であってほしいと思うのは普通のことなのかもしれない。

ーー私もかつてその一人だった。

バイトやサークルなんて世俗的で馬鹿騒ぎしているだけのことなんて、やったって仕方がない。それよりも、もっと大事なことがあるはずだ。

そう口先だけでは立派なことを言っていたけれど、実際にやっていたのは授業を適当にサボってゲームをしたりDiscordでボイスチャットをするくらいだった。

でも、今考えると、多くの大人は「大学生活を大事にするべきだ」と言うけれど、具体的な答えは誰も教えてくれなかった。

もし私が今、もう一度大学生に戻ったのなら、何をするのだろう？

## 今の大学生は「あまのじゃく」

今の大学生は、「普通に生きるなんて嫌だ」と言う人がとても多い。普通に就職して社会の歯車になるのが嫌だ、と。

実際、大学生のうちに「何者か」になろうと努力する人はたくさんいる。大学時代、弁護士・会計士・税理士の勉強をしていると話していた人はたくさんいた。

今の日本経済はスタグフレーションに陥っており、税金・社会保険料で給料の半分が取られ、そもそも少子高齢化で縮小されると言われているほどお先が真っ暗だ。「何者か」になることで、楽な生活ができるようになりたいと思うのは普通のことなのかもしれない。

しかし、「普通に生きるなんて嫌だ」と言うわりに、自分を変える努力もしない人は多いように感じる。実際、先程の資格受験生の大半は勉強している所なんて見たことがなかった。

ここまで意識が高くなくても、私たち「陰キャ」は、無意識的に「陽キャ」を下に見がちだ。馬鹿騒ぎすることに何の意味があるのか、と。でも実際に意味のあることができるわけではなく、スマホをいじるだけで日々を浪費している。

あの時の私は、「何者か」になろうとして、「ただの捻くれ者」になっていたのだろう。

## 「今の私」の再現性

幸いにも私は今、アカデミックな分野での成功を経てエンジニアになり、本を出版してベストセラーになり、会社も設立した。

だが、**もしもう一度大学四年生に戻ったとしても、この状況を再現できる自信はまったくない。**

最初に成功した「#100日チャレンジ」から得られた偶然の出会いやチャンスは、**数え切れないほどのトライアンドエラーの末に手に入ったもの**だ。同じことをもう一度やれと言われても、同じ形での成功は不可能だろう。

それに、あの時と同じスキルで、あの時と同じくらい上手くできるとは思えない。当時の私は全力を尽くしたと思う。後悔はないし、あれ以上のことはできないと思う。

今も数々の挑戦と失敗を繰り返しているけれど、20回のトライのうち19回失敗する構図は全く変わっていない。一つ一つの成功を掴んで大切に成長させ続けてきたからこそ、「1回の成功」がどれだけ貴重で重いのかをはっきりと認識している。

この「1回目の成功」を、この構図を一切知らずに得られたことは奇跡だと思う。

## 普通の人生を選ぶ意義

では、もう一度大学生をやり直すならどのような道を歩むべきだろうか？

**私はあえて「普通の人生」を選ぶだろう。**普通であることは、実は一番再現性が高いのだ。

大企業に入ったり、安定した人生を歩んでいる人に共通するのは、**サークルやアルバイト、授業や研究、適度な遊びを「普通に」経験してきたこと**だ。

彼らは特別優秀だから大企業に入ったのではない。むしろ「**マイナスを避ける」ことを上手にやった結果**である。私自身も大企業に入ったことがある今だからこそ分かるけれど、大企業に入った人の多くは必ずしも仕事ができる人ばかりではない。むしろうわべだけまともな人が出世する印象だ。

一度大企業に勤めると、転職するにも独立するにも役に立つ。転職は前職の会社の看板を見せれば30代前半までは簡単にできる。

独立する場合でも、普通の人が独立する際は、大企業に勤めている友人を顧客にする場合が多い。つまり、いわゆる「陽キャの馬鹿騒ぎ」は、陽キャのほとんどが大企業に勤めることを考えると、**ある意味合理的**なのだ。

## 普通から飛び出すタイミング

とはいえ、ただ普通で終わるだけでは人生の面白さは半減してしまう。そもそも、「普通に生きたい」人はこの記事を読まないだろう。何か、自分の目標や、「回避したい未来」があるからこそ、読んでいるのだろう。

１つの成功を掴むのに本当に大切なのは、**「面白そうなチャンスが訪れた時に、そこへ全力で飛び込む勇気を持つための構造を作ること」**だ。

私にとっての「#100日チャレンジ」はまさにそれだった。このチャレンジに私のほとんどの時間を投入することで、本来なら得られないようなチャンスを引き寄せた。

ポイントは、この時**「大学を辞めたり休学せずに済んだ」**ことだ。私は強力な「サボり技術」を駆使して、大学での義務を可能な限り効率化・自動化し、自分の挑戦に使える時間を最大化した。その結果、偶然の成功を引き寄せ、今の私を形作る基盤が生まれた。

もちろん、失敗しても大した痛手はない。「普通の人生」を歩めばいいだけだ。

現在もメディア露出や受託開発、研究活動を並行して進めており、リスクを分散しつつ効率化を徹底している。

もしもう一度大学生に戻れるなら、「普通」の安定をベースにしながら、適切なタイミングで自分の情熱を全力投入することを心がけたい。

## おわりに

「普通にはなりたくない」と思う気持ちは、誰しも心のどこかにある。それはとても自然なことだ。

しかし、今なら分かる。本当に特別になるためには、「普通」という安定した土台を築いたうえで、決定的な瞬間に勇気をもって行動を起こすことが不可欠だと。

もし、もう一度大学生になれるなら、私はまず「普通」になりすますことから始めるだろう。そのうえで、時間を確保するために「サボり技術」を磨き、人生が提示するチャンスを待つ。

ーーその機会が来た際に迷わず飛び込ことで、自分だけのストーリーを再び紡いでいけるはずだ。
