# 私はなぜ、アプリに500万円を注ぎ込んだのか【500万円回収記#0】

本は売れた。講演依頼も絶えない。受注開発の仕事も順調

――独立してからの生活は、“成功”そのものだった。

受注開発の単価も上がり、真面目に仕事をしていれば安定して稼げる。
講演の仕事も増え、企業や大学からの依頼が続く。
このままでも十分に生きていけるし、むしろ、今どき珍しい、「安定して食べていける道」が見え始めていた。

けれど、そんな時に私はふと思った。
――このままで、本当にいいのだろうか。

誰かの依頼をこなして、予定された時間に話をして、納品し、お金を貰う。

確かに、安定はある。けれど、安定というのは、どこか退屈なものがある。

講演ではいつも、「宿題をサボる才能だけで成功した」と、派手な生き方を話しているけれど、実際に行っていることは、9割が泥臭い、誰もがやっているような地に足ついた仕事だった。

だからこそ、もう一度原点に戻りたい。
――私はそんな、綺羅びやかな世界を、望んでいたのかもしれない。

## 「この人、本当にエンジニアなの？」

2025年8月、私はスタジオパーソルに取材され、記事になった。その後、ネット上で少し話題になった。

[https://studio.persol-group.co.jp/nama/250821-1](https://studio.persol-group.co.jp/nama/250821-1)

この記事のインパクトは凄まじかった。私のツイートが2日間で100万近く見られた。この記事はおそらく、数百万回見られているだろう。多くの人に私の存在を知ってもらえたし、実際に企業からの講演依頼も増えた。

だが、同時に、酷い誹謗中傷もされた。
SNS上では「AIの情報屋だ」「講演で稼いでるだけ」「この人、本当にエンジニアとして稼いでるの？」といった書き込みが並んだ。

さまざまな発信をしている以上、批判されることは覚悟していたつもりだ。しかし、ここまで正面から「技術力を疑われる」ような批判を受けたのは初めてだった。

私は自分の活動を「技術と物語の両輪」として捉えてきた。
作家活動を表で行いつつ、エンジニアとして生計を立てていた。

確かに本と講演で稼いでいたのは事実だけど、余った残りの時間はすべてソフトウェア開発の受注に注ぎ込んでいた。

でも、守秘義務契約があるから、それを証明することができない。それがとても悔しかった。

## それは、1年前から予想していた

批判してくる人がたくさん出てくる――それは、今からずっと前からわかっていた。

2024年4月、新卒であるSIerに入社してすぐ、私は、出版社で本の出版をするために、どんな企画を持ち込むか悩んでいた。

大学を卒業してすぐ、SIerに入社したばかりの4月。
私は出版社に持ち込む企画を考えていた。

AIの入門書か、キャリア論か、それとも技術書か──。どれを書いても、「誰が読むの？」という疑問がつきまとった。経験も実績もない新人が技術書を書いても、説得力はない。だからこそ、私は「#100日チャレンジ」という、私自身の物語を書くことにした。

それは“技術書ではない技術書”だった。
「ChatGPTと共に100個のアプリを作る」という挑戦を軸に、自分がどう成長していったのかを描いた。はっきり言って"妥協案"だが、私は確信していた。
──“技術の手前にある物語”が、人の心を動かす、と。

それに、エンジニア1年目は本当に不遇な時期だ。
スキルがあっても「経験年数」で見られ、どんなに頑張っても評価されない。それに、なにか大きなプロジェクトに参加しないと、技術は学べないし、適当にアテを決めて勉強するのも効率が悪い。

だからこそ、私は7ヶ月間、すべての時間と労力を本に捧げた。
結果として、『#100日チャレンジ』はベストセラーになり、7刷を重ね、Amazonカテゴリ1位にもなった。

実際、周りのエンジニアからは、ソフトウェア開発をしっかりと学び、ちゃんとソフトウェア開発の仕事をしているんだな、と思ってもらえたらしい。
批判らしい批判は、本を出してから、一度もなかった。それだけに、あの批判は心を抉るようなものがあった。

## エンジニア歴9ヶ月で何ができるのか？

2025年1月。私は「受注開発業」を始めた。
理由はシンプルだ。お金が欲しかったから。会社員をするより、独立したほうが、収入は３〜４倍よくなる。そのため、私は、本を書き終えるとすぐに独立し、受注開発業をスタートさせた。

「本を出したことで、単価の安い仕事ではなく、信頼ベースの仕事を受けられるようになった」

そして、ここでようやく「エンジニアとしての報酬」を得られるようになった。
「本を出したことで、単価の安い案件ではなく、信頼ベースの仕事が来るようになった」──そう実感したのもこの頃だ。

## ここから始まる“500万円回収記”

2025年6月、PyCon JPから講演オファーをもらった。

[https://note.com/amiotsuka/n/n9d2da7eb753a](https://note.com/amiotsuka/n/n9d2da7eb753a)

「これだ」と思った。
どうせなら、このステージを“人生最大の演出”にしてみよう、と。

そもそも、２年前、初めてコードを書き始めた頃、アプリストアに出せるようなアプリを作れることに憧れていた。そんな、かつての夢を叶える、絶好のチャンスだと思った。

だから私はその日、2025年9月27日に合わせて、『メモるん（MemoLune）』を完成させることを決めた。
開発の仕事をすべて止め、収入を断ち、開発に専念した。7月以降、受託案件を断り続けた。
でも、それでも完成させたかった。
PyConのステージで、数百人の前でライブリリースをするために。

あの日の拍手は、一生忘れない。

でも、現実はシビアだった。リリースから1週間の売上──9円。

ノリで始めた企画とはいえ、なにか足跡を残したい。
流石にこれで終わるのは、なにか思い残すものがある。

“500万円回収記”は、9円から始まった。
