# 経営学A評価の同級生はみんな会社員になり、D評価の私は起業した

ほぼすべての授業がD評価。真剣に受けた講義はゼロ。

大学時代の私は、成績表の下から数えた方が早い学生だった。出席日を予測して最低限だけ顔を出し、残りの時間はゲームやSNSで暇を潰す。教室ではなく、スマホの中に私の大学生活があった。

一方で、同級生たちは“当たり前”のように努力を積み重ねていた。授業前にはレジュメを暗記し、テスト前は何十時間も勉強。単位も成績もきっちり確保し、口では「授業なんて適当でいい」と言いながら、行動はまるで正反対。周囲から見れば、私は怠け者か無能か、そのどちらかに見えたはずだ。

しかし数年後、経営学でA評価を取ったはずの同級生は、大企業の会議室で「前例に倣って」と発言し、年に一度の昇給を待っている。一方の私は、授業中に居眠りしていたくせに、起業し、自分の裁量で未来を変える日々を送っている。

なぜ、成績最下位の私が、今は彼らと全く別の場所に立っているのだろうか？

## 成績Dの私と、A評価の同級生

私は大学の授業は、ただ単位をもらう場所だと思っていた。むしろ、退屈な授業の時間にどうやって暇をつぶすかが重要だった。上手く出席扱いにしてもらえるように、出席を取る日を推測して出席したり、不本意にも出席をしないといけない日には、ゲームをして時間を潰した。

一方、周りの同級生は、「授業なんて単位を取れたらなんでもいい」と口では言いながらも、教科書やレジェメの用語を完璧に覚え、小テストも期末テストも何十時間もかけて準備をしていた。

授業、バイト、サークル活動。毎日、色々なことに一生懸命に取り組んでいた同級生に、なんでそんな色々と頑張るのか聞いたら、「当たり前」と言われた。しかも、彼ら・彼女らは、その現状を「自由」と称していて、拍子抜けした。

## SNSの遊びがビジネスに変わった

大学4年のある日、私は「#100日チャレンジ」というタグで毎日SNSに作ったゲームやツールの動作動画を投稿を始めた。最初はただの遊びで、フォロワーにちやほやされたいだけだった。けれど継続するうちに、自然とプログラミング技術が磨かれていった。

スペインの学会に行き、メディアの取材を受け、その後出版社から声がかかり、本を出すことになった。そしてソフトウェア開発の仕事を貰ったり、その本がきっかけで講演や相談の依頼を貰ったりし、気がつけば私は起業していた。

## コアコンピタンスは机上では生まれない

経営学では「コアコンピタンス（**他社に真似できない核となる能力**）」という言葉がある。A評価の同級生は、その定義や事例を完璧に暗記していた。

けれど、自分のコアコンピタンスをつくろうと行動する人はほとんどいなかった。単位、サークル、バイト、就活――やるべきことをこなすのに精一杯で、何かに夢中になるのを恐れていたようだった。

一方で、私は本を出版したことが、そのまま唯一無二の強みになった。周りから見れば、ただの偶然かもしれない。でも実際は、それ以前も、何かに取り憑かれたように色々なことをしていた。フリマアプリで変な人に絡まれてイライラしたことをきっかけに法律を本気で学んだり、株で楽して儲けたいと思い簿記を学んだり。
（注：簿記を学んでも株は儲からない。確定申告ができるようになるだけ）

そんな、机上で覚えた知識ではなく、些細なきっかけから、何かに取り憑かれたように生きるその生き方こそが、私のコアコンピタンスになった。

## D評価は私にとっての勲章だった

A評価の同級生たちは、今も会社で“優等生”として働き、評価され続けているだろう。きっとあの頃と同じように、与えられたルールを守り、期限までに課題をこなし、上司に評価され、安定した会社員生活を送っているのだろうか。

会社員は安定している。でもその安定は、ルールの中にいる限りの話だ。私はルールを飛び出した瞬間から、何も保証されない世界に放り出された。

[https://note.com/amiotsuka/n/n4ac7f6908fa3](https://note.com/amiotsuka/n/n4ac7f6908fa3)

代わりに、自分の選択で報酬も未来も変えられる世界に来た。会社員が月給をもらうためにする努力と、独立して売上を立てるためにする努力は、似ているようで全く違う。

あのD評価は、私にとっての出発点だった。完璧な準備も、高い評価もいらない。ただ興味のあることに飛び込めば、あとから形になる。

A評価の同級生は、今もちゃんと枠の中で高得点を取り続けている。私はその外側で、点数のつかない勝負をしている。
