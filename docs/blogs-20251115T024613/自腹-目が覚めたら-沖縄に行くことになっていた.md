# 【自腹】目が覚めたら、沖縄に行くことになっていた

「まもなく出発です——沖縄行き搭乗手続き。」

朝、目を覚ますとiPhoneの画面にそう表示されていた。カレンダーには確かに「沖縄行きのフライト」と予定が入っている。寝ぼけた頭で「なんで？」と思いながらも、昨夜のことを思い出した。

Xで流れてきた、沖縄で行うイベント告知に、「沖縄、行きたいな」私はつい軽い気持ちでポストしてしまったのだ。

するとフォロワーから「行くしかない」「ネタになる」と反応が相次いだ。

深夜、友人とスペースを開いた。冗談半分のやり取りのはずが、気づけば本気で検索し、0時を回る頃、私はためらいながらも予約ボタンを押していた。

## 心を揺らす、南国の旅

すべての始まりは、Xで流れてきた一つのイベント告知だった。
AI駆動開発に関する勉強会。
[AI駆動開発 #2（沖縄開催）](https://aid.connpass.com/event/363417/)

イベントページを見た瞬間、心が少し揺れた。
「いいなぁ、沖縄か。」

こういう勉強会、セッション後には登壇者を中心に懇親会がある。正直に言えば、私はこの「飲み会」の方を楽しみにするタイプだ。

それを知っているのか、運営の方は、私が登壇しなくても、なぜか飲み会だけに誘ってくれる。

ただ、沖縄は遠い。繁忙期の航空券を調べると、片道35,000円以上。往復で7〜8万円。やらないといけないこともたくさんある。プログラムの納品、講演準備やメール返信。それに、明日すぐ行くなんて、現実的に無理だろう。そう思い、「明日は流石に無理だね」とツイートした。

## 軽いつぶやきは運命をも動かす

夜になって、返信が帰ってきた。「明日のお昼に羽田空港発で那覇空港からタクシー乗れば間に合いそうです」

すると、それが引き金になった。
フォロワーからの反応が次々と飛んでくる。
「行くしかないでしょ」
「ネタになる」
「投票取って決めなよ」

気づけば、7割以上が「つべこべ言わずに行け」と回答するアンケートになっていた。
その結果を見て笑いながらも、心の奥では少し期待していたのかもしれない。

でも「行きたい」と「行く」の間には、大きな壁がある。
金額の問題もあるし、やらなきゃいけない仕事もある。
勇気がなかった。自分の衝動を信じきれなかった。

## 夜のスペース

夜11時過ぎ。私はXのスペースを立ち上げた。
ぬこちゃんをはじめ、何人かの友人を呼んで「沖縄に行くべきか？」を相談する。

「行った方が絶対おもしろい」
「いやでも高いよね」
「予定があるからすぐに帰らないといけないけどどうする？」

みんなが本気で考えてくれた。
私自身は「ネタになるけど、8万円は痛いなぁ。３万円くらいだったらなぁ」とまだ迷っていた。
でも、いろいろ調べているうちに「スカイメイト」という制度を見つける。
当日予約なら片道22,000円で飛べるという。

その瞬間、現実味が一気に増した。 日付が変わって、飛行機が空いていたら行こう。そう決めた。

そして日付が変わってすぐの12:05。
私は予約ボタンを押していた。

## 理屈よりも、ときめく方を選べ

今、私は沖縄行きの飛行機の中でこの記事を書いている。
朝から最低限の仕事だけを片付けて、昼の便に飛び乗った。
睡眠は4時間半で、眠くて仕方がない。

久しぶりに、純粋なワクワクを感じている。
最近の私は、見える未来のために努力することが多かった。
結果を出すために、計算された行動ばかりをしてきた。

でも今日は違う。
「理屈よりも、ときめく方を選べ」
その言葉に導かれるように、自分の衝動に従った。

この選択が6万円以上の価値を生むかどうかは、まだわからない。
でも少なくとも今、機内で胸が高鳴っているこの感覚こそが、旅に出た理由なのだと思う。
