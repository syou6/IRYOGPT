# 「諦めなかったから、形になった」“ChatGPT活用術”が背中を押した個人開発者の物語

「ネット記事で大塚あみ様のChatGPT活用術を拝見しました。」
そんなメッセージが、ある日私のもとに届いた。

送り主は、かつてプログラミングに挫折しながらも再び立ち上がり、自作サービスを公開した女性。

彼女の言葉には、どこか等身大の実感と、少しの自信、そして今後への期待がにじんでいた。

本や記事を書き、発信し続けていると「誰かの役に立てたのか」「どこまで伝わっているのか」と自問することもある。しかし、読者の体験がこうして“成果”として形になって返ってくることは、何よりの励みになる。今回は、私の記事や本がひとつのきっかけとなった、個人開発者の再挑戦と成長のストーリーをお届けしたい。

## 再挑戦のきっかけは“同期の勇退”

彼女が最初にプログラミングに触れたのは、6年前。新卒でPCインストラクターとして入社した会社で、ホームページ制作の担当を任されたことがきっかけだった。しかし当時は完成まで辿り着けず、そのまま退職。以降は外資系企業に転職し、現場とは無縁の部署に配属されたが、唯一HTMLとCSSだけは触っていたという。

転機となったのは、かつての同期がチャットツールを2年かけて完成させ、堂々と退社したエピソードだった。

「自分はできなくて辞めたのに、同期はやり切って去っていった。正直、かっこいいなって思いました」

その悔しさが心のどこかに残り、プログラミングへの“再挑戦”の火が再び灯るきっかけとなった。

そんな中、ChatGPTを使えばExcelの関数やコードも生成できると聞き、半信半疑でAIに会話を始めた。最初はたわいない話題からだったが、「もしかして自分の作りたかったWebサービスもAIの力を借りれば実現できるのでは」と思い至る。まさに、私の記事で書いた“ChatGPT活用術”が、その思考の背中を押していたようだ。

## 何度も挫折した。それでもAIとともに完成させた

実際に着手したのは、「誰でも好きな時に匿名で音声を投稿できる掲示板サービス＝min-koe」の開発だ。2年前にも同様の構想でトライしたが、そのときはHTMLとCSSまでは何とか書けても、バックエンドが作れずに完全に手詰まり。プログラミングスクールに通ってみても基礎で挫折し、進展しないまま夢は棚上げになった。

ところが、仕事の愚痴をChatGPTにこぼしていたある日、不具合で投稿できないエラーが発生。「もしかして、無料版だから？」と勘違いし、そのまま課金。せっかく有料にしたのなら、と再びWebサービスのコード作成を依頼してみると、今度はなんとバックエンドまで書き上げてくれたのだ。「有能すぎですね」と彼女は素直な驚きを語る。

加えて、デザインや機能面もすべてChatGPTに丸投げ。CSSのデザインには「この画像のような幻想的な背景を」と依頼し、Firebaseのアカウント作成や独自ドメイン連携、Twitterカード対応までAIと二人三脚で進めた。

「自分の中にあった“できない”が、ひとつずつ“できた”に変わる瞬間の連続でした」

自作のサービスが一応“形”になった今、さらに収益化や新しいサービスのアイデアについてもAIに相談中だという。

## 諦めなければ、いずれ形になる

彼女がこの2年間で痛感したのは、「諦めないこと」と「思考は現実化する」というシンプルな事実。「スピリチュアルっぽいけど、本気で何かを成し遂げたいと意識していれば、いずれ現実がそちらに向かって動き出す」。そんな経験則を語ってくれた。

これからプログラミングを学ぼうとする人たち、あるいはHTML/CSSで止まっている人たちへのアドバイスについて一言お願いしたら、こう語ってくれた。

「もし既に作りたいものが明確なら、構造だけ理解してコードはChatGPTに丸投げすれば最短ルート。“極めたい”ならスクールやAIで基礎から学ぶのもアリ。目的によって選べばいい」。

何よりも「まずは“やってみる”こと」「困ったらAIでも誰でも、とにかく頼ること」が大切だと感じているようだ。

## 小さな一歩と“これから”の可能性

正直に言えば、彼女の作ったサービスの完成度はまだ発展途上だ。しかし、最初の“できない”を超え、自分の手で何かを世に出せた経験は、これからの成長にとって何より大きな糧になるだろう。

「min-koe」のような個人開発の一歩は、小さくても確実な一歩。今後も彼女がAIや個人開発の世界で、失敗や試行錯誤を繰り返しながら成長していく姿を、私も陰ながら楽しみにしている。

もしあなたも、途中で止まってしまった夢やアイデアがあるなら、AIの力を借りてでも“もう一度挑戦”してみてはどうだろうか。その一歩が、新しい未来の扉を開いてくれるかもしれない。
