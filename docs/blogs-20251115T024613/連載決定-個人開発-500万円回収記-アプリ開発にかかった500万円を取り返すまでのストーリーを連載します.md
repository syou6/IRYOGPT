# 【連載決定】個人開発・500万円回収記──アプリ開発にかかった500万円を取り返すまでのストーリーを連載します！

**「アプリを作るのに、500万円かかりました。」**

これを聞いた人はだいたい驚く。
「そんなに？」とか「それ、勤め先の会社の話でしょ？」とか。

でも、違う。全部、**私の財布**から出ている。

生成AIツール、API費用、クラウド費用、特許取得費、失った収入。

ひとつひとつはたいしたことがない。でも、それが積み重なる。
結果、気づけば“500万円のアプリ”が出来上がっていた。

そのアプリは、私が「アプリ開発者」と名乗るために、3ヶ月かけて作った、まさに夢そのものだった。

私の世界観を込めて、メモるん（Memolune)と名付けた。

でも、リリースして1週間、売上は──**9円**。

現実は、容赦なく数字で叩きつけてくる。

けれど、私はやっとこれで決心がついた。「#100日チャレンジ」の次の挑戦はこれだ、と。

## 失敗じゃない、“未完”なんだ

この500万円は“無駄遣い”ではない。
むしろ、ここまでやって初めて見えてきた景色がある。

アプリを作るというのは、受注開発とは違う。
**「自分の世界感を形成する」行為だ。**

サーバーのスキーマ設計を考え、UIをデザインし、
ロゴを決め、価値を伝える。
そのすべてを、ひとりでやってみた結果が、500万円。

[https://note.com/amiotsuka/n/nd4ca1b9b7de3](https://note.com/amiotsuka/n/nd4ca1b9b7de3)

ただ、私はいまこう思う。
「これで終わりじゃない。まだ“未完”なんだ」と。

この連載『500万円回収記』は、単なる数字の報告ではなく、
この“未完”を完成へと近づけていく過程の記録。

言い換えれば、「起業チャレンジ」というドキュメンタリーだ。

## 「どうやって回収するか」の試行錯誤を全部書く

世の中には、“成功した人の話”は溢れている。
でも、“成功するまでの泥臭い過程”は、ほとんど出てこない。実際、成功までの長い道のりは、誰も公開してくれない。

だから私は、その空白を埋めたい。この連載では、実際にこれらの取り組みを行い、公開していく。

-

課金モデルの構築

-

UIのデザインで離脱率を下げる実験

-

各種メディア媒体での発信による効果測定

-

広告費をどこにどう使うか

-

時間をどこにどれだけ割いて、どのような取り組みをしていくか

私が持っている時間、知識、情熱、そして残りの体力を、全部ここに注ぐ。
目的はただひとつ──「500万円を取り戻す」こと。
でもその回収額以上に、**個人開発者がどう戦えるかのリアル**を見せたい。

数字が増えた日も、ゼロだった日も、ちゃんと書く。
バズった日も、誰にも読まれなかった投稿も、全部残す。

嘘のないプロセスこそが、この時代の価値だと思うから。それは、#100日チャレンジも同じだった。

## 企画のルール

今回の企画を行う上で、以下のようなルールで進める。

-

500万円を、アプリ内収益だけで稼ぐ。

-

ここで、アプリ内収益とは、アプリ内で、実際に利用者が課金したり、広告を閲覧したことによって得られる収益のことをいう。

-

以下のような収入は含めない。

-

講演やメディア露出における出演料等

-

noteやxの収益

-

チップや協賛された方からの支援金

-

受注開発等、他の仕事を請け負ったことにより得た収益

-

その他、「アプリで稼いだ」と主張すると語弊を生むような収入

-

お金にものを言わせて広告を出すのはズルなので、広告費をかけた場合、アプリ内収益から広告費分を控除した分で、500万円の収益を達成する。

基本的には、ツールや私自身の生活費等は当面はnote, x, 出演料等、企画に関係ない収入で補うことにし、ほとんどすべての時間や労力をこの企画に注いでいきます。

## 個人開発者として生きる覚悟

この「500万円回収記」は、単なる金額の話ではない。
たぶん、私にとっては“お金”よりも“覚悟”の方が重かった。

アプリをつくるって、技術や時間だけの問題じゃない。自分の信念と、どれだけ真正面から向き合えるかの勝負だ。失敗したら恥ずかしいし、誰かに笑われるかもしれない。

実際、プライドだけを維持するためなら、このアプリは、「Pyconのために作った」と言って、没にすればよかった。講演と受注開発で頑張って回収すればいずれ回収できるから、金額だけの問題なら、それで問題ない。そっちのほうが、よっぽど稼げると思う。

でも、私は、これで、本当に稼げるのかどうか、自らを実験台にして、試してみたい。3ヶ月前は個人開発が流行していたけれど、今は下火だと言われている。それが下火なのか、それとも、うまく行ってない人の声が大きいだけなのか。

──それでも、私が信じたいのは「個人でも、世界を変えるアプリを作れる」という可能性だ。

たぶんこの連載は、華やかなストーリーにはならない。

**――「苦労はアニメやゲームの主人公を眺めるだけでよくて、私は楽な人生を歩みたい」**

私はそう思っているけれど、おそらく、泥臭い物語になる。けれど、その現実の中にこそ、“リアルな希望”があるはずだ。

500万円を回収できるかどうかは、正直、わからない。けれど、たとえ結果がどうであっても、この挑戦が「個人開発で食べていく」という未来を少しでも照らす光になれば、それだけで意味がある。

この物語は、失敗から始まる成功譚だ。
そして、これからが本番。
