# 【還付金欲しい】人生初の確定申告してきた！

気づけばもう3月。確定申告の期限まであと5日しかない。

「来週やろう」と言ってから、たぶん５回は週末を見送った。正直、このまま永遠にやらなくて済む方法を誰かが発明してくれないだろうかと、真剣に願った。

SNSを開くたびに**「確定申告はお早めに！」**という投稿を目にして、地味にストレスが溜まっていった。「面倒だけど、やらないといけないこと」ランキングがあったら、私の中では確実にトップ3に入る。

会社を立ち上げたばかりで毎日が忙しく、気がつけば後回しになっていた。
だが、今回やるのはあくまで会社員としての確定申告。法人と比べれば個人の確定申告の作業は簡単なはずだ。それでも、何をどうすればいいのか分からず、後回しにしてきた結果が今のこの状況である。

ため息をつきながら、部屋の中を見渡す。片付けるべき書類が積まれている机の上、床に散らかった領収書。こんな部屋だからやる気が起きないのかもしれない。でも、還付金は欲しい。

「やるしかないよね……」

私は小さく呟いた。

## タックスデ―の憂鬱

日曜日の夜、ため息と共にパソコンを開いた。

日本では特別徴収が主流だから、確定申告はあまり馴染がない。

でも、海外では個人で行うことも多く、例えばアメリカでは「Tax Day（タックスデー）」と呼ばれる確定申告の締切日（通常4月15日前後）は、他の日と比べて交通事故が増加するという研究結果があるらしい。どうやら税務作業によるストレスや焦りで、普段冷静な人でも心に余裕がなくなり、運転に影響を与えるというのが理由だそうだ。

私も実際、パソコンを前にすると急に気持ちが焦ってきた。入力する項目を見ただけで、なぜか頭が痛くなるような気がする。しかもやり方がいまいちわからない。ネットで検索しても、変な広告や素人のYoutubeチャンネルがたくさん出てきて混乱する。

結局、意味もなく寝転がったり、スマホをいじったりして時間を浪費した。ようやくパソコンの前に戻った頃には、すでに時計の針が深夜を指していた。

確かにアメリカ人の気持ちは分かる。私も今、自分が運転していたら危なかったかもしれない。歩いていても電柱に頭をぶつけるのに。それくらいイライラと焦りが募っている。ただ、いつまでも先延ばしにはできない。

心を落ち着かせて、深呼吸をしてから申告手続きを始めた。

## 源泉徴収票を求めて

会社を作るときに一度税務署に行っていたので、個人の確定申告はマイナポータルで申告できることは知っていた。しかし、実際にやってみると、想定外に手間取った。

まず「源泉徴収票」が必要らしい。部屋のどこにしまったか、記憶をたどりながら、普段使わない引き出しや書類の山をひっくり返す。

15分後、ようやく発掘した。ついでに、ふるさと納税の証明書と大学から送られてきた支払証明書も見つけた。

私は去年2万4千円を寄付したので、住民税が2万2千円浮くらしい。でも、ふるさと納税で貰えるものは食品が多くて外食派の私は特に欲しい返礼品がない。結局ネットで欲しいものを探すのも面倒で、親に欲しいもののリンクを聞いて、そっくりそのまま全部実家に直送した。

さらに7000円の大学での委託収入も記入する。経費についての欄があったので、家から大学までの交通費を入力した。

「こういう細かい費用もちゃんと経費にして、税金を減らさないとね！」

※雑所得は20万円までは課税されないので、**税金は変わらない。**

## 数字に突きつけられた現実

色々面倒なことはあったけれど、入力自体は拍子抜けするほど簡単だった。

社会人一年目の給与収入は決して多くない。すんなりと数字を入力した。画面には、「所得税3万円」と書いてあった。月ではない。年である。

「あれ、これしか引かれてなかったの！？いつも毎月5万円くらい何かが天引きされてたのに。」

所得税をあまり払っていないことに驚いた。税金が高いせいで手元にお金が残らないと思っていたのに。

しかし、画面をよく見てみると、社会保険料が約40万円と書いてある。

「会社員の収入って、老人に取られちゃう部分が随分多いんだなぁ…。」

給料に対する社会保険料の割合の大きさに驚いた。これが社会の仕組みか、と感じた。

## 確定申告の恩恵

ようやく入力を終えると、私は達成感でいっぱいだった。

**「やっと終わったー！」**

ここまで大変だった。2時間もかかった。もう夜の1時だけど、やってよかった。3万円のふくらはぎマッサージ機に5回くらいマッサージをしてもらいつつ、眠気を堪えながら頑張ってよかった!

ネットでは多くの人がふるさと納税をし、税金対策をしていると書いてある。だから、きっとたくさんお金が返ってくるに決まっている。3万円くらい戻って来るといいな。

私はドキドキしながら還付額を確認した。
