# 契約書テンプレート

医療機関と契約する際に使用する書類のテンプレート集。

## ファイル一覧

| ファイル | 内容 | 使うタイミング |
|----------|------|----------------|
| `service-agreement.md` | サービス利用契約書 | 新規契約時 |
| `dpa.md` | データ処理契約（DPA） | 患者データ取扱いの取り決め |
| `sla.md` | サービスレベル合意（SLA） | 稼働率保証が必要な場合 |
| `nda.md` | 秘密保持契約（NDA） | 機密情報を扱う場合 |

## 使い方

1. 必要なテンプレートをコピー
2. `[医療機関名]` `[日付]` などのプレースホルダーを置換
3. Word/PDFに変換して送付
4. 双方で署名・押印

## 注意

- これらはテンプレートであり、法的助言ではありません
- 重要な契約は弁護士にレビューを依頼してください
- 契約内容は案件ごとにカスタマイズが必要な場合があります
