# サービス利用契約書

合同会社AMOR（以下「甲」という）と [医療機関名]（以下「乙」という）は、甲が提供するAI予約システム「よやくらく」（以下「本サービス」という）の利用に関し、以下のとおり契約を締結する。

---

## 第1条（目的）

本契約は、甲が乙に対して本サービスを提供し、乙がこれを利用することに関する基本的な事項を定めることを目的とする。

## 第2条（サービス内容）

1. 甲は乙に対し、以下のサービスを提供する。
   - AIチャットボットによる予約受付システム
   - 予約データのGoogleスプレッドシート連携
   - 予約確認・リマインドメールの自動送信
   - 管理ダッシュボードの提供
   - 導入後3ヶ月間の無制限サポート

2. サービスの詳細仕様は、甲のWebサイト（https://yoyakuraku.com）に掲載する内容に準ずる。

## 第3条（契約期間）

1. 本契約の有効期間は、[開始日] から1年間とする。
2. 期間満了の1ヶ月前までに、甲または乙から書面による解約の申し出がない場合、本契約は同一条件で1年間自動更新されるものとし、以後も同様とする。

## 第4条（料金および支払い）

1. 初期導入費用: 300,000円（税別）
2. 月額利用料: 100,000円（税別）
3. 支払方法: 銀行振込またはクレジットカード決済
4. 支払期限:
   - 初期導入費用: 本契約締結後、サービス開始前まで
   - 月額利用料: 毎月末日締め、翌月末日払い

## 第5条（乙の義務）

乙は、以下の事項を遵守するものとする。

1. 本サービスを通じて取得した患者情報について、個人情報保護法その他関連法令を遵守すること
2. 患者に対し、本サービスの利用およびAI処理について適切に説明し、必要な同意を取得すること
3. 本サービスを通じて行われた予約について、最終的な確認および管理を行うこと
4. 甲から提供されたアカウント情報を適切に管理すること

## 第6条（甲の義務）

甲は、以下の事項を遵守するものとする。

1. 本サービスを安定的に提供するよう努めること
2. 乙から預かった情報について、適切な安全管理措置を講じること
3. 本サービスに関する問い合わせに誠実に対応すること

## 第7条（データの取扱い）

1. 本サービスを通じて取得される患者情報について、乙がデータ管理者、甲がデータ処理者となる。
2. 甲は、乙の指示に基づき、本サービスの提供に必要な範囲でのみ患者情報を処理する。
3. 患者情報の詳細な取扱いについては、別途締結するデータ処理契約（DPA）に定める。

## 第8条（禁止事項）

乙は、以下の行為を行ってはならない。

1. 本サービスを第三者に再販売または転貸すること
2. 本サービスの運営を妨害すること
3. 本サービスを利用して違法行為を行うこと
4. 甲の知的財産権を侵害すること

## 第9条（免責事項）

1. 甲は、本サービスの完全性、正確性、有用性等について保証しない。
2. AIの誤認識等により生じた予約の誤り、患者とのトラブルについて、甲は責任を負わない。
3. 甲の損害賠償責任は、甲の故意または重過失による場合を除き、乙が過去12ヶ月間に甲に支払った利用料金の総額を上限とする。
4. 天災、停電、第三者サービス（OpenAI、Google、Vercel等）の障害に起因する損害について、甲は責任を負わない。

## 第10条（解約）

1. 乙は、1ヶ月前までに書面で通知することにより、本契約を解約することができる。
2. 甲は、乙が本契約に違反した場合、催告なく本契約を解除することができる。
3. 解約時点で未払いの利用料金がある場合、乙は速やかにこれを支払うものとする。
4. 既に支払われた料金は、理由の如何を問わず返金しない。

## 第11条（秘密保持）

甲および乙は、本契約に関連して知り得た相手方の秘密情報を、本契約の目的以外に使用せず、第三者に開示しないものとする。ただし、法令に基づく開示要求がある場合はこの限りでない。

## 第12条（契約の変更）

本契約の変更は、甲乙双方の書面による合意によってのみ行うことができる。

## 第13条（準拠法および管轄裁判所）

1. 本契約の解釈は、日本法に準拠する。
2. 本契約に関する紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とする。
