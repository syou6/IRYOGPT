# サービスレベル合意書（SLA）

本書は、合同会社AMOR（以下「甲」という）が提供するAI予約システム「よやくらく」のサービス品質について定めるものである。

---

## 1. サービス稼働率

### 1.1 目標稼働率

甲は、本サービスの月間稼働率について **99.5%** を目標とする。

### 1.2 稼働率の計算

```
月間稼働率 = (月間総時間 - 計画外停止時間) / 月間総時間 × 100
```

- 月間総時間: 該当月の総時間（例: 30日 = 720時間）
- 計画外停止時間: 計画メンテナンス以外でサービスが利用不能だった時間

### 1.3 除外事項

以下の事由による停止は、稼働率の計算から除外する。

- 計画メンテナンス（事前通知済み）
- 乙の設定ミスまたは操作に起因する障害
- 第三者サービス（OpenAI、Google、Vercel等）の障害
- 天災、停電、その他不可抗力

---

## 2. 計画メンテナンス

### 2.1 事前通知

甲は、計画メンテナンスを実施する場合、原則として **3営業日前** までに乙に通知する。

### 2.2 実施時間帯

計画メンテナンスは、原則として以下の時間帯に実施する。

- 平日: 22:00〜翌6:00（日本時間）
- 土日祝: 終日可

### 2.3 緊急メンテナンス

セキュリティ上の緊急対応が必要な場合、甲は事前通知なくメンテナンスを実施できる。この場合、可能な限り速やかに乙に通知する。

---

## 3. 障害対応

### 3.1 障害レベルの定義

| レベル | 定義 | 例 |
|--------|------|-----|
| **緊急** | サービス全体が利用不能 | 全ユーザーが予約できない |
| **重大** | 主要機能が利用不能 | 予約登録ができない |
| **軽微** | 一部機能に問題がある | 表示の乱れ、速度低下 |

### 3.2 対応目標時間

| レベル | 初動対応 | 状況報告 | 復旧目標 |
|--------|----------|----------|----------|
| 緊急 | 30分以内 | 1時間ごと | 4時間以内 |
| 重大 | 2時間以内 | 4時間ごと | 24時間以内 |
| 軽微 | 翌営業日 | 必要に応じて | 1週間以内 |

### 3.3 連絡方法

- 障害発生時の連絡: メールおよび電話
- 状況報告: メール
- 復旧報告: メール

## 6. サービスクレジット

### 6.1 クレジットの適用

月間稼働率が目標を下回った場合、乙は以下のサービスクレジットを請求できる。

| 月間稼働率 | クレジット |
|-----------|-----------|
| 99.0%〜99.5% | 月額料金の5% |
| 95.0%〜99.0% | 月額料金の10% |
| 95.0%未満 | 月額料金の25% |

### 6.2 請求方法

- 乙は、該当月の翌月末までに書面でクレジットを請求する
- クレジットは翌月以降の利用料金から控除する

### 6.3 上限

サービスクレジットの上限は、該当月の月額料金の25%とする。

2. 第三者サービスの障害に起因する停止について、甲は責任を負わない。
3. サービスクレジットは、乙の唯一かつ排他的な救済手段とする。
